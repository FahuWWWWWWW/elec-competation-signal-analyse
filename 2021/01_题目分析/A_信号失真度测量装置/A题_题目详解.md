# 2021年全国大学生电子设计竞赛 - 信号及仪器仪表类题目详解

## A题：信号失真度测量装置（本科组）

### 一、任务概述
设计制作一个**信号失真度测量装置**，对来自函数/任意波形发生器的周期信号（输入信号）进行采集与分析，测得输入信号的**总谐波失真（THD）**，并可在手机上显示测量信息。

**核心难点**：
- **THD测量精度**：如何在宽频带（1kHz~100kHz）内保持高精度（误差≤3%）。
- **动态范围**：输入信号从30mV到600mV，跨越了20倍动态范围。
- **实时性与频谱分析**：要求在10秒内完成测量并显示基波与谐波的归一化幅值。

### 二、基本要求详解
1. **输入信号参数**：
   - **峰峰值电压（Vpp）**：300mV ~ 600mV。
     - *解释*：此范围相对较窄，主要考察装置在标准幅度下的基本测量能力。需要保证ADC在该幅度范围内有足够的分辨率。
   - **基频**：固定1kHz。
   - **失真度范围**：5% ~ 50%。
     - *解释*：THD=5%意味着谐波总能量约为基波的1/20，而50%则意味着谐波能量与基波相当。测量装置需要在这两个极端之间都能准确计算。
   - **测量误差**：绝对值 $\Delta = |THD_x - THD_o| \leq 5\%$。
     - *解释*：这里的5%是指THD测量值与标称值之间的绝对误差。例如，若标称THD=10%，则测量值应在5%~15%之间。
   - **时间限制**：测量与显示用时不超过10秒。

### 三、发挥部分详解
1. **输入电压扩展**：30mV ~ 600mV。
   - *深刻解释*：扩展至30mV（低幅值）对前端模拟电路提出了极高要求。此时信号接近噪声 floor，需要低噪声放大器（LNA）和良好的信号调理电路。若放大器引入额外失真，将直接导致THD测量误差增大。
2. **基频扩展**：1kHz ~ 100kHz。
   - *深刻解释*：基频上限扩展至100kHz，意味着需要考虑**5次谐波（500kHz）**的测量。根据奈奎斯特采样定理，ADC采样率至少需达到**1MSPS**以上。此外，FFT的频率分辨率与采样点数相关，若要分辨100kHz信号的谐波，需要足够长的采样时间或高分辨率FFT。
3. **测量精度提高**：$\Delta \leq 3\%$。
   - *解释*：精度要求提高，意味着系统噪声、ADC量化误差、频谱泄漏等因素的影响必须被严格控制。
4. **波形与频谱显示**：
   - 显示一个周期波形：考察装置的波形重构能力，通常通过DAC或屏幕绘制实现。
   - 显示基波与2~5次谐波的归一化幅值：
     - *解释*：归一化幅值是指各次谐波幅值相对于基波幅值的比值。例如，若基波幅值为$U_{m1}$，2次谐波为$U_{m2}$，则归一化幅值为$U_{m2}/U_{m1}$。这直接反映了失真的频谱构成。
5. **手机显示**：
   - 要求通过蓝牙/WiFi等无线方式与手机通信，实时传输测量数据。考察参赛者的无线通信协议（如BLE）应用能力。

### 四、关键参数解析
- **THD（总谐波失真）**：定义为所有谐波分量的有效值与基波分量有效值之比：$THD = \frac{\sqrt{U_2^2 + U_3^2 + ... + U_n^2}}{U_1} \times 100\%$。本题限定计算至5次谐波。
- **频谱泄漏**：若采样时间不是信号周期的整数倍，FFT会产生频谱泄漏，导致谐波能量扩散到相邻频点，严重影响THD计算精度。解决方案包括**整周期采样**或使用**窗函数**（如汉宁窗）抑制泄漏。
- **带通采样定律**：当信号为高频窄带信号时，若直接按奈奎斯特定理采样（>2倍最高频率），对ADC速率要求极高。带通采样允许采样频率低于最高频率，但需满足特定条件，可大幅降低ADC速率要求。

## J题：周期信号波形识别及参数测量装置（高职高专组）

### 一、任务概述
设计一个周期信号的**波形识别及参数测量装置**，能够识别给定信号的波形类型（正弦、三角、矩形等），并测量信号的频率、峰峰值、占空比等参数。

**核心难点**：
- **波形特征提取**：如何在时域或频域区分不同波形。
- **宽频带测量**：频率范围从1Hz到50kHz，低频测量需要极长采样时间，高频测量需要高速ADC。
- **实时性**：要求3秒内完成识别与显示。

### 二、基本要求详解
1. **输入信号范围**：$1V \leq V_{PP} \leq 5V$，$100Hz \leq f \leq 10kHz$。
   - *解释*：标准幅度与频率范围，适合初学者入门。
2. **波形识别**：正弦波、三角波、矩形波。
   - *深刻解释*：
     - **正弦波**：频谱纯净，仅含基频分量。
     - **三角波**：频谱包含基频及奇次谐波，且谐波幅度以$1/n^2$衰减。
     - **矩形波**：频谱包含基频及奇次谐波，谐波幅度以$1/n$衰减，且高频分量丰富。
     - 识别方法通常基于**频谱分析**（FFT后观察谐波结构）或**时域特征**（如斜率、过零点密度）。
3. **频率测量精度**：相对误差绝对值 $\leq 1\%$。
   - *解释*：例如1kHz信号，测量误差需控制在10Hz以内。可通过测量多个周期取平均提高精度。
4. **峰峰值测量**：相对误差绝对值 $\leq 1\%$。
   - *解释*：需要精确的ADC参考电压和信号调理电路。通常采用**过采样**或**数字滤波**提高精度。
5. **占空比测量**：矩形波占空比 $D$ 范围 $20\% \sim 80\%$，绝对误差 $\leq 2\%$。
   - *解释*：占空比 $D = \frac{t_{高电平}}{t_{周期}} \times 100\%$。测量时需准确捕获上升沿和下降沿时刻。

### 三、发挥部分详解
1. **量程扩展**：$50mV \leq V_{PP} \leq 10V$，$1Hz \leq f \leq 50kHz$。
   - *深刻解释*：
     - **50mV低幅值**：接近系统噪声水平，需要自动增益控制（AGC）或程控放大器将信号放大至ADC最佳输入范围。
     - **1Hz超低频**：一个周期长达1秒，若按整周期采样测频，单次测量至少需要数秒。为满足3秒响应要求，必须采用**过零检测+相位估计**或**等效采样**技术。
     - **50kHz高频**：ADC采样率至少需100kSPS以上。
2. **实时性**：识别结果和所有测量参数同时显示，反应时间小于3秒。
   - *解释*：要求系统具备良好的并行处理能力，通常采用高速MCU（如STM32F4系列）或DSP。
3. **类型扩展**：识别波形不少于6种。
   - *解释*：可能增加锯齿波、梯形波等。锯齿波频谱谐波丰富（含所有整数次谐波），梯形波则介于矩形波和三角波之间。
4. **参数扩展**：增加测量参数不少于3个。
   - *解释*：可能包括有效值（RMS）、平均值、绝对平均值、正/负脉冲宽度、上升/下降时间等。

### 四、关键参数解析
- **相对误差**：定义为 $\frac{|测量值 - 实际值|}{实际值} \times 100\%$。对于1%的精度要求，系统需具备高分辨率ADC（建议12位以上）和稳定的基准源。
- **波形对称性**：矩形波和三角波通常关于时间轴对称，而某些失真波形可能不对称。识别算法可利用对称性作为辅助判据。
- **抗噪声能力**：在50mV低幅值下，噪声可能导致误触发。通常采用**迟滞比较器**或**数字滤波**消除抖动。
