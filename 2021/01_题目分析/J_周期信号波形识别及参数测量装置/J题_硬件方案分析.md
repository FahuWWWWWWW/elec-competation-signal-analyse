## 方案A：纯MCU方案（STM32H743VGT6）

### 1. 系统架构

```
信号输入 (50mV~10Vpp, 1Hz~50kHz)
    │
    ├── [输入保护]  TVS PESD5V0S1UB + BAT54S钳位
    │
    ├── [程控衰减/放大]  
    │      前端电阻衰减网络 (×0.1/×0.2/×1)
    │      PGA LTC6912 (双通道, 增益×1~×100, SPI控制)
    │
    ├── ┌─────────────────────┬────────────────────┐
    │    │ [信号支路]           │ [比较器支路]        │
    │    │ 抗混叠LPF (fc=100kHz) │ TLV3501 高速比较器  │
    │    │ 3阶巴特沃斯            │ 自适应阈值(DAC生成)  │
    │    └─────────┬─────────────┴────────┬───────────┘
    │              │                      │
    │              v                      v
    │         [STM32H743]            [TIM2/3 输入捕获]
    │         ADC1/ADC2              频率/占空比/脉宽
    │         16bit @ 1MSPS
    │              │
    │              v
    │         ┌───────────── DMA ─────────────┐
    │         │ FFT频谱分析 (波形分类)         │
    │         │ 时域特征提取 (峰谷值/Vpp/RMS)  │
    │         │ 参数融合: 频率+Vpp+占空比+...   │
    │         └────────────────────────────────┘
    │
    └── [输出]  TFT-LCD + 蓝牙/手机
```

### 2. 前端调理电路

**2.1 输入保护**
- TVS: PESD5V0S1UB (双向5V钳位, 响应<1ns)
- 串入 1kΩ 限流电阻
- BAT54S 肖特基钳位至 VCC+0.3V / GND-0.3V

**2.2 程控增益PGA**
- LTC6912-1: 双通道低噪声PGA
  - 增益范围: ×0, ×1, ×2, ×5, ×10, ×20, ×50, ×100 (3线SPI)
  - 带宽: 10MHz@G=10, 4MHz@G=100
  - 输入噪声: 8nV/√Hz
  - 轨到轨输出, 单电源2.7~10.5V
- 前端电阻衰减网络:
  | 输入Vpp | 衰减比 | PGA增益 | ADC输入(近似) |
  |---------|--------|---------|---------------|
  | 50~100mV | ×1 | ×20~×40 | 1.0~2.0V |
  | 100mV~1V | ×1 | ×5~×10 | 0.5~1.0V |
  | 1V~3.3V | ×1 | ×1~×2 | 1.0~2.0V |
  | 3.3V~10V | ×0.33 | ×1 | 1.1~3.3V |

**2.3 抗混叠LPF**
- 类型: 3阶巴特沃斯 (Sallen-Key+RC)
- 截止频率: fc = 100kHz (保护50kHz内有效信号, 抑制>100kHz噪声)
- 计算: 取C1=C2=330pF, R = 1/(2π × 100kHz × 330pF) ≈ 4.82kΩ, 取4.7kΩ
- 1阶节: R=4.7kΩ, C=330pF → f=102.6kHz
- 运放: OPA2170 (双通道, 3MHz GBW, 轨到轨)

**2.4 比较器支路**
- TLV3501: 高速推挽比较器, 传播延迟4.5ns
- 阈值电压: 由STM32H743的DAC输出提供 (自适应中点阈值)
  - 阈值 = (采样信号最大值 + 采样信号最小值) / 2
  - DAC精度: 12bit
- 迟滞: 正反馈电阻网络 Rf=100kΩ, R_in=10kΩ → Vhyst≈3mV
- 输出至TIM2_CH1输入捕获, 记录上升/下降沿时间

### 3. 后处理与算法

**3.1 ADC配置**
- ADC1: 16bit, 1MSPS
- 触发方式: TIM3定时器触发
- 采样点数: N=1024 (可调, 依据频率自适应)
- DMA: 双缓冲循环采集

**3.2 频率测量（自适应算法）**
- 低频段 (1Hz~200Hz): 周期法
  - 定时器测量过零间10个周期总时间T_total
  - f = 10 / T_total
  - 耗时: 10/f ≤ 10s (1Hz时10秒, 超3秒要求 → 降为3个周期)
  - 改进: 3个周期平均, f = 3/T_3 (1Hz时3秒完成)
- 中频段 (200Hz~5kHz): 计数法
  - 固定门限时间 T_gate = 0.5s
  - f = N_cnt / T_gate
- 高频段 (5kHz~50kHz): 计数法
  - 门限 T_gate = 0.1s → 至少500个周期
  - f = N_cnt / T_gate

**3.3 波形识别（分层分类器）**

第一层（时域粗筛）:
```
定时器捕获 → 判断是否方波/脉冲波（占空比稳定且有明显平顶）
             ↓
ADC采样 → 过零斜率分析（正弦波过零斜率最大且线性度最好）
```

第二层（频域FFT分类）:
```
1024点FFT → 提取特征:
  • 是否有显著2次谐波: 三角/梯形/锯齿有, 正弦几乎无
  • 3次谐波与基波幅值比:
    - 正弦: U₃/U₁ < -40dB
    - 三角: U₃/U₁ ≈ -19.6dB (理论1/9)
    - 方波: U₃/U₁ ≈ -10.5dB (理论1/3)
    - 矩形脉冲: 频谱梳状, 零点间距=1/τ
    - 锯齿: 所有整数次谐波, U₂/U₁ ≈ -13.5dB
    - 梯形: U₃/U₁在一定范围内可调
```

第三层（确认）:
```
相关系数法: 与各波形模板计算Pearson相关系数
```

**3.4 参数测量**
- Vpp: 采样序列最大值-最小值 → 乘以增益修正系数
- RMS: 真有效值 Vrms = √(Σx²[n]/N)
- 占空比: 定时器输入捕获测量高电平时间/周期
- 上升时间: 10%~90%沿时间 (限高频信号)
- 直流分量: 采样均值

**3.5 响应时间预算**
| 阶段 | 耗时 | 说明 |
|------|------|------|
| 增益自适应 | 20ms | 预采样50点估算幅度 |
| 频率测量 | 100ms | 高频段0.1s门限 |
| 波形采集 | 2ms | 1024点@1MSPS |
| FFT | 0.15ms | 1024点Cortex-M7 DSP |
| 特征分类 | 0.05ms | 简单逻辑判断 |
| 显示刷新 | 50ms | SPI LCD |
| **总计** | **~172ms** | 远小于3s |

### 4. 关键参数验证

**验证1：低频1Hz测量的响应时间**

> 周期法: 需要捕获N=3个过零周期  
> T_acq = 3 × 1s = 3s (恰好等于3s要求)  
> 若f=0.5Hz: T_acq = 6s > 3s → 不可行  
> 但题目频率下限1Hz, T_acq=3s刚好满足  
> 改进方向: 允许误差放宽, 采用单周期法, f=1/T, 耗时1s

**验证2：ADC分辨率满足50mV信号测量**

> 50mV信号经LTC6912放大×40 = 2.0V  
> ADC满量程3.3V, 信号占满量程60%  
> H743 ADC 16bit @ 3.3V: 1LSB = 3.3/65536 ≈ 50.4μV  
> 折算到输入端: 50.4μV / 40 = 1.26μV/LSB  
> 50mV/1.26μV ≈ 39682 LSB → 有效分辨率 ≈ 15.3bit  
> SNR ≈ 6.02×15.3 + 1.76 ≈ 93.8dB  
> 满足1% Vpp测量精度 (至少需10bit有效分辨率)

**验证3：频率测量精度**

> 计数法: 门限T_gate=0.1s, 50kHz信号 → N=5000周期  
> 计数器误差: ±1计数 → ε = ±1/5000 = ±0.02%  
> 周期法: 3周期@1Hz, 72MHz定时器 → T=3×1s=3s  
> 定时器分辨率: 13.9ns → ε = 13.9ns/3s = 4.6×10⁻⁷  
> 均满足≤1%要求

### 5. 优缺点

**优点:**
- 单芯片方案, 集成度高, BOM精简
- H743内置ADC和DAC满足全部需求
- Cortex-M7 DSP高效FFT, 响应时间充足
- 自适应增益和频率算法灵活

**缺点:**
- 超低频(1Hz)响应时间接近极限
- 内置比较器功能不如专用比较器
- 模拟和数字部分在同一芯片上易产生耦合噪声
- 50mV小信号时需高增益(×40), 对PCB布局敏感

## 方案B：MCU+专用芯片方案（STM32H723ZGT6 + AD7606C-16）

### 1. 系统架构

```
信号输入 (50mV~10Vpp, 1Hz~50kHz)
    │
    ├── [输入保护]  TVS+钳位
    │
    ├── [无源衰减网络]  ×1/×0.2/×0.1 (继电器切换)
    │
    ├── [仪表放大器]  AD8221 (高CMRR, G=1~1000)
    │
    ├── ┌──────────────────────┐
    │    │ 专用ADC: AD7606C-16   │
    │    │ 16bit, 1MSPS, ±10V输入  │
    │    │ 内置PGA (×1/×2/×4/×8)  │
    │    │ 内置抗混叠滤波器 (fc可配)  │
    │    │ 并行/SPI双接口       │
    │    └──────────┬───────────┘
    │               │
    │               v
    │    ┌──────────────────────┐
    │    │ STM32H723ZGT6          │
    │    │ SPI+DMA读AD7606数据   │
    │    │ FFT频谱分析           │
    │    │ 波形分类器            │
    │    │ 参数计算              │
    │    └──────────────────────┘
    │               │
    ├── [比较器]  LM393 (低速) + TLV3501 (高速) 双路
    │             TIM8输入捕获 → 频率/占空比
    │
    └── [输出]  TFT-LCD + 蓝牙/手机
```

### 2. 前端调理电路

**2.1 仪表放大器**
- AD8221ARZ: 精密仪表放大器
  - CMRR: 90dB min (G=10), 110dB min (G=100)
  - 带宽: 470kHz@G=100, 1.8MHz@G=1
  - 输入噪声: 8nV/√Hz
  - 增益: 外部电阻 RG (G=1+49.4kΩ/RG)

**2.2 专用ADC: AD7606C-16**
- 分辨率: 16bit (无失码)
- 采样率: 1MSPS (所有通道)
- 通道数: 8路同步采样
- 输入范围: ±10V, ±5V, ±2.5V (软件可选)
- 内置PGA: ×1/×2/×4/×8
- 内置抗混叠滤波器: 一阶RC, 截止频率可配置
- SNR: 89dB typ @ 10kHz
- INL: ±0.5LSB typ
- 接口: 并行(16bit) / SPI (最高40MHz)

**2.3 比较器电路**
- 双通道设计:
  - TLV3501: 高速比较器, 用于50kHz方波边沿检测
  - LM393: 低速比较器, 用于1kHz以下信号
- 阈值: 由STM32 DAC + 外部基准分压提供

### 3. 后处理与算法

**3.1 ADC配置**
- 采样率: fs = 500kSPS (AD7606C-16)
- 输入范围: ±5V (50mV~10Vpp适配)
- 同步: ADC内部OSC提供转换时钟
- SPI: 模式0, 40MHz时钟, 读取16bit × 1通道
- DMA: 双缓冲, 每帧N=1024点

**3.2 波形识别算法**
- 同方案A分层分类器
- 利用AD7606的高SNR(89dB)提升低幅值信号的识别准确率
- 增加特征维度:
  - 波形对称性: skew = (T_rise - T_fall) / (T_rise + T_fall)
  - 峰度系数: 区分三角(1.8)和正弦(1.5)
  - 过零点附近曲率: 通过二次差分近似

**3.3 参数测量**
- 频率: 计数法为主, FFT插值法为辅 (交叉验证)
- Vpp/RMS: 利用AD7606的±10V输入范围直接测量, 无需外部衰减
- 占空比: 比较器+定时器捕获
- 增加参数:
  - 上升时间: 10%~90%幅值间时间
  - 脉冲宽度: 正/负脉宽
  - 占空比偏移: D-50%

### 4. 关键参数验证

**验证1：AD7606动态范围覆盖50mV~10V**

> AD7606 @ ±5V量程: 1LSB = 10V/65536 ≈ 152.6μV  
> 50mV信号占: 50mV/152.6μV ≈ 328 LSB → 约8.4bit有效  
> 若使用内部PGA×8: 信号放大至400mV → 2620 LSB → 11.4bit有效  
> 10V信号 @ ±10V量程: 满量程, 16bit全分辨率  
> 结论: 需要PGA协助低幅值测量

**验证2：ADC采样率满足50kHz信号Nyquist**

> f_max = 50kHz, Nyquist: fs ≥ 100kSPS  
> 实际: fs = 500kSPS  
> 满足余量: ×5  
> 50kHz每周采样点数: 500k/50k = 10点 → 足够波形还原

**验证3：占空比测量精度**

> 50kHz方波: 周期=20μs, 要求精度≤2%=0.4μs  
> TLV3501传播延迟: 4.5ns (可忽略)  
> 定时器捕获分辨率: 72MHz → 13.9ns  
> 误差: 13.9ns / 20μs = 0.07% << 2%  
> 满足要求

### 5. 优缺点

**优点:**
- AD7606集成度高 (8ch ADC+PGA+滤波器+基准)
- 16bit高精度, ±10V宽输入范围无需外部衰减
- 内置抗混叠滤波器减轻前端设计压力
- 8通道同步采样可扩展多路信号分析

**缺点:**
- AD7606单价较高(¥35+)
- SPI模式下1MSPS传输需40MHz SCLK
- 芯片封装较大 (LQFP64/48)
- 不同于纯MCU方案, 多芯片增加了复杂性

## 方案C：FPGA异构方案（Artix-7 XC7A35T + STM32F407VGT6）

### 1. 系统架构

```
信号输入 (50mV~10Vpp, 1Hz~50kHz)
    │
    ├── [输入保护]  TVS+钳位
    │
    ├── [PGA]  AD8251 (×1/×2/×4/×8)
    │
    ├── [抗混叠LPF]  4阶巴特沃斯, fc=100kHz
    │
    ├── [ADC]  AD9226 (12bit, 65MSPS, 本配置10MSPS)
    │
    ├── ┌─────────────────────────────────────────────┐
    │    │  Artix-7 XC7A35T                              │
    │    │   • ADC采样控制 (连续模式@10MSPS)              │
    │    │   • 1024/2048/4096点缓存 (BRAM)             │
    │    │   • 实时硬件FFT (Xilinx FFT IP)              │
    │    │   • 频谱特征提取引擎 (谐波幅值/峰值搜索)       │
    │    │   • 时域特征提取 (Vmax/Vmin/过零检测)         │
    │    │   • FIFO + SPI Slave → 发送结果至MCU         │
    │    └─────────────────────────────────────────────┘
    │                         │
    │    ┌────────────────────┴────────────────────────┐
    │    │  STM32F407VGT6                                │
    │    │   • SPI Slave 接收FPGA端处理结果               │
    │    │   • 波形分类最终决策 (决策树/阈值联合)         │
    │    │   • 参数显示控制 (TFT-LCD)                    │
    │    │   • 蓝牙通信                                  │
    │    │   • 按键/菜单管理                             │
    │    └─────────────────────────────────────────────┘
    │
    └── [输出]  TFT-LCD + 蓝牙
```

### 2. 前端调理电路

**2.1 PGA**
- AD8251: 增益×1/×2/×4/×8 (SPI控制)
- 前端衰减: ×0.1 (10V量程)

**2.2 抗混叠LPF**
- 类型: 4阶巴特沃斯 (2个2阶Sallen-Key级联)
- fc = 100kHz (保护50kHz基频, 抑制>100kHz混叠)
- 元件计算:
  - 第1节: Q=0.541 (归一化系数)
    - R1=R2=4.7kΩ, C1=680pF, C2=330pF
  - 第2节: Q=1.306
    - R1=R2=4.7kΩ, C1=1nF, C2=150pF
- 运放: OPA2822 (双通道, 80MHz GBW, 适合滤波)

**2.3 ADC: AD9226**
- 配置: 10MSPS, 12bit并行
- 输入范围: 2Vpp
- 采样时钟: 由FPGA内部PLL产生 (10MHz)

**2.4 比较器**
- 可选: 由FPGA内部通过ADC数据进行数字过零检测 (无需外部比较器)
- 硬件比较器备选: TLV3501 (用于50kHz方波)

### 3. 后处理与算法

**3.1 FPGA高速采集**
- AD9226 @ 10MSPS持续采集
- 采样深度自适应:
  - f ≥ 1kHz: N=2048点 → 采集时间=204.8μs
  - f < 1kHz: N=1024点(降速)或等效采样
- 数据存入BRAM (可用双端口BRAM 36Kb × 多个)

**3.2 FPGA FFT引擎**
- Xilinx FFT IP核: 2048点, 16bit定点, 流水线
- 处理时间: ~2048+延迟@10MHz = ~250μs
- 窗函数: Hann窗 (识别用, 旁瓣-31dB足矣)
- 频谱特征提取:
  - 基频峰值搜索 (最高幅值bin)
  - 2~5次谐波幅值提取 (数字化能量比)
  - 频谱梳状结构识别 (矩形脉冲的零点)

**3.3 时域特征提取 (FPGA并行)**
- 硬件峰值检测: 滑动窗口比较器
- 过零检测: 符号变化检测 (50ns分辨率)
- 波形重建: 从BRAM读出一周期数据, 通过SPI发送至MCU显示
- 占空比: 硬件计时器 (FPGA内部100MHz计数器, 10ns分辨率)

**3.4 MCU (STM32F407) 处理**
- 接收FPGA的频谱特征和时域特征
- 波形分类: 决策树分类器 (预训练阈值)
- 参数综合计算:
  - THD = 从谐波数据计算
  - 有效值 = MCU从时域数据计算
- LCD显示: 波形+参数+识别结果
- 蓝牙传输: 手机APP实时显示

### 4. 关键参数验证

**验证1：过采样提升波形识别信噪比**

> AD9226 @ 10MSPS, 12bit, SNR=70dB  
> 所需采样率: fs_need = 2 × 50kHz = 100kSPS  
> OSR = 10MSPS / 100kSPS = 100  
> ΔSNR = 10·log₁₀(100) = 20dB  
> 等效SNR = 70 + 20 = 90dB → ENOB ≈ 14.7bit  
> 远超12bit要求

**验证2：响应时间分析**

> 采集: 2048点@10MSPS = 204.8μs  
> FFT: ~250μs  
> 特征提取: ~50μs (硬件并行)  
> SPI传输: 100bytes @ 10MHz = 80μs  
> MCU处理: ~50μs  
> 显示: 50ms (LCD刷新)  
> 总时间: <51ms (不含LCD)  
> 远小于3s限制

**验证3：频率测量分辨率**

> FPGA内部100MHz计数器, 测量1Hz信号时:  
> 门限T_gate=1s → N=100M计数  
> 误差 = ±1/100M = 10⁻⁸ → 可忽略  
> 测量50kHz信号: 200个周期@10MSPS  
> 等效时间分辨率 = 100ns → 误差 = 100ns/20μs = 0.5%

### 5. 优缺点

**优点:**
- FPGA硬件并行处理, 响应时间极短 (ms级)
- 10MSPS高速采样充分过采样提升分辨率
- 灵活的FFT长度和窗函数配置
- 硬件可重构, 适应多种信号处理算法
- FPGA内部实现比较器功能, 减少外部器件

**缺点:**
- 开发难度高, VHDL/Verilog + C双平台
- FPGA编程耗时, 调试困难
- BOM成本较高
- 功耗较大 (~600mA @ 3.3V)
- 低频率(1Hz)下需大量存储深度

## 三方案对比总结

| 维度 | 方案A (纯MCU) | 方案B (MCU+AD7606) | 方案C (FPGA异构) |
|------|-------------|-----------------|----------------|
| 核心优势 | 集成度高/BOM低 | ADC性能优/±10V直连 | 处理快/可重构 |
| 采样 | 内置16bit@1MSPS | 外置16bit@1MSPS | 外置12bit@10MSPS |
| 等效ENOB | ~15.3bit(小信号) | ~15.5bit(大动态) | ~14.7bit(过采样) |
| 波形识别 | FFT+特征+决策树 | 同左+更多特征 | FF硬件加速+并行 |
| 响应时间 | ~170ms | ~200ms | ~50ms |
| 开发难度 | ★★ (低) | ★★★ (中) | ★★★★★ (高) |
| BOM成本 | ~¥170 | ~¥240 | ~¥307 |

**推荐**: 方案A (H743纯MCU) 性价比最高, 适合大部分参赛队伍。方案B适合有预算且追求测量精度的队伍。方案C适合有FPGA开发经验、追求极致响应速度和技术创新的队伍。
