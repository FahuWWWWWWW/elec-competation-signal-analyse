### 2. 前端调理电路

#### 2.1 TDR脉冲发生器 + 等效采样前端

| 模块 | 器件 | 参数 |
|------|------|------|
| 脉冲产生 | STM32H743 HRTIM + EL7156 | 上升沿<3ns, 幅度5V, 宽度10ns |
| 阶跃整形 | SMMD835 SRD | tr=150ps, 产生亚纳秒阶跃 |
| 接收缓冲 | OPA847 (BW=3.9GHz, SR=950V/μs) | G=+2, 带宽~500MHz |
| 比较器 | TLV3501 (4.5ns传播延迟) | 检测阈值可调 |

**抗混叠滤波器**：TDR采用等效采样，等效采样率≥4GSPS时，信号带宽约500MHz。前置抗混叠滤波器采用5阶椭圆低通，fc=500MHz：

截止频率计算（椭圆滤波器Cauer拓扑）：
$$f_c = \frac{1}{2\pi RC} = 500\text{MHz}, \text{取} R=50\Omega, C=6.4\text{pF}$$

#### 2.2 4线电阻测量前端

| 模块 | 器件 | 参数 |
|------|------|------|
| 恒流源 | REF200 + NPN扩流 (BC847) | 10.0mA ±0.5%, 温度系数25ppm/°C |
| 差分放大 | INA333 (G=100, BW=15kHz) | CMRR=100dB, Voffset=25μV |
| 低通滤波 | 2阶RC, fc=1kHz | 抑制50Hz工频干扰 |
| ADC | H743内置ADC3 (16-bit) | 采样率1kSPS, 分辨率15μV |

**4线法实现**：端口1的引脚1、2通恒流(驱动线)，端口2的引脚1、2测电压(检测线)，消除接触电阻影响。

#### 2.3 30MHz衰减测量前端

| 模块 | 器件 | 参数 |
|------|------|------|
| 信号源 | H743 TIM+DAC + 外部LPF | 30MHz正弦, 1Vpp |

**终端匹配**：接收端RJ45引脚3、6对地各接50Ω电阻(串联得100Ω差分匹配)。

### 3. 后处理与算法

#### 3.1 TDR处理

| 步骤 | 方法 | 参数 |
|------|------|------|
| 等效采样 | HRTIM 10ps步进 → 触发ADC | 等效采样率100GSPS, 累加32次平均 |
| 反射检测 | 滑动窗差分法 + 自适应阈值 | 检测精度±1个采样点(±0.05m) |
| 距离计算 | L = v·Δt/2, v=0.67c | VF校准: 用已知20m线缆标定 |
| 故障类型 | 反射极性判断开路/短路/阻抗变化 | Γ正→开路, Γ负→短路 |

**FFT反射分析**：采集1024点TDR波形，做FFT分析阻抗变化频率分布。

#### 3.2 电阻测量算法

| 步骤 | 方法 |
|------|------|
| 差分ADC | 采集INA333输出Vdiff |
| 电阻计算 | R = Vdiff / (G × I_source) = Vdiff / (100 × 10mA) |
| 温度补偿 | 内部温度传感器校正铜线温漂 (0.4%/°C) |

#### 3.3 30MHz衰减算法

| 步骤 | 方法 |
|------|------|
| 校准 | 直通校准(0dB参考) + 开路/短路校准 |
| 测量 | AD8361输出ADC → 查功率-电压表 |
| 衰减计算 | Atten(dB) = 20log10(Vout_cal / Vout_meas) |

### 4. 关键参数验证

#### 4.1 TDR时间分辨率 → 距离分辨率

时间分辨率目标：5ns（对应0.5m@1%精度@50m）

等效采样实现：
$$f_{seq} = N \times f_s = 32 \times 3.6\text{MSPS} = 115.2\text{MSPS}$$

实际：HRTIM步进10ps，等效采样率：
$$f_{eq} = \frac{1}{10\text{ps}} = 100\text{GSPS}$$

对应距离分辨率：
$$\Delta L = \frac{v \times \Delta t}{2} = \frac{0.67 \times 3\times10^8 \times 10\times10^{-12}}{2} = 1.005\text{mm} \ll 0.5\text{m} \checkmark$$

#### 4.2 电阻测量精度

10mA恒流源经100Ω线缆产生压降：V = 10mA × 100Ω = 1V

INA333 G=100 → 输出100V? NO — INA333输出受电源限制(3.3V供电)。

重新计算：R_max=50m×0.17Ω/m = 8.5Ω，V_diff_max = 10mA×8.5Ω = 85mV
INA333 G=50 → Vout = 50×85mV = 4.25V > 3.3V ❌

修正：G=20 → Vout_max = 20×85mV = 1.7V ✅

ADC 16-bit @ 3.3V参考：
$$\text{分辨率} = \frac{3.3V}{2^{16}} = 50.3\mu\text{V}$$
$$\text{电阻分辨率} = \frac{50.3\mu\text{V}}{20 \times 10\text{mA}} = 0.25\text{m}\Omega \ll 5\% \times 0.17\Omega \checkmark$$

#### 4.3 30MHz衰减测量动态范围

AD8361输出传递函数：Vout = 1.05 × Vrms + 0.15V（线性RMS-DC）

输入0.01Vrms → Vout = 0.16V
输入1.0Vrms → Vout = 1.20V

ADC 12-bit分辨率：3.3V/4096 = 0.8mV
dB分辨率 ≈ 20log10(1+0.8mV/Vout) ≈ 0.01dB @ 1Vrms ✅（远好于10%要求）

### 5. 优缺点

| 优点 | 缺点 |

| 模块 | 器件 | 参数 |

**抗混叠滤波器**：TDR信号带宽~500MHz，采用5阶LC椭圆低通fc=500MHz：
$$f_c = \frac{1}{2\pi\sqrt{L_1C_1}} \approx 500\text{MHz}$$
取L1=10nH, C1=5pF，对应截止频率503MHz。

### 3. 后处理与算法

#### 3.1 TDR处理

实时采样60MSPS → 距离分辨率：
$$\Delta L = \frac{0.67 \times 3\times10^8}{2 \times 60\times10^6} = 1.675\text{m}$$

不能满足1%精度@50m(0.5m)。需要等效采样增强：

等效采样：Si5351 70ps步进 → 等效14.3GSPS
$$\Delta L_{eq} = \frac{0.67 \times 3\times10^8 \times 70\times10^{-12}}{2} = 7.04\text{mm} \checkmark$$

#### 3.2 电阻测量算法

MCP3421 18-bit, PGA=8, Vref=2.048V:
- LSB = 2.048V / (8 × 2^18) = 0.977μV
- 电阻分辨率 = 0.977μV / (100 × 10mA) = 0.977μΩ ✅

#### 3.3 衰减测量算法

AD8307: Vout = 0.025 × Pin(dBm) + 2.0 (斜率25mV/dB, 截距2.0@-75dBm)
直接ADC采样电压 → 查公式得dBm → 计算衰减。

### 4. 关键参数验证

#### 4.1 TDR脉冲上升沿要求

被测距离范围1-50m。最短距离1m对应往返时间：
$$t_{round} = \frac{2 \times 1\text{m}}{0.67 \times 3\times10^8} = 9.95\text{ns}$$

脉冲上升沿<5ns时，开路反射在9.95ns后可识别，无上升沿重叠问题 ✅

#### 4.2 Si5351注入30MHz信号质量

Si5351输出方波，5阶椭圆LPF后：
- 30MHz基波保留，衰减<0.5dB
- 3次谐波(90MHz)抑制>40dB
- 5次谐波(150MHz)抑制>60dB
- 总谐波失真(THD) < 1% ✅（满足衰减<10%要求）

#### 4.3 信噪比预算

- Si5351 30MHz输出 -3dBm
- 50m线缆衰减 -10dB
- 接收端信号 -13dBm
- AD8307灵敏度 -75dBm
- SNR margin = -13 - (-75) = 62dB ✅

### 2. 前端调理电路

#### 2.1 TDR子系统（FPGA核心优势）

| 模块 | 器件 | 参数 |

| 脉冲发生 | FPGA PISO @ 500MHz → ONET8501P | tr<200ps, 宽度2ns, 幅度3.3V |
| 接收缓冲 | OPA855 (BW=8GHz, SR=3000V/μs) | G=+2, BW=4GHz |
| ADC | ADC08D500 (8-bit, 500MSPS) | NRZ模式下单通道500MSPS |
| 数字FIFO | FPGA内部BRAM 8K×16bit | 连续存储4μs波形 |

**抗混叠滤波器**：实时采样500MSPS，奈奎斯特频率250MHz，采用3阶贝塞尔低通fc=250MHz：
$$f_c = \frac{1}{2\pi RC} = 250\text{MHz}, R=50\Omega, C=12.7\text{pF}$$

**TDR脉冲品质**：FPGA内部使用500MHz时钟，通过OSERDES产生2ns脉冲，tr=200ps：
$$\text{空间分辨率} = \frac{0.67 \times 3\times10^8 \times 200\times10^{-12}}{2} = 2.01\text{cm} \ll 0.5\text{m} \checkmark$$

### 3. 后处理与算法

#### 3.1 TDR实时处理（FPGA流水线）

```
ADC @ 500MSPS → 串行LVDS → FPGA ISERDES → 异步FIFO → 累加平均器
                                                          ↓
                                              实时阈值检测 (FPGA比较器)
                                                          ↓
                                              反射时间戳 → F407
                                                          ↓
                                              距离计算 + 显示
```

**FPGA处理延迟**：从ADC采样到反射时间戳输出 < 500ns (250个时钟周期@500MHz)。

**等效采样模式**（选配）：ADC08D500支持DES模式(双沿采样) → 1GSPS。若需要更高分辨率，使用FPGA逐次逼近等效采样：脉冲重复频率1MHz，每周期步进1ns → 等效1GSPS。

#### 3.2 电阻测量（MCU）

ADS1256 + INA333组合：
- PGA=64, Vref=2.5V
- LSB = 2.5V / (64 × 2^24) = 2.33nV
- 最终电阻分辨率 ≈ 2.33nV / (20×10mA) = 1.16×10^-8Ω ✅

#### 3.3 30MHz测量（FPGA数字下变频）

```
ADL5904 → FPGA XADC → 数字RMS计算 → dBm转换
Si5351  → FPGA控制 → 频率/幅度设定 → 扫频
```

FPGA内可做数字FFT分析干扰频谱，分离30MHz基波。

### 4. 关键参数验证

#### 4.1 TDR实时500MSPS采样距离分辨率

$$f_s = 500\text{MSPS}, \Delta t = \frac{1}{500\text{MHz}} = 2\text{ns}$$

$$\Delta L = \frac{v \times \Delta t}{2} = \frac{2\times10^8 \times 2\times10^{-9}}{2} = 0.2\text{m}$$

满足1%精度@50m = 0.5m要求 ✅

#### 4.2 ADC08D500 SNR验证

8-bit ADC理论SNR：
$$\text{SNR} = 6.02N + 1.76 = 6.02 \times 8 + 1.76 = 49.9\text{dB}$$

实际@500MSPS, fin=30MHz：
- 典型ENOB = 7.2-bit (数据手册)
- 实际SNR = 6.02×7.2+1.76 = 45.1dB ✅

反射信号幅度可低至满量程的1/10（5线衰减-20dB反射），此时信号功率比满量程低20dB：
$$\text{SNR}_{reflection} = 45.1 - 20 = 25.1\text{dB} > 10\text{dB} \checkmark$$

#### 4.3 30MHz衰减测量链路预算

| 节点 | 功率 | 说明 |
|------|------|------|
| Si5351输出 | -3dBm | 方波基波分量 |
| 7阶LPF | -1dB | 插入损耗 |
| 巴伦+驱动 | -1dB | 差分转换损耗 |
| 注入线缆 | -3dBm | 100Ω匹配→双绞线 |
| 50m线缆衰减 | -10dB | Cat5e@30MHz典型值 |
| 接收端 | -13dBm | |
| ADL5904检波 | +20dB | 检波转换增益 |
| XADC采样 | 0.5-2.5V | 动态范围2V |

XADC 12-bit, 动态范围2V：LSB = 2V/4096 = 0.488mV
对应功率分辨率 ≈ 0.05dB ✅（远超10%要求）

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| FPGA实时采样500MSPS，无需等效采样 | BOM成本高（~¥620） |
| TDR脉冲质量最优（tr<200ps） | 布局布线难度大，需6层以上PCB |
| FPGA硬件FFT加速，速度快 | 功耗高（~3W），需散热 |
| ADC08D500+OPA855达到仪器级性能 | FPGA开发周期长，Verilog编写调试复杂 |
| 流水线处理，测量时间<1s | XC7A35T封装小，BGA焊接困难 |
| 扩展性好，可升级为网络分析仪 | 需掌握FPGA+MCU协同设计 |
