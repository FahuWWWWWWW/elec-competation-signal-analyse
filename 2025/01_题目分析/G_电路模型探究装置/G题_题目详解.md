# 2025年电赛G题「电路模型探究装置」题目详解

## 一、题目核心要求拆解

### 1.1 任务概述

设计并制作两个部分：
1. **已知模型电路**：RC有源滤波电路，传递函数 $H(s) = \frac{5}{10^{-8}s^2 + 3\times10^{-4}s + 1}$
2. **探究装置**：能自动产生正弦信号，控制已知电路输出幅度，并能对未知RLC电路进行学习建模

### 1.2 关键指标一览

| 要求项 | 具体指标 | 分值 | 考核本质 |
|--------|---------|------|---------|
| **基本要求(1)** | 搭建已知电路H(s)，输出幅度误差≤10% | 14分 | **RC有源滤波器实现** |
| **基本要求(2)** | 探究装置产生正弦波：100Hz~1MHz步进100Hz，误差≤5%，Vpp≥3V | 10分 | **DDS信号源** |
| **基本要求(3)** | 控制已知电路输出2Vpp，误差≤5% | 10分 | **幅度自动控制+传递函数计算** |
| **基本要求(4)** | 控制已知电路输出1~2Vpp(步进0.1V)，误差≤5% | 16分 | **反馈控制+校准** |
| **发挥(1)** | 对未知RLC电路学习建模，2分钟内完成，显示滤波类型 | 5分 | **系统辨识** |
| **发挥(2)** | 输入信号经探究装置生成与未知电路相同的输出，误差≤10% | 40分 | **数字滤波器实时实现** |

## 二、技术原理深度解析

### 2.1 已知模型电路分析

**传递函数**：
$$H(s) = \frac{5}{10^{-8}s^2 + 3\times10^{-4}s + 1} = \frac{5}{(\frac{s}{\omega_n})^2 + 2\zeta(\frac{s}{\omega_n}) + 1}$$

**标准二阶系统参数**：
- 自然频率：$\omega_n = \frac{1}{\sqrt{10^{-8}}} = 10^4$ rad/s → $f_n = \frac{10^4}{2\pi} \approx 1.59$ kHz
- 阻尼系数：$2\zeta/\omega_n = 3\times10^{-4}$ → $\zeta = \frac{3\times10^{-4} \times 10^4}{2} = 1.5$
- **ζ=1.5 > 1**：过阻尼系统

**幅频特性**：
$$|H(j\omega)| = \frac{5}{\sqrt{(1 - (\omega/\omega_n)^2)^2 + (2\zeta\omega/\omega_n)^2}}$$

**直流增益**：$|H(0)| = 5$ (14dB)

**截止频率**（-3dB点）：
- 过阻尼系统的截止频率 ≈ ωn = 10^4 rad/s ≈ 1.59kHz
- 但更精确的计算：令 |H(jω)| = 5/√2，解得 ωc ≈ 0.64ωn ≈ 6.4k rad/s ≈ **1.02kHz**

**电路实现**：
这是一个二阶低通滤波器，可用**Sallen-Key结构**或**多重反馈(MFB)**实现。

**Sallen-Key实现**：
```
        R1      R2
    ━━/\/\━━/\/\━━━→ (-) 运放
    │             │
    C1            C2
    │             │
    ━━━━━━━━━━━━━━→ (+)
```

对于增益=5的Sallen-Key低通：
- 选择 R1=R2=10kΩ, C1=C2=10nF 可得 ωn≈1/(RC)=10k rad/s (接近)
- 需要精确调整元件值以匹配传递函数

### 2.2 探究装置信号源

**DDS正弦波产生**：
- 使用DDS芯片（AD9833/AD9850）或MCU直接合成
- 频率范围：100Hz ~ 1MHz
- 步进：100Hz
- 精度要求：≤5%

**幅度控制**：
- 使用PGA/VGA（AD8367）或DAC+乘法器
- 需要根据H(jω)计算所需输入幅度

### 2.3 自动控制算法

**基本原理**：
已知电路传递函数 $H(s)$，要求输出 $V_{out,pp} = 2V$。

则输入幅度应为：
$$V_{in,pp} = \frac{V_{out,pp}}{|H(j\omega)|}$$

**控制流程**：
1. 设置输出频率 f
2. 计算 $|H(j2\pi f)|$
3. 计算所需输入幅度 $V_{in} = V_{target} / |H(j2\pi f)|$
4. 设置信号源输出幅度
5. （可选）测量实际输出，微调输入幅度

### 2.4 未知电路学习建模（系统辨识）

**核心问题**：给定一个RLC二端口网络，通过测量其频率响应，确定滤波器类型和传递函数。

**学习算法**：
1. **频率扫描**：输出不同频率的正弦波，测量输入/输出幅度比和相位
2. **Bode图绘制**：记录幅频和相频特性
3. **类型识别**：
   - 低频增益高、高频衰减 → 低通
   - 低频衰减、高频增益高 → 高通
   - 中间频段增益高、两边衰减 → 带通
   - 中间频段衰减、两边增益高 → 带阻
4. **参数估计**：拟合标准二阶传递函数

**RLC电路传递函数形式**：

| 类型 | 传递函数形式 | 特征 |
|------|-------------|------|
| **低通** | $H(s) = \frac{1}{LCs^2 + RCs + 1}$ | 低频通过，高频衰减 |
| **高通** | $H(s) = \frac{LCs^2}{LCs^2 + RCs + 1}$ | 高频通过，低频衰减 |
| **带通** | $H(s) = \frac{RCs}{LCs^2 + RCs + 1}$ | 窄带通过 |
| **带阻** | $H(s) = \frac{LCs^2 + 1}{LCs^2 + RCs + 1}$ | 窄带阻断 |

**参数范围**：
- R: 1kΩ ~ 10kΩ
- L: 1mH ~ 10mH
- C: 10nF ~ 100nF

**自然频率范围**：
$$\omega_n = \frac{1}{\sqrt{LC}}$$
- 最小：$1/\sqrt{10\text{mH} \times 100\text{nF}} = 1/\sqrt{10^{-6}} = 1000$ rad/s ≈ 159Hz
- 最大：$1/\sqrt{1\text{mH} \times 10\text{nF}} = 1/\sqrt{10^{-8}} = 10000$ rad/s ≈ 1.59kHz

### 2.5 实时信号生成（数字滤波器）

**问题**：已知输入信号x(t)和传递函数H(s)，如何实时生成y(t) = H(s)[x(t)]？

**解决方案——数字滤波器**：
1. 将H(s)离散化为H(z)（双线性变换）
2. 用差分方程实时计算输出

**双线性变换**：
$$s = \frac{2}{T_s} \frac{1 - z^{-1}}{1 + z^{-1}}$$

**二阶IIR滤波器差分方程**：
$$y[n] = b_0 x[n] + b_1 x[n-1] + b_2 x[n-2] - a_1 y[n-1] - a_2 y[n-2]$$

### 3.2 关键电路设计

#### 3.2.1 已知模型电路（Sallen-Key低通）

**目标传递函数**：$H(s) = \frac{5}{10^{-8}s^2 + 3\times10^{-4}s + 1}$

**Sallen-Key结构实现增益=5的二阶低通**：

```
              R1        R2
Vin ──→ ──/\/\──/\/\──┬──→ Vout
                      │
                     C1
                      │
    ━━━━━━━━━━━━━━━━━━┷━━━━━━━━━━━━━━
                      │
                     C2
                      │
                     GND
```

Wait, standard Sallen-Key uses an op-amp. Let me describe it properly.

**标准Sallen-Key低通（增益=1+R4/R3）**：
```
              R1        R2
Vin ──→ ──/\/\──/\/\──┬──→ (-)OpAmp──→ Vout
                      │        │
                     C1       R3
                      │        │
    ━━━━━━━━━━━━━━━━━━┷━━━━━━━━┷━━━━━━━→ (+)
                      │
                     C2
                      │
                     GND
```

Actually, a simpler approach: use twoRC低通级联 + 增益放大。

**两级RC低通+放大实现**：
- 第一级：RC低通，R=15kΩ, C=10nF, fc≈1.06kHz
- 第二级：RC低通，R=15kΩ, C=10nF, fc≈1.06kHz
- 放大级：同相放大，增益=5
- 总传递函数近似二阶低通

## 四、关键设计难点与解决方案

### 4.1 难点1：精确实现H(s)

**问题**：需要精确匹配 $H(s) = \frac{5}{10^{-8}s^2 + 3\times10^{-4}s + 1}$

**解决方案**：
- 使用Sallen-Key标准结构，查表计算元件值
- 或直接用两级RC+放大器近似
- 实际元件有误差（5%），需要微调或软件校准

### 4.2 难点2：未知电路的系统辨识

**问题**：2分钟内识别RLC电路类型和参数

**解决方案**：
- 快速频率扫描：扫描10-20个频点
- 每个频点测量仅需几个周期（如1kHz信号，10ms即可测稳态）
- 总扫描时间 < 1分钟
- 用FFT或幅度检测快速测量

### 4.3 难点3：实时数字滤波

**问题**：实时计算IIR滤波器，要求无失真

**解决方案**：
- STM32H7的FPU可轻松处理二阶IIR
- 采样率：≥100kSPS（覆盖50kHz信号）
- 使用双精度或定点数避免量化误差

### 4.4 难点4：信号类型匹配

**问题**：输入是矩形波和其他周期信号，如何生成匹配输出？

**解决方案**：
- 对任意周期信号，用FFT分解为谐波
- 每个谐波分量通过H(jω)得到输出谐波
- 叠加生成输出波形
- 或直接采样输入，通过数字滤波器实时处理

## 五、评分策略与失分陷阱

### 5.1 失分陷阱

| 陷阱 | 后果 | 避免方法 |
|------|------|---------|
| **已知电路H(s)误差大** | 基础要求(1)失分 | 精确计算元件值，预留微调 |
| **DDS频率精度不够** | 基础要求(2)失分 | 使用晶振基准，校准DDS |
| **未实现幅度自动控制** | 基础要求(3)(4)失分 | 必须实现基于H(jω)的计算 |
| **未知电路识别错误** | 发挥(1)失分 | 多做几种类型识别验证 |
| **实时滤波有延迟/失真** | 发挥(2)失分 | 使用IIR最小相位实现 |

### 5.2 得分策略

**基础分 (50分)**:
- 已知电路实现 (14分) — **核心**
- 信号源产生 (10分) — **简单**
- 幅度控制2Vpp (10分) — **核心**
- 幅度可调1-2Vpp (16分) — **核心**

**发挥分 (50分)**:
- 未知电路识别 (5分) — **建议做**
- 信号匹配输出 (40分) — **分值高，建议重点做**
- 其他 (5分) — 如：
