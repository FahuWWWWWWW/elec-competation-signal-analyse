# 2025年电赛F题「简易自动接收机」产业视角与技术溯源

## 一、技术定位：无线通信的"最后一公里"

### 1.1 产业分类

简易自动接收机属于**超外差接收机(Superheterodyne Receiver)** 技术范畴，这是现代无线通信系统（广播、通信、雷达）中最广泛使用的接收机架构。

```
无线接收技术
├── 直检式/零拍接收机 (Direct Conversion)
│   └── 简单，成本低，但镜像抑制差
├── 超外差接收机 (Superheterodyne)
│   ├── 广播收音机 ←── F题所在层次
│   ├── 移动通信基站
│   ├── 卫星通信地面站
│   └── 军用电子战接收机
├── 数字中频接收机 (Digital IF)
│   └── ADC后直接数字下变频
└── 软件定义无线电 (SDR)
    └── 全数字化处理，最灵活
```

### 1.2 产业链位置

| 层级 | 参与者 | 代表企业/产品 |
|------|--------|-------------|
| **上游** | 射频芯片（LNA、Mixer、VGA） | NXP, ADI, TI, Skyworks |
| **中游** | 接收机模块/收音机 | Sony, Sangean, Tecsun |
| **下游** | 消费电子、通信设备 | 手机厂商、汽车电子 |

## 二、技术溯源与演进

### 2.1 历史发展

| 年代 | 技术阶段 | 特征 |
|------|---------|------|
| **1918** | 超外差发明 | Edwin Armstrong发明超外差接收机，解决高频率接收难题 |
| **1930s** | 调频广播(FM)诞生 | Armstrong再次发明FM调制，抗干扰优于AM |
| **1950s** | 晶体管收音机普及 | 超外差结构小型化，成为消费电子产品 |
| **1980s** | 集成电路接收机 | 单片FM收音机芯片（如CXA系列）出现 |
| **2000s** | 数字调谐收音机 | DSP控制，自动搜索，频率合成(PLL) |
| **2010s** | SDR软件定义无线电 | 通用硬件+软件处理，一颗芯片覆盖多制式 |

### 2.2 FM广播接收机演进

**关键里程碑**：
1. **TDA7000 (Philips, 1980s)**：首款单片FM收音机芯片，集成LNA+混频+IF+鉴频
2. **SA612/NE602 (Signetics, 1990s)**：经典FM接收前端，至今仍被业余无线电爱好者使用
3. **Si470x (Silicon Labs, 2000s)**：数字调谐FM收音机芯片，I2C控制，数字RSSI
4. **RTL-SDR (2012)**：廉价DVB-T电视棒改造为SDR，开启业余无线电SDR热潮

## 四、关键技术深度解析

### 4.1 超外差原理的数学基础

**混频（下变频）**：

$$V_{IF}(t) = V_{RF}(t) \cdot V_{LO}(t) = A\cos(\omega_{RF}t) \cdot B\cos(\omega_{LO}t)$$

$$= \frac{AB}{2}[\cos((\omega_{RF}-\omega_{LO})t) + \cos((\omega_{RF}+\omega_{LO})t)]$$

**中频滤波器**提取差频分量：$\omega_{IF} = \omega_{RF} - \omega_{LO}$

**镜像频率**：
$$f_{image} = f_{LO} - f_{IF} = f_{RF} - 2f_{IF}$$

对于FM广播（IF=10.7MHz）：
- 接收98MHz → LO=108.7MHz → 镜像=119.4MHz（超出频段，易滤除）
- 高端注入天然抑制镜像

### 4.2 灵敏度与噪声系数

**接收机灵敏度公式**：

$$P_{min} = kT \cdot B \cdot NF \cdot SNR_{req}$$

其中：
- $kT$ = -174dBm/Hz (热噪声功率谱密度)
- $B$ = 信号带宽 (FM约200kHz → 53dBHz)
- $NF$ = 整机噪声系数 (dB)
- $SNR_{req}$ = 解调所需信噪比 (FM约10dB)

**计算F题-95dBm灵敏度要求**：

$$P_{min} = -174 + 53 + NF + 10 = -111 + NF$$

要求 $P_{min} \leq -95$dBm：
$$-111 + NF \leq -95 \Rightarrow NF \leq 16\text{dB}$$

**容易满足！** 典型超外差接收机NF=5-10dB。

**实际限制**：
- LNA的NF决定整机NF（Friis公式）
- 若LNA NF=2dB, Gain=20dB，后续级联NF贡献很小
- 整机NF ≈ 2 + (后续级NF)/100 ≈ 2-3dB

### 4.3 FM鉴频器原理

**正交鉴频器(Quadrature Detector)**：

```
FM IF信号 ──→ [限幅器] ──→ [相移网络(90°@f0)] ──→
    │                                                  ├──→ [乘法器] ──→ [低通] → 音频
    └──────────────────────────────────────────────────→
```

**原理**：
- 相移网络对中心频率相移90°，对频偏产生额外相移
- 乘法器输出与相移成正比，即与频偏成正比
- 输出电压：$V_{out} \propto \Delta f$

**CD4046 PLL鉴频**：
- PLL锁定在10.7MHz载波
- 调制信号导致VCO控制电压变化
- VCO控制电压 = 解调输出

### 4.4 AGC环路分析

**AGC环路方程**：

$$V_{ctrl} = G_{det} \cdot \int (V_{ref} - V_{det}) dt$$

$$G_{VGA} = G_{max} - K \cdot V_{ctrl}$$

**稳定性条件**：
- 环路带宽 << 最低调制频率 (300Hz)
- 避免AGC跟踪音频信号
- 通常AGC环路截止频率 < 50Hz

## 七、前沿研究方向

### 7.1 软件定义无线电(SDR)

- 将ADC直接放在RF或IF，后续全数字处理
- RTL-SDR就是此思想的廉价实现
- 但电赛不允许使用现成SDR模块

### 7.2 认知无线电(Cognitive Radio)

- 自动感知频谱占用情况
- 动态选择空闲频段通信
- 与F题的"自动搜索"思想一致

### 7.3 零中频(Direct Conversion)接收机

- RF直接下变频到基带，无IF
- 省去中频滤波器，但存在直流偏移和I/Q不平衡问题
- 现代手机普遍使用
