# 2025年电赛F题「简易自动接收机」题目详解

## 一、题目核心要求拆解

### 1.1 任务概述

设计并制作一台**简易自动接收机**，工作于88-108MHz FM广播频段：
- **自动搜索**：扫描88-108MHz，步进100kHz
- **自动识别**：区分单频载波、FM、AM信号
- **自动解调**：解调FM或AM信号
- **AGC控制**：输出幅度自动稳定在1V±0.1V
- **灵敏度**：基本-60~-85dBm，发挥≤-95dBm(AM)

### 1.2 关键指标一览

| 要求项 | 具体指标 | 分值 | 考核本质 |
|--------|---------|------|---------|
| **基本要求(1)** | FM解调，Δf=5~75kHz，输出≥0.9Vpp | 15分 | **FM接收链路+鉴频** |
| **基本要求(2)** | AM解调，m=30~60%，输出≥0.9Vpp | 15分 | **AM接收链路+包络检波** |
| **基本要求(3)** | AGC控制：输出1V±0.1V | 10分 | **自动增益控制环路** |
| **基本要求(4)** | 响应时间≤10s | 10分 | **搜索速度+解调收敛** |
| **发挥(1)** | 自动识别单频/FM/AM，显示结果 | 23分 | **调制识别算法** |
| **发挥(2)** | AM灵敏度≤-95dBm | 12分 | **低噪声前端+高增益** |
| **发挥(3)** | 响应时间≤5s | 10分 | **快速搜索+快速AGC** |

## 二、技术原理深度解析

### 2.1 超外差接收机架构

**经典超外差(Superheterodyne)结构**：

```
天线/输入 → [LNA] → [Mixer] → [IF Filter] → [IF Amp] → [Demodulator] → [Audio Amp] → 输出
                ↑        ↑                                          ↑
                │   [LO/PLL]                                    [AGC]
                │        │                                          │
                └────────┘← 频率合成器(DDS/PLL/VCO)                 │
                                                                     │
[MCU控制] ←─────────────────────────────────────────────────────────┘
```

**信号流**：
1. **射频输入** (88-108MHz, -95~-60dBm)
2. **LNA放大** (低噪声放大，增益20-30dB)
3. **下变频** (混频器：RF + LO → IF)
4. **中频滤波** (10.7MHz陶瓷滤波器，带宽±150kHz)
5. **中频放大** (固定增益60-80dB)
6. **解调** (FM鉴频 / AM包络检波)
7. **音频放大** (驱动8Ω负载)

### 2.2 频率规划

**中频选择**：
- 传统FM收音机使用 **10.7MHz** 中频
- 本振频率：$f_{LO} = f_{RF} + 10.7MHz$ (高端注入)
- 当接收98MHz时：$f_{LO} = 108.7MHz$

**镜像频率**：
- $f_{image} = f_{RF} + 2 \times IF = f_{RF} + 21.4MHz$
- 对于98MHz信号，镜像在119.4MHz（超出频段，易被滤除）

### 2.3 关键模块设计

#### 2.3.1 LNA（低噪声放大器）

| 参数 | 要求 | 设计要点 |
|------|------|---------|
| **增益** | ≥20dB | 补偿后续损耗 |
| **NF** | ≤3dB | 决定整机灵敏度 |
| **带宽** | 88-108MHz | 覆盖FM广播频段 |
| **IP3** | ≥-10dBm | 抗大信号阻塞 |

**推荐器件**：
- BFP740（NXP）：NF=1.1dB@1GHz, Gain=19dB
- SAV-541+（Mini-Circuits）：NF=1.5dB, Gain=20dB

#### 2.3.2 混频器

| 类型 | 优点 | 缺点 | 适用性 |
|------|------|------|--------|
| **无源二极管** | 线性度高，IP3好 | 变频损耗大(6dB) | 高性能接收机 |
| **有源Gilbert** | 增益高，集成方便 | 噪声稍大 | 集成方案 |
| **集成混频器** | 简单，稳定 | 参数固定 | **电赛推荐** |

**推荐器件**：
- ADL5350（ADI）：变频增益5dB, NF=9.8dB
- SA602/NE602：经典FM接收芯片，含LNA+混频+OSC

#### 2.3.3 中频滤波器

**10.7MHz陶瓷滤波器**是FM接收的标准器件：
- 带宽：±150kHz (300kHz总带宽)
- 插入损耗：4-6dB
- 带外抑制：>40dB@±300kHz

#### 2.3.4 FM鉴频器

**方案1：正交鉴频器(Quadrature Detector)**
- 最常用的FM解调方式
- 原理：将FM信号相移90°，与原信号相乘，输出与频偏成正比
- 器件：CD4046 PLL芯片可工作在10.7MHz

**方案2：斜率鉴频器(Slope Detector)**
- 用LC谐振回路失谐，将频率变化转为幅度变化
- 简单但线性度差

**方案3：PLL鉴频**
- 用PLL跟踪载频变化，控制电压即解调信号
- 灵敏度高，但捕获范围有限

#### 2.3.5 AM包络检波器

**二极管包络检波**：
- 最简单，但失真大，不适合低载波电平

**同步检波**：
- 用恢复的载波与信号相乘
- 失真小，但需要载波恢复电路

**电赛推荐**：二极管检波+AGC控制（因为载波电平变化范围大）

### 2.4 AGC环路设计

**自动增益控制(AGC)环路**：

```
解调输出 → [检波(幅度检测)] → [低通滤波] → [误差放大] → [VGA控制电压]
                                                              ↓
IF信号 → [VGA] ─────────────────────────────────────────────────→ [输出]
```

**VGA选择**：
- AD8367：500MHz带宽, 0-40dB增益控制
- VCA810：DC-35MHz, -40~+40dB

**AGC时间常数**：
-  attack时间：1-10ms（快速响应信号增强）
-  decay时间：100ms-1s（慢速恢复，避免增益抖动）

### 2.5 信号类型自动识别

**三种信号特征对比**：

| 特征 | 单频载波(CW) | FM | AM |
|------|-------------|----|----|
| **频谱宽度** | 窄（单根线） | 宽（~200kHz） | 窄（载波+2边带） |
| **幅度变化** | 恒定 | 恒定 | 随调制变化 |
| **过零率** | 低 | 中 | 中 |
| **频谱对称性** | 无 | 对称 | 对称 |

**识别算法**：
1. 扫描到某频率，检测信号强度(RSSI)
2. 若RSSI > 阈值，进行类型分析
3. 分析IF输出：
   - 测量信号带宽（FM宽，AM窄，CW极窄）
   - 测量包络变化（AM有包络变化，FM/CW恒定）
   - 测量频谱对称性
4. 判决输出类型

## 三、系统架构设计

### 3.1 完整接收机链路

```
┌─────────────────────────────────────────────────────────────────┐
│                        简易自动接收机                             │
│                        (88-108MHz FM/AM)                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [SMA输入] → [带通滤波] → [LNA] → [Mixer] → [10.7MHz陶瓷滤波] → [IF Amp] → [VGA] → [Demod]
│      50Ω        88-108MHz    +20dB     +LO     ±150kBW       +60dB    AGC控制  FM/AM
│                              (BFP740)  (Si5351)              (多级)    │      切换
│                                                                           │
│  ┌────────────────────────────────────────────────────────────────────────┘
│  │
│  │  [Demodulator]
│  │    ├── FM模式: [正交鉴频器/CD4046] → [音频低通] → [音频功放]
│  │    └── AM模式: [包络检波器] → [音频低通] → [音频功放]
│  │
│  │  [Audio Amp] (驱动8Ω负载)
│  │    ├── 增益：~20dB
│  │    └── 输出功率：≥(1Vpp/2√2)²/8Ω = 31mW
│  │
│  │  [AGC环路]
│  │    ├── 检波：幅度检测
│  │    ├── 积分：RC低通(attack/decay)
│  │    └── 控制：VGA电压
│  │
│  └─→ [MCU: STM32H7]
│        ├── 控制Si5351频率扫描
│        ├── 采样RSSI（信号强度指示）
│        ├── 信号类型识别算法
│        ├── 自动切换FM/AM解调
│        ├── 控制显示(LCD/OLED)
│        └── 按键/一键启动
│
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 增益预算分析

| 模块 | 增益/损耗 | 电平(dBm) |
|------|----------|----------|
| **输入(最低)** | — | -85dBm |
| **LNA** | +20dB | -65dBm |
| **Mixer** | -6dB | -71dBm |
| **IF滤波器** | -5dB | -76dBm |
| **IF Amp** | +70dB | -6dBm |
| **VGA(AGC)** | 0~-40dB | -6~-46dBm |
| **解调损耗** | -3dB | -9~-49dBm |
| **音频功放** | +20dB | +11~-29dBm |
| **输出(8Ω)** | — | ≥0.9Vpp |

0.9Vpp into 8Ω = 0.32Vrms → P = 0.0125W = 12.5mW → +11dBm

**结论**：IF增益70dB是必要的，且需要AGC防止大信号过载。

## 四、关键设计难点与解决方案

### 4.1 难点1：-95dBm灵敏度

**-95dBm = 31.6pW = 1.26μV (50Ω)**

**要求总噪声系数**：
- 解调器需要约+11dBm输出
- 从-95dBm到+11dBm需要106dB增益
- 考虑解调器信噪比要求(SNR>10dB)
- 要求整机NF < 8dB

**实现方案**：
- LNA的NF < 2dB（BFP740满足）
- 混频器NF < 10dB
- IF级联增益 > 80dB（多级放大）

### 4.2 难点2：响应时间≤5s

**搜索88-108MHz，步进100kHz = 200个频点**

**时间分配**：
- 每个频点驻留时间：5s/200 = 25ms
- 25ms内需要：调谐→检测RSSI→判断→（若信号）解调→AGC稳定

**优化方案**：
- 使用硬件RSSI（SA612内部RSSI输出）
- AGC用模拟环路（快于数字AGC）
- 信号检测用阈值比较器（快于ADC采样）
- 先快速扫描找信号，再精细解调

### 4.3 难点3：5V单电源，≤500mA

**功率预算**：
- 总功率 = 5V × 0.5A = 2.5W
- LNA：50mA × 5V = 250mW
- 混频器：20mA × 5V = 100mW
- IF放大器：30mA × 5V = 150mW
- 音频功放：100mA × 5V = 500mW（Class AB）
- 其他：200mW
- **总计**：约1.2W < 2.5W ✅

### 4.4 难点4：自动识别FM/AM/CW

**实用算法**：
1. 扫描到信号后，先测量RSSI
2. 用ADC采样IF输出，做FFT：
   - 带宽>150kHz → FM
   - 带宽<30kHz且有边带 → AM
   - 带宽<1kHz → CW
3. 测量包络变化：
   - 包络变化明显 → AM
   - 包络恒定 → FM或CW
4. 结合两个特征判决

## 五、评分策略与失分陷阱

### 5.1 失分陷阱

| 陷阱 | 后果 | 避免方法 |
|------|------|---------|
| **不用超外差** | 灵敏度差，选择性差 | 必须用超外差结构 |
| **无AGC** | 输出幅度不稳 | 必须实现模拟或数字AGC |
| **响应时间>10s** | 直接扣分 | 优化搜索算法，用硬件RSSI |
| **灵敏度不够** | -95dBm拿不到 | 低噪声LNA + 足够IF增益 |
| **输出失真** | 解调输出波形失真 | 音频滤波器带宽300-3400Hz |

### 5.2 得分策略

**基础分 (50分)**:
- FM解调 (15分) — **核心**
- AM解调 (15分) — **核心**
- AGC控制 (10分) — **必做**
- 响应时间 (10分) — **优化搜索速度**

**发挥分 (50分)**:
- 自动识别 (23分) — **建议做，分值高**
- 高灵敏度 (12分) — **需要低噪声设计**
- 快速响应 (10分) — **与基础响应时间配合**
- 其他 (5分) — 如：
  - 显示信号强度柱状图
  - 存储频率频谱
  - 显示调制参数（频偏/调制度）
