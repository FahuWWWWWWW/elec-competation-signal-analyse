### 2. 前端调理电路

#### 2.1 射频前端

| 模块 | 器件 | 参数 |
|------|------|------|
| 天线输入 | SMA 50Ω | MP双脊天线或1/4λ鞭状天线 |
| 带通滤波 | 3阶LC巴特沃斯 | 88-108MHz, 插入损耗<2dB, 带外抑制>30dB@±20MHz |
| LNA | BFP740 + 偏置网络 | Gain=19dB@100MHz, NF=1.1dB, IP3=+8dBm |
| 混频器 | SA612A | 变频增益14dB, NF<5dB, LO驱动-10dBm |
| 本振 | Si5351A (CLK0) | 98.7-118.7MHz, 步进100kHz, 相位噪声-130dBc/Hz@10kHz |

**前置带通滤波器设计**（3阶巴特沃斯）：
中心频率 fc = 98MHz, BW = 22MHz (88-108MHz)

归一化LC值（3阶巴特沃斯，50Ω系统）：
$$L_1 = \frac{50 \times 1.5}{2\pi \times 98\text{MHz}} = 121.7\text{nH}, C_1 = \frac{1.333}{2\pi \times 98\text{MHz} \times 50} = 43.3\text{pF}$$
$$L_2 = \frac{50 \times 1.5}{2\pi \times 98\text{MHz}} = 121.7\text{nH}, C_2 = \frac{0.5}{2\pi \times 98\text{MHz} \times 50} = 16.2\text{pF}$$

#### 2.2 IF放大链路

| 模块 | 器件 | 增益 | 说明 |
|------|------|:----:|------|
| 陶瓷滤波器 | 10.7MHz ±150kHz | -5dB | 村田SFEC10.7M |
| IF前级 | BFR92 + 谐振负载 | +30dB | 共射放大, 10.7MHz谐振 |
| VGA | VCA810 | -40~+40dB | DC-35MHz, 线性dB控制 |
| IF后级 | BFR92 ×2 | +40dB | 两级共射放大 |
| 缓冲 | OPA847 | +6dB | 驱动ADC/CD4046 |

**总IF增益**：-5+30+40+6 = 71dB，满足增益预算要求。

#### 2.3 FM鉴频模块

实现方式：ADC采样10.7MHz IF信号 + MCU数字鉴频。

ADC采样率选择：对10.7MHz IF信号，直接采样需fs > 2×10.7MHz = 21.4MSPS。H723内置ADC最高3.6MSPS，不足。

**修正方案**：下变频到低IF。使用次级混频（SA612A第二级）将10.7MHz降到455kHz：
- 第二本振：Si5351 CLK1 → 10.245MHz
- 第二IF：10.7 - 10.245 = 455kHz
- ADC采样率：3.6MSPS > 2×455kHz = 910kHz ✅

数字鉴频实现（在MCU中）：
$$f_{dev} = \frac{\Delta \phi}{2\pi \cdot \Delta t} \times f_s$$

或者采用过零检测法统计单位时间过零次数。

#### 2.4 AM包络检波

| 模块 | 器件 | 参数 |
|------|------|------|
| 检波二极管 | BAT54S | Schottky, Vf=0.3V, Ct=10pF |
| 低通滤波 | RC 1阶, fc=15kHz | R=10kΩ, C=1nF |
| 缓冲 | OPA365 | BW=50MHz, SR=25V/μs |
| ADC | H723 ADC2 (16-bit) | 采样率48kSPS |

### 3. 后处理与算法

#### 3.1 FM解调算法

```
ADC @ 3.6MSPS → 数字下变频(455kHz→基带) → CORDIC鉴相 → 差分→音频
```

| 步骤 | 方法 | 说明 |
|------|------|------|
| DDC | 数字混频+NCO | 455kHz→基带I/Q两路 |
| 低通 | FIR 30阶, fc=100kHz | 滤除2×IF分量 |
| 鉴相 | CORDIC求φ=atan2(Q,I) | 相位解包裹 |
| 鉴频 | φ差分→f_dev | f_dev = Δφ/Δt × fs/(2π) |
| 去加重 | 单极点HPF | τ=50μs, FM预加重补偿 |
| 音频 | 低通FIR fc=15kHz | 输出音频 |

#### 3.2 AM解调算法

```
ADC @ 48kSPS → 绝对值 → LPF → DC分量去除 → 音频输出
```

或数字包络检波：`env[n] = sqrt(I[n]^2 + Q[n]^2)`

#### 3.3 自动搜索算法

```
1. 设置Si5351频率 = 88MHz + 10.7MHz = 98.7MHz
2. 等待PLL锁定 + IF建立 (5ms)
3. 读取RSSI (ADC采样VGA控制电压或检波输出) (1ms)
4. 若RSSI > 阈值:
   a. 采样IF信号 (10ms, 36000点@3.6MSPS)
   b. FIR带宽分析 → 窄/FM/AM识别
   c. IF带宽>150kHz → FM, 否则AM/CW
   d. 包络变化检测 → AM识别
   e. 存储频率+类型
5. 步进100kHz → 下一频点
6. 200频点×17ms = 3.4s < 5s ✅
```

#### 3.4 AGC算法

数字AGC控制VCA810：
```c
Verr = Vtarget - Vrms_measured;  // 误差
Vctrl += Kp * Verr + Ki * integral(Verr);  // PI控制
// Attack: Kp=0.1, Ki=0.01
// Decay: Kp=0.01, Ki=0.001
```

### 4. 关键参数验证

#### 4.1 灵敏度验证（-95dBm）

接收机总噪声系数NF（级联公式）：
$$NF_{total} = NF_1 + \frac{NF_2 - 1}{G_1} + \frac{NF_3 - 1}{G_1G_2} + ...$$

| 级 | 器件 | Gain(dB) | NF(dB) |
|:--:|------|:--------:|:------:|
| 1 | 带通滤波+PCB损耗 | -2 | 2 |
| 2 | BFP740 LNA | +19 | 1.1 |
| 3 | SA612A混频器 | +14 | 5 |
| 4 | 陶瓷滤波器 | -5 | 5 |
| 5 | BFR92 IF放大 | +30 | 3 |

$$NF_{total} = 2 + \frac{1.26 - 1}{0.63} + \frac{3.16 - 1}{0.63 \times 79.4} + \frac{3.16 - 1}{0.63 \times 79.4 \times 25.1} + \frac{2 - 1}{0.63 \times 79.4 \times 25.1 \times 0.316}$$

$$= 2 + 0.41 + 0.043 + 0.0054 + 0.0025 = 2.46 \text{dB}$$

-95dBm灵敏度所需最小SNR（FM解调需SNR>12dB）：
$$P_{min} = -174\text{dBm/Hz} + 10\log_{10}(200\text{kHz}) + NF_{total} + SNR_{min}$$
$$= -174 + 53 + 2.46 + 12 = -106.5\text{dBm}$$

Sensitivity @ -95dBm要求：margin = 106.5 - 95 = 11.5dB ✅

#### 4.2 IF选择性验证

10.7MHz陶瓷滤波器 SFEC10.7M:
- 中心频率: 10.70MHz ±30kHz
- 带宽: ±150kHz (-6dB)
- 邻频选择性: >40dB @ ±300kHz

考虑频偏75kHz FM信号，带宽150kHz，落在滤波器通带内：
- 中心频率偏差: 75kHz < 150kHz ✅
- 边带频率分量全部在300kHz通带内 ✅

#### 4.3 AGC控制精度

目标输出1V±0.1V。VCA810控制特性：Vctrl=0~-1V对应增益-40~+40dB，线性斜率 = 80dB/V。

ADC测量RMS：16-bit, Vref=3.3V
LSB = 3.3V/65536 = 50.3μV

AGC稳态误差：
$$\Delta V_{ctrl} = \frac{\Delta V_{out}}{V_{out}} \times \frac{1}{S_{VCA}} = \frac{0.1V}{1V} \times \frac{1}{80\text{dB/V}} = 1.25\text{mV}$$

RSSI ADC 16-bit分辨率可分辨50.3μV << 1.25mV ✅

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| 单MCU方案，系统简洁 | 需要次级混频到455kHz才能ADC采样 |

### 2. 前端调理电路

#### 2.1 射频前端

同方案A，优化LNA偏置：

BFP740偏置设计：
- Vce = 3V, Ic = 10mA
- R1 (基极偏置) = (3V - 0.7V) / (10mA/200) = 46kΩ
- R2 (发射极负反馈) = 0V (使用电流源偏置)

输入匹配网络：L=22nH, C=5pF → 50Ω匹配 @ 100MHz。

#### 2.2 CD4046 PLL鉴频器

CD4046在10.7MHz工作的关键参数：

| 参数 | 值 | 说明 |
|------|------|------|
| VCO最高频率 | 1.4MHz@Vdd=5V | ❌ 不足10.7MHz |
| 修正方案 | 使用CD74HC4046 | 最高21MHz@Vdd=5V ✅ |

**CD74HC4046鉴频参数**：
- VCO中心频率: f0 = 10.7MHz
- 频率捕获范围: ±150kHz
- 鉴频灵敏度: Kd = Vdd/4π ≈ 0.4V/rad

鉴频输出：
$$V_{out} = K_d \times 2\pi \times \Delta f \times R_t C_t$$

当Δf=75kHz, Rt=10kΩ, Ct=1nF：
$$V_{out} = 0.4 \times 2\pi \times 75\text{kHz} \times 10\text{k}\Omega \times 1\text{nF} = 1.88\text{V} \text{(峰峰值)} \checkmark$$

#### 2.3 AGC环路（模拟）

```
IF输入→VCA810→IF放大→CD4046/检波→AD8361→OPA365积分→VCA810控制
                                               ↑
                                        DAC/Vref设定目标幅度
```

**VCA810**: -40~+40dB线性dB控制(Vc=0~-1V), BW=35MHz @ G=+40dB。
**攻击/释放时间**：
- Attack: τ_a = R_a × C = 1kΩ × 1μF = 1ms
- Decay: τ_d = R_d × C = 220kΩ × 1μF = 220ms

### 3. 后处理与算法

#### 3.1 自动搜索流程（效率优化）

```
主扫描 (快速):
  for f = 88MHz to 108MHz step 200kHz:
   设置Si5351 → 延时2ms → 读RSSI(ADC) → 记录RSSI表

精细扫描:
  for f in RSSI峰值 > 阈值:
   设置频率 → 延时5ms → 读RSSI → 确认信号
   → 切换解调模式 → 测量带宽 → 判决调制类型 → 存储

总时间估算:
  101频点 × 2ms = 202ms (快速扫描)
  + 6电台 × 15ms = 90ms (精细扫描)
  = 292ms << 5s ✅
```

#### 3.2 调制识别算法

| 特征 | 测量方法 | FM | AM | CW |
|------|---------|:--:|:--:|:--:|
| IF带宽 | 扫CD4046 VCO频率+测输出 | >150kHz | <30kHz | <1kHz |
| 包络变化 | ADC采样检波输出Vpp/均值比 | <5% | >20% | <2% |
| 频谱对称 | 检查FM正交输出直流偏移 | 对称 | 对称 | 对称 |

判决树：
```
IF带宽 > 150kHz → FM
IF带宽 < 150kHz:
  包络变化 > 10% → AM
  包络变化 < 10%:
    带宽 < 1kHz → CW
    否则 → AM (弱信号)
```

#### 3.3 AGC设定

F407 DAC输出Vref到AGC积分器同相端，设定目标输出电压。
- DAC 12-bit, Vref=3.3V
- 设定值 = 1.0V / 3.3V × 4096 = 1241 (对应输出1V)

### 4. 关键参数验证

#### 4.1 镜像抑制比

镜像频率公式：f_image = f_RF + 2 × IF

RF=88MHz: f_image = 88 + 21.4 = 109.4MHz
RF=108MHz: f_image = 108 + 21.4 = 129.4MHz

109.4MHz处于88-108MHz频段边缘(仅高1.4MHz)，镜像可能进入。

**抑制措施**：
前置带通滤波器3阶巴特沃斯在109.4MHz处衰减：
$$A_{image} = 10 \times \log_{10}\left[1 + \left(\frac{109.4-98}{11}\right)^{2 \times 3}\right] \approx 18.6\text{dB}$$

不足！需要增加一级前置带通或改用SA612A的高镜像抑制特性(SA612内部平衡混频器可提供约20dB镜像抑制)。

总镜像抑制 ≈ 18.6 + 20 ≈ 38.6dB ✅（满足一般接收需求）

#### 4.2 AM -95dBm灵敏度验证

AM解调所需最小信噪比：
- AM包络检波通常需要SNR > 6dB (FM需要更高)
- IF带宽: 陶瓷滤波器±150kHz, 但AM仅需±7.5kHz

窄带IF滤波(AM模式)：使用455kHz陶瓷滤波器(±5kHz)替换10.7MHz滤波器。

采用两次变频：
1. 10.7MHz陶瓷滤波器 ±150kHz (第一IF)
2. 455kHz陶瓷滤波器 ±5kHz (第二IF, AM模式)
3. FM模式跳过第二IF

-95dBm灵敏度验证(AM)：
$$P_{min} = -174 + 10\log_{10}(10\text{kHz}) + 2.46 + 6 = -174 + 40 + 2.46 + 6 = -125.5\text{dBm}$$

margin = 125.5 - 95 = 30.5dB ✅

#### 4.3 CD74HC4046鉴频线性度验证

PLL锁定范围设计：
- VCO增益: Kv = 2π × 300kHz / 1V = 1.88×10^6 rad/V·s
- 相位检测器增益: Kd = Vdd/4π = 5/12.56 = 0.398 V/rad
- 环路滤波器: R1=10kΩ, R2=100kΩ, C=1μF
- 环路自然频率: ωn = sqrt(Kd·Kv/(R1·C)) = sqrt(0.398×1.88×10^6/10^4×10^-6) = 8650 rad/s
- f_n = 1.38kHz
- 阻尼系数: ζ = 0.5×R2/R1×ωn = 0.5×10×8650 = 0.43

FM鉴频输出与频偏线性度在±75kHz内：
$$V_{out} \approx \frac{K_d \cdot K_v}{1 + s/\omega_n} \times \Delta f$$

在f < ωn/2π = 1.38kHz范围内，输出线性度优于0.5% ✅

### 3. 后处理与算法

#### 3.1 数字鉴频（FPGA流水线）

```
I/Q @ 1MSPS → CORDIC鉴相(φ=atan2(Q,I)) → 差分鉴频(f=Δφ/Δt) → 去加重HPF
```

FPGA实现延迟：8个时钟周期 @ 130MHz = 61.5ns

**CORDIC IP核**：Xilinx CORDIC 6.0, 16-bit, 流水线模式, 延迟15clk。
**差分器**：f_dev[n] = φ[n] - φ[n-1], 乘以fs/(2π)得频率偏差。

#### 3.2 自动搜索调度（PS侧）

```
PS (ARM A9 @ 766MHz) 控制流程:

1. 设置Si5351 → SPI写 (10μs)
2. 等待PLL锁定 (100μs)
3. 读取FPGA内RSSI寄存器 (1μs, AXI读)
4. 若RSSI > 阈值:
   - 读取FPGA内1024点FFT结果 (含幅度谱)
   - 分析信号带宽、对称性、包络变化
   - 判决调制类型
   - 存储电台信息
5. 步进100kHz → 下一频点
6. 总搜索时间: 200频点 × 200μs = 40ms << 5s ✅
```

#### 3.3 数字AGC（FPGA+PS协同）

```
FPGA侧: 实时计算I²+Q² → CORDIC求RMS → 与Vtarget比较
PS侧: 每100μs读取FPGA误差 → PI计算 → DAC输出控制VCA810
```

PI参数：
- Kp = 0.05, Ki = 0.005
- 积分器饱和限制: ±1V
- Attack < 1ms, Decay: 由积分器自然衰减

#### 3.4 调制识别（FFT加速）

FPGA内实现1024点FFT @ 130MSPS (或抽取后1MSPS)：
- Xilinx FFT IP 9.1, 1024点, 流水线IO
- 变换时间: 1024/130M = 7.9μs

从FFT幅度谱提取特征：
1. 信号带宽 = 幅度>阈值×频谱宽度
2. 对称性 = 载波左右±75kHz能量比
3. 包络变化 = ADC时域幅度标准差/均值

### 4. 关键参数验证

#### 4.1 数字DDC信噪比

LTC2208 @ 10.7MHz:
- SNR = 77.5dB (典型值, 数据手册)
- SFDR = 98dBc @ fin=10.7MHz

NCO 32-bit量化:
- 相位噪声 = -6.02×32 - 1.76 = -194dBc/Hz << ADC本底噪声

数字混频SNR：
$$\text{SNR}_{DDC} = \text{SNR}_{ADC} + 10\log_{10}(f_s/2B)$$
$$= 77.5 + 10\log_{10}(130MHz/300kHz) = 77.5 + 26.4 = 103.9\text{dB} \gg 12\text{dB} \checkmark$$

#### 4.2 FM鉴频输出信噪比

FM解调输出SNR与输入CNR关系（FM阈值效应）：
$$\text{SNR}_{out} = \text{CNR}_{in} + 10\log_{10}\left(\frac{3}{2} \times \frac{B_{IF}}{f_m} \times \left(\frac{\Delta f}{f_m}\right)^2\right)$$

输入CNR = -95dBm + 106.5dBm = 11.5dB (参考方案A计算)
B_IF = 300kHz, f_m = 15kHz, Δf = 75kHz:

$$\text{SNR}_{out} = 11.5 + 10\log_{10}\left(\frac{3}{2} \times \frac{300}{15} \times \left(\frac{75}{15}\right)^2\right)$$
$$= 11.5 + 10\log_{10}(1.5 \times 20 \times 25) = 11.5 + 10\log_{10}(750) = 11.5 + 28.75 = 40.25\text{dB}$$

远高于可听需求(SNR>20dB) ✅

#### 4.3 镜像抑制（数字域增强）

在FPGA内使用数字镜像抑制混频器结构：

采用Hartley结构(90°相移)在数字域实现镜像抑制：
$$\text{IRR} = \frac{1 + 2(1-2\epsilon)\cos(\theta) + (1-2\epsilon)^2}{1 - 2(1-2\epsilon)\cos(\theta) + (1-2\epsilon)^2}$$

其中ε=幅度误差(0.1%), θ=相位误差(0.5°)：
$$\text{IRR} \approx \frac{1 + 2 \times 0.998 \times 0.99996 + 0.998^2}{1 - 2 \times 0.998 \times 0.99996 + 0.998^2} \approx 66\text{dB}$$

数字域IRR远优于模拟方案 ✅
