### 1.2 核心原理

#### 1.2.1 MCU内置ADC配置

STM32H743内置2个16-bit SAR ADC，最大采样率 **3.6MSPS**（单通道模式）。通过双重ADC交错采样，可实现 **7.2MSPS**。

$$\text{SNR}_{\text{ADC}} = 6.02 \times N + 1.76 \text{ (dB)}$$

- 16-bit 理论 SNR = 98.1 dB
- 实际有效位数 ENOB ≈ 12.5-bit（@100kHz 输入）⇒ SNR ≈ 77 dB
- 等效输入噪声 ≈ 0.15 LSB rms

**ADC速率与分辨率权衡**（H743内置ADC）：

| 采样率 | 分辨率 | ENOB (typ) | 适用场景 |
|--------|--------|------------|---------|
| 3.6 MSPS | 16-bit | 12.5 bit | 2021-A（500kHz信号，≈7×过采样） |
| 7.2 MSPS (交错) | 14-bit (有效) | 11 bit | 2023-D（2MHz载波，3.6×过采样） |
| 1 MSPS | 16-bit | 13.0 bit | 2021-J（50kHz信号，20×过采样） |
| 200 kSPS | 16-bit | 14.0 bit | 2024-B（50Hz，4000×同步采样） |

#### 1.2.2 外扩ADC方案

对于高速需求（TDR），使用外部ADC + FMC并行接口：

- **AD9288**（双通道8-bit，100MSPS，¥25）—— 用于TDR脉冲捕获
- **AD7606**（8通道16-bit，200kSPS同步采样，¥80）—— 用于功率分析/多通道采集
- **THS1206**（单通道12-bit，6MSPS，¥30）—— 中速通用ADC

#### 1.2.3 等效采样原理（覆盖TDR需求）

对周期性信号，等效采样利用**精密延迟线**在多个周期内逐次偏移采样点：

$$t_{\text{eff},k} = t_0 + k \times \Delta t_{\text{eff}}$$

其中 $\Delta t_{\text{eff}} = \frac{1}{f_{\text{trigger}}} - \frac{N}{f_{\text{ADC}}} \approx$ 延迟步进

**2023-B参数验证**：
- 目标：等效采样率 ≥ 4 GSPS → Δt_eff ≤ 0.25 ns
- 延迟步进时钟频率 = 1 / 0.25ns = 4 GHz（用FPGA或专用DLL）
- 使用 **MC100EP195** 可编程延迟线：10-bit精度，步进10ps，范围10ns
- 若每周期采1点，100周期即可构建4GSPS×100点 = 40ns窗口

**精度验证**：
$$\Delta t_{\text{eff}} = \frac{1}{f_{\text{ADC}}} \times \frac{1}{M}$$

其中 $M$ = 等效倍数（延时步进数）。100MSPS ADC × 40倍等效 = 4 GSPS。

$$\text{空间分辨率} = c \times v_f \times \Delta t_{\text{eff}} / 2$$

- 电缆缩短系数 $v_f = 0.67$（CAT5e）
- $\Delta t_{\text{eff}} = 0.25 \text{ ns} \Rightarrow$ 空间分辨率 = $3\times10^8 \times 0.67 \times 0.25\times10^{-9} / 2 = 2.5 \text{ cm}$

### 1.3 超外差前端（覆盖2025-F, 2022-F需求）

使用 **SA612A**（¥5）+ **10.7MHz陶瓷滤波器**（¥1）：

```
天线 → LNA(BFP740) → SA612A混频器 → 10.7MHz陶瓷滤波器 → IF放大 → MCU ADC采样(20MSPS)
                                  ↑
                              Si5351本振(PLL)
```

**频率规划**：
- FM 88-108MHz → LO 98.7-118.7MHz → IF = 10.7MHz
- $f_{\text{image}} = f_{\text{RF}} + 2f_{\text{IF}} = f_{\text{RF}} + 21.4\text{MHz}$
- 镜像抑制：前端LC带通滤波器（BW=20MHz）提供 ~30dB 抑制 → 满足需求

**IF采样要求**：
- 10.7MHz IF信号，带宽 ±150kHz（陶瓷滤波器）
- 奈奎斯特要求：$f_s \geq 2 \times (10.7 + 0.15) = 21.7\text{ MSPS}$
- 实际选用 **AD9648**（双14-bit，80MSPS）或H743内置ADC欠采样模式

### 2.2 核心原理

#### 2.2.1 FPGA加速的关键计算

| 功能 | 算法 | FPGA实现方式 | 加速比 (vs STM32纯软) |
|------|------|-------------|:--------------------:|
| FFT | 1024点复数FFT | 流水线Radix-4, 48周期完成 | **~20×** |
| FIR滤波 | 64阶FIR | 乘加树, 1周期/抽头 | **~32×** |
| CORDIC | I/Q解调, 三角函数 | 流水线, 1输出/周期 | **~50×** |
| 等效采样 | 多周期累加+重排 | 状态机+Block RAM | **硬件实现** |

**FFT加速验证**：
- STM32F407: 1024点FFT ≈ 108k cycle / 168MHz = **643 μs**
- XC7S50 (逻辑): 1024点流水线FFT ≈ 48 cycle / 100MHz = **0.48 μs**
- 加速比 ≈ 1340×（但受SPI传输瓶颈限制，实际约20-50×）

#### 2.2.2 等效采样FPGA实现

FPGA内置 **DLL/PLL** 产生精密延迟时钟：

```
主时钟 100MHz (T = 10ns)
→ PLL倍频至 1GHz (T = 1ns)
→ 精细延迟链 (IODELAY, 步进 ~50ps)
→ 等效时间分辨率 Δt_eff = 10ns / M
```

对于 **M = 50**，等效采样率 = 100MSPS × 50 = **5GSPS**。

与方案1的MC100EP195方案对比：
- FPGA方案：延迟步进 **50ps**，空间分辨率 ≈ **0.5cm**
- MC100EP195方案：延迟步进 **10ps**，空间分辨率 ≈ **0.1cm**
- FPGA优势：无需额外IC，可灵活调整

#### 2.2.3 FPGA内部DDS设计

采用 **32-bit相位累加器** + **查找表（12-bit幅度）**：

$$f_{\text{out}} = \frac{\text{FTW}}{2^{32}} \times f_{\text{clk}}$$

**频率分辨率验证**：
- $f_{\text{clk}} = 200\text{MHz}$（内部DDS时钟）
- 频率分辨率 $\Delta f = 200\text{MHz} / 2^{32} = 0.0466 \text{ Hz}$
- 远优于题目要求（2025-G要求频率误差<5%，即50Hz@1kHz）

**SFDR（无杂散动态范围）验证**：
- 12-bit LUT幅度 ⇒ 理论SFDR ≈ 74 dBc
- 实际SFDR（含DAC）≈ 65 dBc @ 10MHz
- 2022-F AM/FM调制度测量要求 SNR ≈ 46dB（m=0.5%可检测）⇒ **满足**

#### 2.2.4 FM接收机FPGA实现（替代SA612A）

FPGA可配合 **高速ADC（200MSPS+）** 实现软件定义FM接收：

```
天线 → LNA(BFP740) → ADC(AD9648, 14-bit, 80MSPS) → FPGA数字下变频
```

**数字下变频（DDC）原理**：
$$I[n] = x[n] \times \cos(2\pi f_{\text{LO}} n / f_s)$$
$$Q[n] = x[n] \times (-\sin(2\pi f_{\text{LO}} n / f_s))$$

LPF后得到基带I/Q，然后：
$$f_{\text{inst}}(t) = \frac{1}{2\pi} \frac{d}{dt} \arctan\left(\frac{Q(t)}{I(t)}\right)$$

**参数验证**：
- ADC采样率 80MSPS ⇒ 奈奎斯特带宽 40MHz
- FM 88-108MHz 需**带通采样**：$f_s = \frac{2f_H}{m}$，选择m=3 ⇒ $f_s = \frac{2\times108}{3} = 72\text{ MSPS}$（80MSPS够用）
- 下变频后基带I/Q输出 ≈ 100kSPS（FM带宽±150kHz ⇒ Nyquist 300kHz）

### 3.2 核心原理

#### 3.2.1 模拟开关矩阵设计

使用 **ADG1408**（8:1模拟多路器，导通电阻 5Ω，带宽 60MHz）构建三级通道选择：

```
输入信号 → 前端调理(5选1) → ADC输入(3选1)
                      ↓
            信号源输出(3选1) → 被测电路
```

**通道串联阻抗验证**：
- 三级 ADG1408 串联 + 一级 ADG633
- 总导通电阻 $R_{\text{on,total}} = 3 \times 5\Omega + 1 \times 45\Omega = 60\Omega$
- 与ADC输入阻抗（典型 1MΩ）分压：衰减 $= \frac{1M}{1M+60} \approx 0.99994$，**可忽略**

**带宽验证**：
- 单通道带宽 60MHz（ADG1408典型值）
- 三级级联：实际带宽 ≈ $60\text{MHz} / \sqrt{2^{1/3} - 1} \approx 24\text{ MHz}$
- 满足 2025-D 30MHz 需求？**临界** → 改用 ADG1208（带宽 500MHz）或射频继电器

**改进方案**：对 >20MHz 信号路径使用 **HF 射频继电器**（如 G6K-2F-RF，DC-1GHz，¥15）

#### 3.2.2 多ADC协同采样

| ADC | 通道数 | 分辨率 | 采样率 | 接口 | 用途 |
|-----|:-----:|:-----:|:-----:|:----:|------|
| AD7606 | 8 | 16-bit | 200kSPS | 并行16-bit | 功率分析、LCR、低频通用 |
| AD9288 | 2 | 8-bit | 100MSPS | 并行8-bit | TDR高速捕获 |
| ADS8320 | 1 | 16-bit | 100kSPS | SPI | 精密DC测量（4线电阻） |
| STM32内置 | 2 | 16-bit | 3.6MSPS | — | 中速通用采样 |

#### 3.2.3 TDR脉冲生成（100MHz带宽）

```
GPIO(STM32) → 高速比较器(ADCMP600, tr=0.7ns) → 可编程延迟(MC100EP195) → DAC输出
```

**脉冲上升时间验证**：
- ADCMP600：tr = 0.7ns ⇒ 等效带宽 BW = 0.35 / 0.7ns ≈ 500MHz
- 对应空间分辨率 $\Delta L = c \times v_f \times t_r / 2 = 3\times10^8 \times 0.67 \times 0.7\times10^{-9} / 2 \approx 7\text{ cm}$
- 2025-D要求开路故障定位精度：满足1m以内

#### 3.2.4 4线电阻测量（2025-D）

**恒流源设计**（Howland拓扑）：

$$I_{\text{out}} = \frac{V_{\text{in}}}{R_{\text{sense}}}$$

**参数验证**：
- $V_{\text{in}} = 1\text{V}$（DAC8565输出）
- $R_{\text{sense}} = 100\Omega$
- $I_{\text{out}} = 10\text{ mA}$

测量电压范围：0.01Ω → 0.1mV; 10Ω → 100mV; 100Ω → 1V

**4线法消除引线电阻**：
$$R_x = \frac{V_{\text{sense}}}{I_{\text{source}}}$$

使用ADS8320（16-bit, 100kSPS）测量V_sense：
- LSB = 5V / 2^16 = 76.3 μV
- 电阻分辨率 = 76.3μV / 10mA = **0.0076 Ω**
- 2025-D要求电阻误差<5% ⇒ 对1Ω测量，误差0.0076/1=0.76% ✅

## 五、公式与参数验证总表

### 5.1 ADC采样相关

| 公式 | 参数验证 |
|------|---------|
| **奈奎斯特采样定理** | 2021-A: f_max=500kHz, f_s≥1MSPS, 实际1MSPS⇒2×过采样 |
| $f_s \geq 2f_{\text{max}}$ | 2023-D: f_c=2MHz, f_s≥4MSPS, 实际7.2MSPS⇒3.6×过采样 |
| | 2025-F: IF=10.7MHz, BW=±150kHz, f_s≥21.7MSPS, 实际AD9648@80MSPS |
| **ADC量化SNR** | 12-bit: SNR=6.02×12+1.76=**74.0dB** |
| $\text{SNR} = 6.02N + 1.76$ (dB) | 16-bit: SNR=6.02×16+1.76=**98.1dB** |
| | AD7606实际ENOB≈12.5@10kHz ⇒ SNR≈**77dB** |
| **抗混叠滤波器阶数** | 条件: 在f_s-f_max处衰减≥SNR |
| $n = \frac{\text{SNR(dB)}/20}{\log_{10}(f_s/f_c - 1)}$ | 2021-A: fd=500kHz, fs=1MHz, SNR=77dB ⇒ n≥7.7阶 |
| | 简化: 用LTC1562 4阶+软件过采样6dB/倍频 ⇒ 可行 |
| **过采样增益** | 每4×过采样 ⇒ +1 bit分辨率 |
| $\text{SNR}_{\text{OS}} = \text{SNR} + 10\log_{10}(\text{OSR})$ | 2024-B: 50Hz信号@2.5kSPS, OSR=25 ⇒ SNR改善14dB |

### 5.2 FFT与频谱分析

| 公式 | 参数验证 |
|------|---------|
| **频率分辨率** | 2021-A: fs=1MSPS, N=8192 ⇒ Δf=122Hz |
| $\Delta f = f_s / N$ | 5次谐波=500kHz, 分辨率122Hz ⇒ 4096个bin可分辨 |
| | 2024-B: fs=2.56kSPS, N=512 ⇒ Δf=5Hz |
| | 10次谐波=500Hz, 分辨率5Hz ⇒ 100个bin可分辨 |
| **FFT处理时间** | STM32H743: 8192点FFT ≈ 200k cycle / 480MHz = **0.42ms** |
| $T_{\text{FFT}} = \frac{\text{cycles}}{f_{\text{CPU}}}$ | FPGA方案: 流水线1024点FFT = 0.48μs |
| | 2025-G系统辨识: 40次FFT(各频率) ≈ 17ms |

### 5.3 等效采样时序

| 公式 | 参数验证 |
|------|---------|
| **等效采样率** | 2023-B: f_real=100MSPS, M=40 ⇒ f_eff=4GSPS |
| $f_{\text{eff}} = f_{\text{real}} \times M$ | Δt_eff=0.25ns ⇒ 空间分辨率=c×vf×Δt/2=**2.5cm** |
| **采样窗口长度** | 2023-B: N=500等效点 × 0.25ns = **125ns窗口** |
| $T_{\text{window}} = N \times \Delta t_{\text{eff}}$ | 电缆长度=c×vf×T/2=3e8×0.67×125ns/2=**12.5m** |
| **捕获时间** | 2025-D: N=500, M=40, 每触发周期=1ms ⇒ T=500×40×1ms=**20s** |
| $T_{\text{capture}} = N \times M \times T_{\text{cycle}}$ | 优化: 周期可降至10μs+ ⇒ T=500×40×10μs=**0.2s** ✅ |

| 公式 | 参数验证 |
|------|---------|
| **输出频率** | AD9834: f_clk=75MHz, FTW=24-bit |
