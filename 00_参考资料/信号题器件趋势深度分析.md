# 2021-2025年信号及仪器仪表类题目器件趋势深度分析

## 一、 信号题器件演变时间线（按题目映射）

| 年份 | 信号题题号 | 题目核心需求 | 清单中"最相关新增/升级器件" | 器件出现的"潜台词" |
|:---|:---|:---|:---|:---|
| **2021** | A题（信号失真度） | 测量1kHz~100kHz信号的THD | 50MHz**双通道**函数发生器、1GHz频谱分析仪、任意波发生器(1MHz)、**DDS芯片** | 需要精确信号源+频谱分析能力 |
| **2021** | J题（波形识别） | 识别正弦/三角/方波，测频率幅值占空比 | 同上 + **频率计(500MHz)** | 需要精确频率测量 |
| **2022** | F题（调制度测量） | 识别AM/FM，测ma/mf，解调输出 | **带AM/FM调制的40MHz信号源**、**乘法器**、功率分析仪 | 需要调制信号产生+解调能力 |
| **2022** | D题（混沌信号） | 产生并观测混沌波形 | 100MHz示波器、高速运放 | 需要高速采集+非线性电路 |
| **2023** | B题（同轴电缆检测） | TDR测长度、识别负载类型 | **矢量网络分析仪(VNA, 1GHz)**、**射频信号源(500MHz)** | 需要S参数/反射测量能力 |
| **2023** | C题（电感电容测量） | 测L/C/Q/D，高频扩展 | **LCR测试仪**、**高频Q表(≥20MHz)**、**可变增益集成宽带放大器** | 需要阻抗测量+宽带放大能力 |
| **2023** | D题（调制识别） | 识别AM/FM/ASK/FSK/PSK/CW | **射频信号源**、**乘法器**、**高速ADC/DAC** | 需要软件无线电(SDR)前端 |
| **2023** | H题（信号分离） | 从混合信号C=A+B中分离出A'和B' | **乘法器**、**DDS**、**高速比较器** | 需要锁相环/相干检测能力 |
| **2024** | B题（单相功率分析仪） | 测功率、功率因数、THD、谐波 | **单相功率分析仪**（仪器，作为对标）、**电压互感器(250V)**、**电流互感器(5A)** | 需要强弱电隔离+精密功率测量 |
| **2024** | C题（无线传输模拟） | 模拟直达信号+多径信号 | **乘法器**、**DDS**、**高速运放**、**高频器件** | 需要AM调制+多径相位控制 |
| **2024** | G题（录音屏蔽） | 超声波干扰录音设备 | **超声波换能器(16~20kHz, 20W)** 及配套变压器 | 需要大功率超声发射 |
| **2025** | D题（网线测试） | 测线序、通断、故障定位 | **UTP/SFTP网线**、**RJ45连接器**、**6~9寸液晶显示屏** | 需要网络物理层测试 |
| **2025** | F题（自动接收机） | 自动搜台、FM/AM接收解调 | **DDS**、**乘法器**、**可变增益集成宽带放大器**、**射频信号源(500MHz)** | 需要超外差接收机全套链路 |
| **2025** | G题（电路模型探究） | 黑箱识别R/L/C及其组合 | **LCR测试仪**、**DDS**、**高速ADC**、**运放/比较器** | 需要扫频阻抗分析能力 |

## 二、 六大核心器件趋势（信号题视角）

### 趋势1：信号源从"函数发生器"走向"射频信号源+DDS双轨制"

**演变轨迹**：
```
2021: 函数发生器(50MHz, 双通道) + 任意波发生器(1MHz) + DDS芯片(清单)
2022: 信号源(10M单通道) + 信号源(40MHz, 带AM/FM调制) + DDS芯片(清单)
2023: 函数发生器(50MHz, 双通道) + 射频信号源(500MHz, -100dBm~0dBm) + DDS芯片(清单)
2024: 函数发生器(20MHz, 双通道) + 无射频信号源 + 无DDS(清单未列但题需)
2025: 信号发生器(100MHz, 双通道) + 射频信号源(500MHz) + DDS芯片(清单)
```

**深刻解读**：
- 2021年还是传统的"函数发生器+任意波+DDS"三位一体。任意波发生器只能到1MHz，说明当时只考虑低频信号。
- 2022年出现了**带AM/FM调制的40MHz信号源**，这是一个强烈的信号：官方在暗示"调制解调"将成为考点。果然当年出了F题（调制度测量）。
- 2023年是一个分水岭：**射频信号源(500MHz)**首次出现。这不是给函数发生器做陪衬的，而是独立的一台仪器。它意味着题目中会出现射频载波（如2023年D题2MHz载频、2024年C题30~40MHz载波）。
- 2024年省赛（模拟电子系统设计专题赛）清单里**没有DDS**，但C题（无线传输）要求产生30~40MHz载波。这说明清单不再是" exhaustive list（穷尽列表）"，而是" baseline（基线）"。DDS没列不代表不需要，反而说明命题人默认学生会自带。
- 2025年**信号发生器跃升至100MHz双通道**，射频信号源保留。这意味着信号题的频率上限已经从几十kHz（2021年THD）→几十MHz（2024年多径模拟）→可能逼近100MHz（2025年接收机）。

**对备赛的启示**：
- DDS模块（AD9833/AD9850/AD9959/AD9954）不再是"可有可无"，而是信号题的**第一优先级器件**。
- 必须学会用射频信号源产生**调幅、调频、ASK、FSK**信号，用于自测和标定。

### 趋势2：示波器带宽的"军备竞赛"：100MHz → 200MHz → 500MHz

**演变轨迹**：
```
2021: 100MHz, 双通道
2022: 100MHz, 双通道
2023: 100MHz, 双通道
2024: 200MHz, 双通道（省赛/模拟邀请赛）
2025: 500MHz, 双通道（国赛）
```

**深刻解读**：
- 2021-2023年连续三年100MHz，说明信号频率主要在几十MHz以下（如2021年A题100kHz、2022年F题10~30MHz、2023年D题2MHz载频）。
- 2024年省赛突然跳到200MHz，而2025年国赛直接跳到**500MHz**。
- 500MHz示波器意味着什么？
  - 可以直接观测几十MHz的载波包络（如AM调制后的波形）。
  - 可以测量纳秒级脉冲（如TDR的10ns脉冲，在500MHz示波器上能看到较清晰的形状）。
  - 可以观测高速数字信号的边沿（如DDS输出的方波高次谐波）。
- **这与产业趋势完全同步**：5G通信、毫米波雷达、高速SerDes（串行解串器）都需要GHz级测量能力。电赛在倒逼学生从"低频模拟思维"转向"射频高速思维"。

**对备赛的启示**：
- 如果实验室只有100MHz示波器，做2025年F题（自动接收机，涉及30~40MHz中频）时，波形观测会很吃力（载波一个周期只有20~25ns，100MHz示波器采样率可能只有1GSPS，一个周期只有40~50个点，边缘模糊）。
- 建议提前熟悉**数字示波器的高级功能**：FFT（频谱分析）、模板测试（Mask Test）、XY模式（李萨如图形/相图）、协议解码（如果涉及数字接口）。

### 趋势3：乘法器的"地位跃升"：从通用器件到调制解调核心

**出现记录**：连续5年（2021-2025）出现在清单中，但其在信号题中的角色逐年加重。

**各年信号题中的具体用法**：

| 年份 | 题目 | 乘法器的核心用法 |
|:---|:---|:---|
| 2022 | F题（调制度测量） | **混频下变频**：将10~30MHz的AM/FM信号与DDS本振相乘，得到低频中频信号，再用普通ADC采样 |
| 2023 | D题（调制识别） | **相干解调**：将2MHz信号与本地载波相乘，提取I/Q分量，判断调制方式 |
| 2023 | H题（信号分离） | **数字鉴相**：将DDS重建信号与原始混合信号相乘，通过低通滤波得到直流分量（反映相位差），驱动PLL锁定 |
| 2024 | C题（无线传输模拟） | **AM调制**：将DDS产生的载波与2MHz调制信号通过乘法器相乘，产生AM波 |
| 2025 | F题（自动接收机） | **超外差接收**：天线信号 × 本振信号 = 中频信号（如10.7MHz），然后鉴频/鉴相 |

**深刻解读**：
- 乘法器（如AD835，带宽250MHz）是**模拟域的"万能变换器"**。它的输出包含两个输入信号的和频与差频。只要配合低通滤波器，就可以实现：
  - **下变频**（取差频，降低频率以便采样）
  - **鉴相**（若两信号同频，输出直流分量正比于cos(相位差)）
  - **鉴频**（若一信号是调频波，先微分再鉴相，或直接测量频率偏移）
  - **调制**（载波 × 调制信号 = AM波）
  - **解调**（AM波 × 载波 = 原信号+2倍频，低通取原信号）
- 电赛官方连续5年把乘法器放在清单里，就是在**反复暗示"模拟混频"是信号题的核心方法论**。

**对备赛的启示**：
- 不要只把乘法器当作"做乘法用的"，要理解它的**频谱搬移本质**。
- 在面包板上提前验证"DDS本振 + 乘法器AD835 + 低通滤波器"链路，确保能稳定输出中频信号。
- 如果AD835买不到或太贵，可以用**模拟开关（如CD4053）+ 运放**搭建简易乘法器，但带宽和线性度会下降，仅适用于低频（<1MHz）。

### 趋势4：从"频谱分析仪"到"矢量网络分析仪(VNA)"：反射与阻抗测量成为新考点

**出现记录**：
- 频谱分析仪：2021、2022、2023、2025（1GHz）——除2024年省赛外，基本全勤
- 矢量网络分析仪(VNA)：仅2023年出现（1GHz）
- LCR测试仪：2023、2025年出现
- 高频Q表：仅2023年出现（≥20MHz）

**深刻解读**：
- **频谱分析仪**是"单向测量"：看信号在频域上的能量分布。适合THD测量（2021年A题）、调制度测量（2022年F题）、调制识别（2023年D题）。
- **矢量网络分析仪(VNA)**是"双向测量"：发射扫频信号，同时测量反射波和透射波，得到S参数（S11反射系数、S21传输系数）。这是射频/微波领域的核心仪器。
- 2023年出现VNA，与当年B题（同轴电缆长度与终端负载检测）**强相关**。做TDR有两种方法：
  - **时域法**：发射窄脉冲，测反射时间（需要高速示波器+高速脉冲发生器）。
  - **频域法**：用VNA扫频测S11，然后通过**逆傅里叶变换(IFFT)**得到时域反射波形。这种方法不需要超高速脉冲，只需要频域数据，然后用算法换算。
- LCR测试仪和Q表的出现，对应2023年C题（电感电容测量）。题目要求测Q值（品质因数）和D值（损耗角正切），这些正是LCR表和Q表的典型功能。

**对备赛的启示**：
- 如果实验室没有VNA（这很正常，VNA价格从几万到几十万），可以通过**"DDS扫频 + 定向耦合器/电阻桥 + 峰值检波器 + MCU做IFFT"**来搭建简易VNA，实现TDR功能。
- 理解**S11参数**的物理意义：$S_{11} = \frac{V_{反射}}{V_{入射}}$，对于开路$\Gamma=+1$，短路$\Gamma=-1$，匹配负载$\Gamma=0$。
- LCR测试仪的原理（自动平衡电桥）可以在电赛中用**串联分压法+相位检测**简化实现。

### 趋势5：TI生态的"强制绑定"：从建议到硬约束

**演变轨迹**：
```
2021: TI MCU系统板（建议）
2022: TI MCU系统板（可含调试模块）
2023: TI MCU系统板 + 题目C题硬约束"必须使用TI公司的MCU"
2024: TI MSPM0系列开发板（指定系列）
2025: TI MSPM0系统板（可含调试模块）
```

**深刻解读**：
- TI不仅是电赛的赞助商，更是通过**清单+题目双重约束**推广其新产品线。
- **MSPM0系列**是TI在2023-2024年推出的基于ARM Cortex-M0+的低成本、低功耗MCU。它集成了TI擅长的模拟外设：
  - 12位4MSPS ADC（MSPM0G3507）
  - 可编程增益放大器(PGA)
  - 12位DAC
  - 比较器(COMP)
  - 温度传感器
- 2023年C题（电感电容测量）官方解析中提到了**MSPM0L1306单芯片方案**：仅用一颗MSPM0L1306+少量无源元件就实现了L/C测量。这正是TI想推广的"集成模拟优势"。
- 2025年清单直接写"TI MSPM0系统板"，意味着**MSPM0已经成为TI在电赛中的主推平台**，可能会逐步替代MSP430/MSP432的地位。

**对备赛的启示**：
- 如果主攻信号题，必须提前熟悉**MSPM0系列**（建议入手MSPM0G3507 LaunchPad或MSPM0L1306开发板）。
- 重点学习MSPM0的**模拟外设配置**：ADC的同步采样（用于功率分析仪的电压电流同步）、DAC+OPA产生正弦波（替代外部DDS的低成本方案）、比较器做高速过零检测。
- 掌握TI的**SysConfig图形化配置工具**和SDK，能大幅减少开发时间。

### 趋势6：传感器与执行器的"声学化"与"网络化"

**演变轨迹**：
- **2021-2023**：传统传感器（光电、角度、麦克风、蜂鸣器、扬声器）
- **2024年突变**：
  - 新增**超声波换能器(16~20kHz, 20W)及配套变压器** → 对应G题（录音屏蔽系统）
  - 新增**电压互感器(250V)**和**电流互感器(5A)** → 对应B题（单相功率分析仪）
  - 新增**霍尔传感器** → 磁悬浮/磁场测量
- **2025年**：
  - 保留**麦克风、扬声器、蜂鸣器、压电陶瓷蜂鸣片**
  - 新增**UTP型网线、SFTP型网线、RJ45连接器** → 对应D题（网线测试仪）
  - 新增**405nm波长蓝紫色点状激光笔** → 配合视觉/光学测量

**深刻解读**：
- **超声波换能器**的出现是一个非常强烈的信号：电赛开始涉足"声学隐蔽/反窃听"这类信息安全相关的内容。这与产业界的**声学安全（Acoustic Security）**和**反录音干扰器（Audio Jammer）**市场直接相关。
- **互感器**的出现说明功率测量不再是简单的"测电压电流"，而是涉及**220V强电安全隔离**和**谐波分析**。这与智能电网、电能质量监测的产业趋势一致。
- **网线/RJ45**的出现则反映了"工业互联网"和"网络物理层测试"的需求。高速以太网（千兆/万兆）的物理层测试（如TDR测线缆、回波损耗、近端串扰）是通信设备厂商（华为、中兴、思科）的常规工作。

**对备赛的启示**：
- 信号题不再局限于"电信号"，开始涉及**声-电转换**（麦克风/扬声器/超声换能器）和**光-电转换**（激光笔/光电传感器）。
- 准备**超声波驱动电路**：用D类功放（如TPA3116）或半桥驱动器（如IR2104）推动超声换能器，频率20~40kHz可调。
- 理解**压电陶瓷蜂鸣片**与**普通扬声器**的区别：蜂鸣片是容性负载，高频响应好，适合超声波；扬声器是感性负载，低频响应好，适合音频。

## 三、 信号题器件-题目映射关系网

下图展示了清单中核心器件与信号题之间的"因果关系"：

```
【信号产生层】
DDS芯片/模块 ────→ 2021-A（THD测量：需精确信号源）
     │
     ├────────────→ 2022-F（调制度测量：需AM/FM信号源）
     │
     ├────────────→ 2023-D（调制识别：需多种调制信号）
     │
     ├────────────→ 2023-H（信号分离：需重建信号A'和B'）
     │
     ├────────────→ 2024-C（无线传输模拟：需30~40MHz载波+2MHz调制）
     │
     └────────────→ 2025-F（自动接收机：需本振信号）

【信号调理层】
运算放大器 ────────→ 所有信号题（放大、滤波、缓冲、加法器）
     │
乘法器 ────────────→ 2022-F（混频/解调）
     │
     ├────────────→ 2023-D（相干解调/I-Q提取）
     │
     ├────────────→ 2023-H（鉴相/PLL）
     │
     ├────────────→ 2024-C（AM调制）
     │
     └────────────→ 2025-F（超外差混频）

可变增益放大器(VGA) ─→ 2023-C（电感电容测量：自动量程切换）
     │
     └────────────→ 2025-F（AGC自动增益控制）

【信号采集层】
A/D转换器 ─────────→ 所有信号题（数字化）
     │
高速ADC (>10MSPS) ──→ 2021-A（5次谐波500kHz，需>1MSPS）
     │
     ├────────────→ 2022-F（10~30MHz信号，需下变频后采样）
     │
     ├────────────→ 2023-D（2MHz信号，需I/Q采样）
     │
     └────────────→ 2024-C（中频采样，如10.7MHz）

【信号分析层】
频谱分析仪 ────────→ 2021-A（看谐波分布）
     │
     ├────────────→ 2022-F（看AM/FM频谱）
     │
     ├────────────→ 2023-D（看调制信号频谱特征）
     │
     └────────────→ 2025-F（看接收信号频谱）

矢量网络分析仪(VNA) → 2023-B（同轴电缆：S11→TDR）

LCR测试仪 ─────────→ 2023-C（对标/校准自制LCR表）
     │
     └────────────→ 2025-G（对标/校准电路模型探究装置）

【特殊功能层】
功率分析仪 ────────→ 2024-B（对标单相功率分析仪）

电压/电流互感器 ───→ 2024-B（220V隔离测量）

超声波换能器 ──────→ 2024-G（录音屏蔽：发射干扰超声）

麦克风 ───────────→ 2024-G（检测环境音频有无）
     │
     └────────────→ 2025-G（声音采集/识别）
```

## 四、 信号题器件的"三级备赛策略"

### Level 1：核心铁三角（必须熟练掌握）

| 器件 | 功能 | 推荐型号 | 信号题应用 |
|:---|:---|:---|:---|
| **DDS模块** | 精确频率合成 | AD9833(AD公司) / AD9850 / AD9959(4通道) | 信号源、本振、重建信号 |
| **模拟乘法器** | 混频/调制/解调/鉴相 | AD835(250MHz带宽) / MC1496(1MHz，低成本) | AM调制、下变频、鉴相 |
| **高速运放** | 信号放大/缓冲/滤波 | THS3001(420MHz) / OPA847(3.9GHz GBW) / NE5532(音频) | 前端放大、滤波器、驱动 |

**为什么叫"铁三角"？**
- 有DDS，你就能产生任何频率的信号（信号源）。
- 有乘法器，你就能把高频信号搬到低频（下变频），或者把低频信号搬到高频（调制）。
- 有高速运放，你就能把微弱信号放大到ADC可测范围，或者滤除噪声。
- 这三者组合，就能搭建出**软件定义无线电(SDR)**的模拟前端，解决90%的信号题。

## 五、 对2026年信号题器件的预测

基于2021-2025年的器件趋势，对2026年信号题可能出现的器件和方向做如下预测：

### 预测1：射频信号源频率可能提升至1GHz
- 2023年500MHz → 2025年500MHz（未变）→ **2026年可能跃升至1GHz或更高**
- 这意味着信号题可能涉及**UHF频段（300MHz~3GHz）**，如无线通信、雷达、卫星导航。

### 预测2：可能出现"直接射频采样ADC"或"射频收发器(RF Transceiver)"
- 2023-2025年射频信号源+VNA的出现，暗示官方在鼓励学生进入射频领域。
- 下一步可能是提供**集成度更高的射频器件**，如ADI AD9361/9364（射频收发器，覆盖70MHz~6GHz），或TI AFE77xx系列。
- 如果清单中出现"射频收发器"，题目可能是"简易软件无线电"或"频谱感知"。

### 预测3：AI/ML信号处理器件或工具链
- 2024年G题已经要求"识别语音和音乐"，这是最简单的模式识别。
- 下一步可能是**"实时语音识别""噪声抑制""调制方式自动分类（深度学习版）"**。
- 器件上可能不会新增硬件（仍然用MCU/DSP），但清单中可能增加**"神经网络加速器"或"边缘AI开发套件"**（如TI TDA4VM、Intel Movidius）。

### 预测4：太赫兹/毫米波器件（小概率但有可能）
- 6G通信的研究热点是毫米波（24GHz以上）和太赫兹（100GHz以上）。
- 如果2026年出现"24GHz雷达模块"或"毫米波天线"，题目可能是"简易雷达测距/测速"。
- 但考虑到成本和安全性，这更可能出现在**信息科技前沿专题赛**而非全国赛。

### 预测5：光通信器件
- 2025年出现了405nm蓝紫激光笔，这比红/绿激光波长更短，可用于**光刻、荧光激发、水下通信**等。
- 下一步可能是**"简易光纤通信系统"**（用LED/激光二极管+光纤+光电二极管搭建数字/模拟光通信链路）。
- 器件可能增加：光纤、光电二极管（PIN/APD）、激光二极管驱动器。
