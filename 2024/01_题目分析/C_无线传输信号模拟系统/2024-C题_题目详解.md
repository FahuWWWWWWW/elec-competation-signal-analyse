# 2024年电赛C题「无线传输信号模拟系统」详解

## 一、题目复述与指标拆解

### 1.1 任务描述

设计制作模拟无线传输信号的系统，包括模拟产生直达传输信号和多径传输信号，并合路得到模拟的无线传输信号。其中，直达信号的初相、幅值可设置，多径信号相对直达信号的时延、初相和幅值衰减可设置。

### 1.2 指标拆解

| 指标项 | 基本要求 | 发挥部分 | 考核本质 |
|--------|----------|----------|----------|
| **载波频率** | 固定30~40MHz内某频点 | **30~40MHz以1MHz步进可调** | **DDS频率合成** |
| **信号类型** | CW或AM (2MHz调制) | 同基本 | **AM调制产生** |
| **直达信号幅度** | 有效值100mV~1V, 100mV步进, 误差≤10mV | 同基本 | **幅度控制/PGA** |
| **AM调制度** | 30%~90%, 10%步进, 误差≤5% | 同基本 | **调制深度控制** |
| **多径时延** | 50~200ns, 30ns步进, 误差≤10ns | 同基本 | **延迟线/数字延迟** |
| **多径衰减** | 0~20dB, 2dB步进, 误差≤1dB | 同基本 | **衰减器/PGA** |
| **多径相位** | — | **0°~180°, 30°步进, 误差≤10°** | **移相器/数字相位** |
| **合路输出** | 正确、稳定 | 同基本 | **信号加法器** |

### 1.3 核心信号模型

**直达信号**:
- CW: $S_D(t) = A_D \cos(2\pi f_c t + \phi_D)$
- AM: $S_D(t) = A_D [1 + m_a \cos(2\pi f_m t)] \cos(2\pi f_c t + \phi_D)$

**多径信号**:
$$S_M(t) = \alpha \cdot A_D \cos(2\pi f_c (t - \tau) + \phi_D + \phi_M)$$

其中:
- $\alpha$ = 幅度衰减因子 (0~1, 对应0~20dB衰减)
- $\tau$ = 时延 (50~200ns)
- $\phi_M$ = 多径引入的初相 (0°~180°)

**合路信号**:
$$S_{Out}(t) = S_D(t) + S_M(t)$$

## 二、核心物理原理

### 2.1 AM调制回顾

**AM信号表达式**:
$$s_{AM}(t) = A_c [1 + m_a \cos(2\pi f_m t)] \cos(2\pi f_c t)$$

其中:
- $A_c$ = 载波幅度
- $m_a$ = 调制度 (0 < $m_a$ ≤ 1)
- $f_m$ = 调制频率 (本题2MHz)
- $f_c$ = 载波频率 (30~40MHz)

**AM频谱**:
- 载频分量: $f_c$, 幅度$A_c$
- 上边频: $f_c + f_m$, 幅度$A_c m_a / 2$
- 下边频: $f_c - f_m$, 幅度$A_c m_a / 2$

**本题要求**: $m_a$ = 30%~90%, 步进10%

### 2.2 多径效应

无线信号从发射到接收，可能经过多条路径。接收信号是各路径信号的叠加：

$$r(t) = \sum_{k=0}^{N-1} \alpha_k s(t - \tau_k) e^{j\phi_k}$$

**直达路径** (k=0): $\alpha_0$≈1, $\tau_0$≈0, $\phi_0$=0

**多径** (k≥1): $\alpha_k$<1, $\tau_k$>0, $\phi_k$随机

**本题简化**: 只模拟一条多径（单径模型）。

### 2.3 时延与相位的等价性

**固定时延τ引入的相移**:
$$\Delta\phi = 2\pi f_c \tau$$

**数值**:
- fc=35MHz, τ=50ns → Δφ = 2π×35×10⁶×50×10⁻⁹ = 11π rad = 1980° (等效360°×5.5)

**结论**: 
- 时延必然引入相移
- 但题目要求的"多径初相"是**额外**的相移（在时延相移基础上的附加相移）
- 或者可以认为：相位控制与时延控制是独立的两个自由度

## 三、调理电路方案设计

### 3.1 推荐方案：DDS + 数字延迟 + 模拟合路

```
[DDS/PLL信号源]  -- 产生30~40MHz载波
    │
    ├──-> [直达通道]
    │         │
    │         ├──-> [幅度控制 PGA]  -- 100mV~1Vrms可调
    │         │         │
    │         │         v
    │         ├──-> [AM调制器]  -- 2MHz调制, ma可调
    │         │         │
    │         │         v
    │         ├──-> [相位控制]  -- 直达信号初相
    │         │         │
    │         │         v
    │         └──-> [直达信号SD]
    │
    └──-> [多径通道]
              │
              ├──-> [固定延迟线]  -- 50~200ns
              │         │
              │         v
              ├──-> [可变衰减器]  -- 0~20dB
              │         │
              │         v
              ├──-> [相位偏移]  -- 0°~180°
              │         │
              │         v
              └──-> [多径信号SM]
                        │
                        v
                 [加法器]  -- SOut = SD + SM
```

### 3.2 关键电路模块

#### (1) 载波产生：DDS

**器件**: AD9850/AD9834 (DDS芯片)
- AD9850: 最高125MHz输出，足够30~40MHz
- 频率分辨率: 0.029Hz (@125MHz参考)
- 可产生CW和AM（外部调制）

**方案B: PLL+VCO**
- 用锁相环产生30~40MHz
- 频率步进1MHz → 参考晶振1MHz
- 稳定度更高，但电路复杂

**推荐**: DDS方案（频率切换快，数字化控制）

#### (2) AM调制

**方案A: 模拟乘法器**
- 器件: AD835 (四象限乘法器, 250MHz带宽)
- 原理: $V_{out} = V_X \cdot V_Y$
- 将载波和(1+m_a·cosωt)相乘

**方案B: 可变增益放大器(VGA)**
- 器件: AD8367 (VGA, 500MHz)
- 原理: 用调制信号控制增益，实现AM
- 缺点: 需要线性增益控制

**方案C: 乘法型DAC**
- 用高速DAC同时输出载波和包络
- 数字化调制，最灵活

**推荐**: 模拟乘法器AD835

#### (3) 幅度控制

**方案**: 数字衰减器或PGA
- HMC624A: 0~31dB, 0.5dB步进, 6-bit控制
- 或简单的电阻分压网络+模拟开关

#### (4) 时延产生

**方案A: 同轴电缆延迟线**
- 同轴电缆速度因子0.66 → 1m电缆≈5ns延迟
- 50ns → 10m电缆
- 200ns → 40m电缆
- **优点**: 宽带、低损耗
- **缺点**: 体积大、不易精确控制

**方案B: LC延迟线**
- 集总参数LC网络实现延迟
- **缺点**: 带宽有限、阻抗匹配难

**方案C: 数字延迟 (基带)**
- 在低频/中频进行数字延迟
- 需要下变频→延迟→上变频

**方案D: 有源延迟线 (all-pass滤波器)**
- 运放+RC网络产生可调延迟
- 一阶all-pass: τ = RC
- 需要多级级联

**推荐**: 
- 基本要求(50~200ns, 30ns步进): 用同轴电缆抽头+继电器切换
- 或all-pass滤波器级联

#### (5) 相位控制

**方案A: 移相网络 (RC/RL)**
- 简单RC网络: 相移=arctan(ωRC)
- 可调电阻改变相位
- **缺点**: 同时改变幅度

**方案B: I/Q调制法**
- 将信号分为I(同相)和Q(正交)两路
- 分别乘以cosφ和sinφ，再相加
- $S_{out} = I\cos\phi + Q\sin\phi$
- **优点**: 幅度不变，相位连续可调
- **缺点**: 需要正交信号源

**方案C: DDS直接调相**
- AD9850支持相位调制(POTW = Phase Offset Tuning Word)
- 直接设置相位偏移
- **最简单！**

**推荐**: DDS直接调相（如果SD和SM各用一个DDS）

### 3.3 合路输出

**加法器**: 简单的电阻加法器或运放加法器
- 需要带宽>40MHz
- 器件: OPA695 (宽带运放, 1.4GHz GBW)

## 四、关键算法设计

### 4.1 DDS频率控制

```python
# AD9850频率调谐字
f_sysclk = 125e6  # 系统时钟125MHz
f_carrier = 35e6  # 载波35MHz

# 频率调谐字 (FTW)
FTW = int(f_carrier / f_sysclk * 2**32)
```

### 4.2 AM调制深度控制

```python
def generate_am_signal(t, fc, fm, Ac, ma):
    """产生AM信号"""
    carrier = np.cos(2 * np.pi * fc * t)
    modulating = np.cos(2 * np.pi * fm * t)
    am_signal = Ac * (1 + ma * modulating) * carrier
    return am_signal
```

### 4.3 多径信号生成

```python
def generate_multipath_signal(t, fc, Ac, alpha, tau, phi_m, phi_d=0):
    """产生多径信号"""
    # 时延引入的相移
    phase_delay = 2 * np.pi * fc * tau
    
    # 总相移 = 时延相移 + 附加相移
    total_phase = phi_d + phase_delay + np.radians(phi_m)
    
    # 衰减后的幅度
    Am = alpha * Ac
    
    # 多径信号 (时域延迟)
    # 注意: 数字实现时需要用插值实现精确延迟
    sm = Am * np.cos(2 * np.pi * fc * (t - tau) + total_phase)
    
    return sm
```

## 五、设计要点与陷阱

### 5.1 DDS输出镜像频率

DDS输出有镜像频率（fsysclk - fout），需要低通滤波器滤除。
- fsysclk=125MHz, fout=35MHz → 镜像在90MHz
- 需要LPF截止频率<62.5MHz (fsysclk/2)
- 建议用7阶LC低通，截止频率~50MHz

### 5.2 AM调制度测量

**调制度定义**: $m_a = \frac{A_{max} - A_{min}}{A_{max} + A_{min}}$

**测量方法**:
- 包络检波提取调制信号
- 测量包络的峰峰值和载波幅度
- 计算ma

**误差来源**:
- 乘法器非线性
- 载波泄漏
- 调制信号幅度不准确

### 5.3 时延精度控制

**30ns步进，10ns误差**:
- 同轴电缆: 1m≈5ns, 精度由长度决定
- 要精确到10ns → 长度精度±2m
- **建议**: 用标准长度电缆（如2m=10ns）组合

### 5.4 相位与时延的耦合

如果使用时延线产生时延，必然引入相移。
- 用户设置"时延100ns"时，系统实际产生的是100ns时延+额外相移
- 如果用户再设置"多径相位60°"，系统需要在此相移基础上再加60°
- **或者**: 将所有相位控制集成到DDS中，时延用模拟延迟线

### 5.5 合路信号的稳定性

直达信号和多径信号相干叠加：
- 如果τ和φ设置不当，可能完全抵消（相消干涉）
- 输出幅度不稳定是正常现象（模拟了真实多径衰落）

## 七、总结

2024-C题是一道**射频信号产生与信道模拟**的题目，核心挑战：

1. **高频载波产生**: 30~40MHz DDS + 滤波
2. **AM调制**: 模拟乘法器 + 调制深度精确控制
3. **多径模拟**: 时延(50~200ns) + 衰减(0~20dB) + 相移(0~180°)
4. **信号合路**: 宽带加法器

**技术路径**:
1. DDS1 → 直达信号 (频率/相位/幅度可调)
2. DDS2 → 多径信号 (频率相同, 相位/幅度/时延可调)
3. 模拟乘法器 → AM调制
4. 延迟线/数字延迟 → 多径时延
5. 运放加法器 → 合路输出

**最深刻的认识**: 无线信道模拟是通信测试的核心。从简单的单径模型到复杂的Jakes衰落模型，本质上都是"产生多条路径信号，按衰减/时延/相位叠加"。本题是最简单的单径模型，但已经涵盖了信道模拟的所有基本要素。
