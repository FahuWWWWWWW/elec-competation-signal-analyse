# 2024-C 无线传输信号模拟系统 — 硬件实现方案分析

## 方案A：纯MCU方案（STM32H723ZGT6 + 模拟射频链路）

### 1. 系统架构

```
[STM32H723ZGT6 @550MHz]
     │
     ├── FSMC/GPIO ── [AD9850 DDS #1] ── [7阶LC LPF fc=50MHz] ──┬── [直达通道]
     │               (载波30~40MHz, 相位/幅度控制)                │
     │                                                            ├── [AD835 AM调制器]
     │                                                            │    ← 2MHz调制(MCU DAC)
     │                                                            │
     ├── FSMC/GPIO ── [AD9850 DDS #2] ── [7阶LC LPF fc=50MHz] ──┼── [多径通道]
     │               (同频载波, 独立相位控制)                      │
     │                                                            ├── [HMC624A数字衰减器 0~31dB]
     │                                                            │
     │                                                            ├── [同轴电缆延迟线 50~200ns]
     │                                                            │     5m/10m/20m/30m抽头+继电器
     │                                                            │
     │                                                            └── [OPA695加法器] ── SMA输出
     │
     └── [TFT 2.4" LCD] + [矩阵键盘]
```

### 2. 前端调理电路

**DDS输出滤波**：
- AD9850输出125MHz系统时钟，输出30~40MHz
- 镜像频率在125-35=90MHz处，需LPF抑制
- 7阶椭圆LC LPF：截止频率50MHz，通带纹波0.1dB
- 电感：Coilcraft 0805CS系列 22nH/33nH/47nH
- 电容：村田GJM系列 5.6pF/8.2pF/12pF

**AM调制**：
- AD835四象限乘法器，带宽250MHz，满量程(±1V)×(±1V)→1V输出
- 直达载波输入X通道，调制信号(1+ma·cos2πfmt)从Y通道输入
- 调制信号由MCU DAC(TIM+DMA)产生，2MHz正弦波±0.5V范围
- AD835输出经OPA695放大至100mV~1Vrms

**多径延迟**：
- 同轴电缆RG316，速度因子0.69，延迟≈4.8ns/m
- 50ns → 10.4m；200ns → 41.6m
- 抽头点：10.4m/20.8m/31.2m/41.6m对应50/100/150/200ns
- 用G6K-2P-RF继电器切换抽头(插入损耗<0.5dB@40MHz)

**多径衰减**：
- HMC624ALP4E：6bit数字衰减器，0.5dB步进，0~31.5dB范围
- 串行/并行控制，50Ω阻抗匹配

### 3. 后处理与算法

- MCU通过并行接口写AD9850频率调谐字(FTW)和相位调谐字(POW)
- AD835调制深度控制：m_a = Vm/Vc，通过DAC输出幅度调节
- 通道切换：GPIO控制继电器矩阵选择延迟线抽头
- 合路后需经过OPA695缓冲输出，SMA接口50Ω匹配
- 显示：TFT LCD显示当前载波频率、幅度、调制度、时延、衰减等参数

### 4. 关键参数验证

**频率精度**：
- AD9850系统时钟125MHz，32bit FTW
- 频率分辨率 = 125MHz / 2^32 = 0.0291Hz
- 30MHz对应FTW = 30e6/125e6 × 2^32 = 0x3C28F5C3
- 1MHz步进误差：< 0.03Hz ✅ 远小于题目要求

**AM调制度精度**：
- m_a = (Vmax - Vmin)/(Vmax + Vmin) = Vm/Vc
- MCU DAC 12bit，3.3V参考，分辨率0.8mV
- 30%调制度：Vm/Vc=0.3 → Vm=0.3×100mV=30mV(DC偏置后)
- DAC精度0.8mV → 调制度误差约0.8/30≈2.7% ✅ (<5%)

**时延精度**：
- 同轴电缆RG316，延迟≈4.8ns/m
- 10.4m → 50ns，裁剪精度±0.1m → ±0.48ns ✅ (<10ns要求)
- 继电器切换产生时延跳变，无累积误差

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| 全模拟射频链路，信号质量好 | 同轴电缆体积大(最长41.6m) |
| AD835乘法器高线性度 | 时延不可连续可调(30ns步进需更多抽头) |
| MCU H723性能充裕 | 继电器机械寿命(但低频切换可忽略) |
| 架构直观，调试方便 | 温度影响电缆延时，无补偿 |

## 方案B：MCU+专用RF芯片方案（STM32F407VGT6 + AD9850×2）

### 1. 系统架构

```
[STM32F407VGT6]
     │
     ├── GPI0 ── [AD9850 DDS #1] ── [7阶LC LPF fc=50MHz]
     │          (直达信号：可调幅度/相位/AM调制)
     │
     ├── GPIO ── [AD9850 DDS #2] ── [7阶LC LPF fc=50MHz]
     │          (多径信号：同频载波 + 独立相位控制)
     │
     ├── DAC/TIM ── 2MHz调制信号 → [AD835 Y输入]
     │                                    │
     │    DDS#1载波 → [AD835 X输入] ─────┤
     │                                    ├── [直达AM信号 SD]
     │                                    │
     │    DDS#2载波 → [AD835 X输入] ─────┘
     │                    │
     │               [HMC624A 衰减器] ←─ MCU SPI
     │                    │
     │               [数字延迟线] ←── FPGA or MCU+移位寄存器
     │                    │
     │            ┌── [加法器 OPA695] ── SMA输出
     │            │
     ├── [ADC采样监测] ── 包络检波 → 回读调制度
     │
     └── [TFT显示] + [编码器旋钮]
```

### 2. 前端调理电路

**直达通道DDS**：
- AD9850输出CW或AM需外置调制
- 当AM调制时：DDS产生纯载波，送入AD835与2MHz调制信号相乘
- 幅度控制：DDS输出后接电阻分压网络+MCU DAC调节AD835 Y偏置
- 直达相位φ_D：通过AD9850相位控制字(POW)设置，11bit分辨率

**多径通道**：
- DDS#2产生同频载波，相位控制字设置多径相位φ_M (0°~180°)
- 时延实现：采用数字延迟方案
  - AD9480高速ADC(1GSPS)采样→FPGA SRAM延迟→DAC恢复
  - 或：使用MCU外接高速DAC+DMA+定时器实现ns级延迟
- 衰减：HMC624A，6bit SPI控制，0.5dB步进

**相位控制**：
- AD9850相位偏移分辨率：360°/2^11 = 0.176°
- 直接写POW即可实现0°~180°任意步进 ✅
- 比RC移相网络精确得多

### 3. 后处理与算法

- MCU计算AD9850 FTW/POW并下载
- AM调制深度控制：校准Vc(载波)+Vm(调制信号)幅度，计算m_a
- ADC回读：AD8318对数检波器监测输出幅度，闭环校准
- 用户界面：旋钮编码器选择参数，TFT屏显
- 批量校准：出厂前每个DDS/衰减器/乘法器做V-I曲线标定，存入Flash

### 4. 关键参数验证

**多径时延分辨率**：
- 数字延迟方案：延迟步进 = 1/f_sample = 1ns (@1GSPS ADC) 或 10ns (@100MHz)
- 50~200ns范围可覆盖，步进可达5ns
- 采用f_s=200MHz DAC → 5ns步进 ✅ (<10ns误差要求)

**相位控制精度**：
- AD9850相位分辨率11bit → 360°/2048 = 0.176°
- 30°步进 → 理论精度0.176°/30°=0.59% ✅
- 实际误差来源：DDS相位噪声+PCB走线不匹配约±1~2°

**直达信号幅度精度**：
- 100mV~1Vrms，100mV步进，误差≤10mV
- AD835后接可变增益放大器(AD8367, 500MHz)
- AD8367增益范围 -11.5~+24.5dB，线性dB/V
- MCU DAC控制VG → 精度±0.5dB → 对应 ±6%(100mV时±6mV) ✅

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| 双DDS独立控制相位，兼顾直达+多径 | 数字延迟方案需要高速ADC/DAC，成本高 |
| AM调制采用AD835乘法器，精度高 | 相位控制与时延存在耦合，需软件补偿 |
| 闭环校准保证输出精度 | 多个RF器件级联，阻抗匹配复杂 |
| 全部程控，自动化程度高 | PCB设计要求高(射频走线+阻抗控制) |

## 方案C：FPGA异构方案（ZYNQ XC7Z010 + AD9850/AD835）

### 1. 系统架构

```
[ZYNQ XC7Z010]
     │
     ├── PL(FPGA)部分:
     │    ├── 双通道DDS数字核 (32bit累加器, 相位/频率独立)
     │    ├── 数字延迟线 (BRAM FIFO, 采样时钟f_s=500MHz)
     │    ├── 数字幅度控制 (乘法器 + CORDIC)
     │    ├── 数字AM调制核 (内部产生2MHz调制信号)
     │    └── 数字加法器合路
     │            │
     │      [DAC5672 双通道14bit 500MSPS]
     │            │
     │      [7阶LC LPF fc=50MHz] ×2
     │            │
     ├── PS(ARM)部分:
     │    ├── 上位机/用户UI控制
     │    │     └── 通过AXI-lite配置PL寄存器
     │    │
     │    └── [HDMI输出] / [UART]
     │
     └── 直达 LPF  ──┬── [AD835 模拟AM调制] ── [OPA695缓冲] ── SMA直达
                      │
                      └── 多径 LPF  ── [HMC624A衰减] ── [OPA695缓冲] ── SMA多径
                                                                    │
                                                              [电阻加法器] ── SMA合路
```

### 2. 前端调理电路

**DAC输出**：
- DAC5672：双通道14bit，500MSPS，差分电流输出
- 输出经RF变压器(ADT1-1WT)转为单端
- 7阶椭圆LC LPF，截止频率50MHz，抑制90MHz镜像

**AM调制**：
- 方案混合：FPGA内完成数字AM调制（乘法器生成AM信号基带包络）
- DAC直接输出AM调制波形（载波+包络合成）
- 这样只需一路模拟乘法器做精调，甚至可取消AD835

**数字延迟**：
- FPGA内置 BRAM FIFO做延迟线，宽度14bit，深度1024
- 采样时钟500MHz → 延迟分辨率2ns
- 延迟范围：2ns×1024=2048ns >> 200ns
- 步进2ns，远优于10ns要求 ✅

**多径相位**：
- 数字域直接控制：两路DDS的相位偏移独立调节
- 精度32bit → 360°/2^32不可察觉
- 可独立设置直达相位和多径相位

### 3. 后处理与算法

- PL(FPGA)内实现全数字信号链路：
  - 两路DDS核：频率/相位独立32bit调节
  - 数字延迟：BRAM FIFO + 读地址偏移
  - 数字滤波：可选数字成形滤波(提高频谱纯度)
  - 数字调制：AM包络与载波在数字域相乘
- PS(ARM Cortex-A9)负责：
  - AXI-lite寄存器配置
  - 用户输入解析(UART/USB键盘)
  - HDMI显示输出(通过VDMA+AXI_GP)
  - 参数校准与非线性补偿
- 无需外部MCU，ZYNQ单片解决

### 4. 关键参数验证

**延迟分辨率**：
- DAC采样时钟 f_s = 500MHz → T_s = 2ns
- 数字延迟线最小步进 = 2ns
- 50~200ns范围，可设置25~100个时钟延迟
- 步进30ns → 15个时钟周期 @500MHz ✅
- 误差=0(数字域精确时钟) ✅

**DDS频率精度**：
- ZYNQ PL系统时钟 f_clk = 250MHz
- 相位累加器32bit
- 频率分辨率 = 250MHz/2^32 = 0.058Hz
- 载波30MHz → 频率调谐字 = 30e6/250e6×2^32 = 0x1999999A

**SFDR(无杂散动态范围)**：
- DAC5672 14bit → 理论SFDR = 6.02×14+1.76 = 86dB
- 实际受DDS相位截断和DAC非线性影响 → 典型值70dBc @35MHz
- 优于方案A/B (AD9850的DDS SFDR典型55dBc)

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| 全数字延迟线，精度极高(2ns步进) | 开发周期长(Vivado HLS/Verilog) |
| ZYNQ单片解决(FPGA+ARM)，集成度高 | BOM成本高(DAC+DDS冗余) |
| SFDR优于AD9850方案 | 功耗偏高(FPGA+500MSPS DAC) |
| 灵活性最强：可扩展OFDM/多径叠加 | 500MHz PCB布局要求严格 |
| 数字相位控制精度32bit | 500MSPS DAC可选器件有限 |

## 三个方案对比总结

| 维度 | 方案A 纯MCU | 方案B MCU+专用RF | 方案C ZYNQ异构 |
|------|------------|-----------------|---------------|
| 核心 | H723 + AD9850×2 | F407 + AD9850×2 | ZYNQ XC7Z010 + DAC5672 |
| 时延方式 | 同轴电缆 10~42m | 高速ADC+DAC数字延迟 | FPGA BRAM FIFO数字延迟 |
| 时延精度 | ±0.5ns(电缆) | ±5ns(DAC步进) | ±2ns(精确时钟) ✅ |
| 相位精度 | AD9850 11bit (0.176°) | AD9850 11bit | FPFA 32bit (理论无穷) ✅ |
| SFDR | ~55dBc | ~55dBc | ~70dBc ✅ |
| 开发周期 | 最短(1周) | 适中(2周) | 最长(4周+) |
| BOM | ¥411 (电缆贵) | ¥292 (性价比最优) | ¥376 |
| 可扩展性 | 低(抽头固定) | 中(DAC延迟可调) | 极高(FIR/Fading/OFDM) |
| **推荐** | **不推荐(电缆体积大)** | **推荐(性价比+性能)** | **推荐(追求性能+扩展)** |
