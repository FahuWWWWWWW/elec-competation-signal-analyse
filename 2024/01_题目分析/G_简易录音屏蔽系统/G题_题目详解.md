# 2024年电赛G题「简易录音屏蔽系统」题目详解

## 一、题目核心要求拆解

### 1.1 任务概述

设计并制作一个**阻塞式录音屏蔽系统**，包含两个核心模块：
1. **录音屏蔽信号发生器**：产生超声波干扰信号，干扰录音设备的正常录制
2. **音频信号监测/识别模块**：检测环境音频信号，自动控制屏蔽器的开关，并能识别语音/音乐类型

### 1.2 关键指标一览

| 要求项 | 具体指标 | 分值 | 考核本质 |
|--------|---------|------|---------|
| **基本要求(1)** | 屏蔽距离≥1m，屏蔽角度≥60° | 35分 | **超声波发射功率+指向性** |
| **基本要求(2)** | 无音频时自动关闭屏蔽，关闭LED | 25分 | **音频检测(VAD) + 自动控制** |
| **基本要求(3)** | 输入功率≤6W，输出功率1~4W可调(1W步进) | 25分 | **功率控制 + 效率** |
| **发挥要求(4)** | 自动识别信号类型(语音/音乐)，选择屏蔽 | 10分 | **音频分类算法** |
| **发挥要求(5)** | 其他创新 | 5分 | |

## 二、技术原理深度解析

### 2.1 为什么超声波能屏蔽录音？

**核心原理：麦克风非线性失真**

录音设备的麦克风在接收高强度超声波时会产生**非线性失真**，将超声波的谐波分量下混频到音频基带（20Hz~20kHz），产生强烈的噪声，淹没正常语音信号。

```
语音信号 (300Hz~3.4kHz) ──→
                                 ├──→ [麦克风非线性] ──→ 混合信号
超声波 (22~25kHz, 高强度) ──→    │                      (语音被噪声淹没)
                                 └──→ 人耳听不见 (或很微弱)
```

**物理过程**:
1. 超声波换能器发射22-25kHz的高强度声波（约90-100dB SPL）
2. 手机/录音笔的MEMS麦克风接收到超声波
3. 麦克风振膜在高声压下产生非线性振动
4. 非线性产生谐波和互调产物，覆盖语音频段
5. 录音设备ADC采样时，超声波也可能因采样混叠（aliasing）进入基带

### 2.2 超声波频率选择

| 频率范围 | 特点 | 适用性 |
|---------|------|--------|
| **20-22kHz** | 部分人耳可闻（年轻人） | 可能干扰正常交流 |
| **22-25kHz** | 绝大多数人耳不可闻 | ✅ **最佳选择** |
| **25-30kHz** | 完全不可闻 | 换能器效率下降 |
| **>30kHz** | 人耳完全不可闻 | 换能器尺寸小但效率更低 |

> **推荐**: 22-25kHz，兼顾人耳不可闻和换能器效率

### 2.3 屏蔽效果评估标准

屏蔽成功意味着：**录音回放时无法听懂语音内容**

常用评估指标：
- **SNR（信噪比）**: 录音中语音与噪声的比值，SNR < 0dB时基本不可懂
- **PESQ/ STOI**: 语音质量/可懂度客观评分，PESQ < 1.0 或 STOI < 0.3 认为不可懂

### 3.2 各模块技术要点

#### 3.2.1 超声波换能器阵列

| 参数 | 设计要点 |
|------|---------|
| **换能器类型** | 压电式超声波换能器（如TCT40-16T/R） |
| **中心频率** | 23kHz 或 25kHz |
| **发射功率** | 单个换能器约0.5-1W，4个并联可达4W |
| **指向性** | 单个换能器波束角约60°，阵列可压缩至30-40° |
| **阵列设计** | 2×2或1×4阵列，间距λ/2≈7mm（23kHz） |

#### 3.2.2 音频监测/识别模块

**功能1: VAD（Voice Activity Detection）**

```python
# 简化版VAD算法流程
音频采样(16kHz) → 分帧(30ms, 50%重叠)
    → 计算短时能量 + 过零率
    → 双门限判决
    → 输出: 有语音 / 无语音
```

**功能2: 音频类型识别（语音 vs 音乐）**

| 特征 | 语音 | 音乐 |
|------|------|------|
| **基频稳定性** | 快速变化 | 相对稳定 |
| **谐波结构** | 简单(男声~150Hz,女声~250Hz) | 复杂，多乐器 |
| **频谱平坦度** | 较低（有共振峰） | 较高（宽带） |
| **时域包络** | 有明显停顿（音节间） | 相对连续 |

**识别算法**:
1. 提取MFCC特征（13维）
2. 计算频谱质心、带宽、滚降点
3. 使用简单阈值或SVM分类器

## 四、关键设计难点与解决方案

### 4.1 难点1：超声波指向性 vs 屏蔽角度

**矛盾**: 要求屏蔽角度≥60°，但超声波换能器指向性越尖锐效率越高

**解决方案**:
- 使用多个换能器构成**宽波束阵列**
- 或者使用**单个宽波束换能器**（牺牲距离换角度）
- 用反射板/喇叭口扩散声波

### 4.2 难点2：功率控制精度

**要求**: 1~4W可调，1W步进

**实现方案**:
- **方案A**: 数字电位器控制功放的增益
- **方案B**: 开关选择不同数量的换能器（1个=1W, 2个=2W, ...）
- **方案C**: PWM控制D类功放占空比（程控，得分高）

### 4.3 难点3：不误伤正常交流

**要求**: "对正常音频交流无影响"

**实现方案**:
- 选择22-25kHz频率，绝大多数人耳不可闻
- 使用定向发射，只覆盖需要屏蔽的区域
- VAD控制：无语音时不发射，节省功耗

### 4.4 难点4：屏蔽距离≥1m

**物理限制**: 声波在空气中衰减约6dB/距离翻倍（球面波）

**实现方案**:
- 增加发射功率（4W可覆盖1-2m）
- 使用阵列增加指向性增益
- 减小换能器波束角，集中能量

## 五、评分策略与失分陷阱

### 5.1 失分陷阱

| 陷阱 | 后果 | 避免方法 |
|------|------|---------|
| **频率过低(<20kHz)** | 人耳可闻，影响交流 | 严格控制在22kHz以上 |
| **功率过高(>6W输入)** | 直接失分 | 严格功率预算设计 |
| **无VAD控制** | 25分拿不到 | 必须实现音频检测自动开关 |
| **无类型识别** | 发挥分10分拿不到 | 至少实现简单的语音/音乐区分 |
| **使用市售产品** | 违规，取消资格 | 全部自制 |

### 5.2 得分策略

**基础分 (85分)**:
- 屏蔽距离≥1m + 角度≥60° (35分) — **核心**
- 自动开关控制 (25分) — **必做**
- 功率可调1-4W (25分) — **必做**

**发挥分 (15分)**:
- 音频类型识别 (10分) — **建议做**
- 其他创新 (5分) — 如：
  - 自适应功率控制（根据距离调整）
  - 手机APP蓝牙控制
  - 多区域独立屏蔽
  - 显示屏实时显示工作状态

## 六、与类似技术的关联

| 技术 | 应用场景 | 与G题的关联 |
|------|---------|-----------|
| **主动降噪(ANC)** | 耳机降噪 | 原理相反：ANC是抵消噪声，G题是制造噪声 |
| **超声波测距** | 倒车雷达 | 换能器可复用，驱动电路类似 |
| **声波武器(LRAD)** | 军事/警用 | 同类技术，功率更高 |
| **语音识别** | 智能音箱 | VAD算法可直接借鉴 |
