# 2024年电赛B题「单相功率分析仪」详解

## 一、题目复述与指标拆解

### 1.1 任务描述

基于TI公司的MCU，设计并制作单相功率分析仪。系统实现测量用电插座上负载的电流、电压、有功功率、功率因数、电流谐波系数（THD）、电流基波及其2~10次谐波分量的有效值等参数。系统由电池供电，电流/电压测量必须采用隔离方式（电流互感器/电压互感器）。

### 1.2 指标拆解

| 指标项 | 要求 | 考核本质 |
|--------|------|---------|
| **电压测量** | AC 220V, 误差≤1% | **隔离采样 + RMS计算** |
| **电流测量** | 有效值4A, 误差≤1% | **电流互感器 + 匝数比设置** |
| **有功功率** | 误差≤1% | **瞬时功率平均** |
| **功率因数** | 误差≤1% | **P/(U×I) 或 cos(φ)** |
| **电流THD** | 至少测到10次谐波, 误差≤2% | **FFT谐波分析** |
| **谐波电流** | 基波+2~10次, 误差≤2% | **频谱分解** |
| **功耗** | ≤50mW (工作状态) | **低功耗设计** |
| **隔离要求** | 必须用互感器或隔离方案 | **安全第一** |
| **MCU限制** | 必须选用TI MCU | **TI生态** |

### 1.3 被测量物理本质

| 被测量 | 定义 | 计算方法 |
|--------|------|---------|
| **电压有效值U** | $U = \sqrt{\frac{1}{T}\int_0^T u^2(t)dt}$ | ADC采样→平方→平均→开方 |
| **电流有效值I** | $I = \sqrt{\frac{1}{T}\int_0^T i^2(t)dt}$ | 同上 |
| **有功功率P** | $P = \frac{1}{T}\int_0^T u(t)i(t)dt$ | ADC同步采样u,i→相乘→平均 |
| **视在功率S** | $S = U \times I$ | 直接计算 |
| **功率因数PF** | $PF = P / S = P / (U \times I)$ | 直接计算 |
| **无功功率Q** | $Q = \sqrt{S^2 - P^2}$ | 间接计算 |
| **THD** | $THD = \frac{\sqrt{\sum_{n=2}^{\infty} I_n^2}}{I_1} \times 100\%$ | FFT→谐波分量提取 |

## 二、核心物理原理

### 2.1 交流信号的数字测量

**有效值(RMS)的离散计算**:
$$U_{RMS} = \sqrt{\frac{1}{N}\sum_{n=0}^{N-1} u^2[n]}$$

**有功功率的离散计算**:
$$P = \frac{1}{N}\sum_{n=0}^{N-1} u[n] \cdot i[n]$$

**功率因数**:
$$PF = \frac{P}{U_{RMS} \cdot I_{RMS}}$$

**注意**: 对于非正弦波，PF ≠ cos(φ)。功率因数可能小于cos(φ)（由于谐波存在）。

### 2.2 谐波与THD

**傅里叶级数**: 任何周期信号都可以分解为基波+谐波。

$$i(t) = I_1\sin(\omega t + \phi_1) + I_2\sin(2\omega t + \phi_2) + I_3\sin(3\omega t + \phi_3) + ...$$

**THD定义**:
$$THD_I = \frac{\sqrt{I_2^2 + I_3^2 + ... + I_{10}^2}}{I_1} \times 100\%$$

**THD的测量方法**:
1. 对电流信号采样一个周期（或多周期）
2. 做FFT
3. 提取基波幅度I1和各次谐波幅度I2~I10
4. 按定义计算THD

### 2.3 互感器原理

**电压互感器(PT)**:
- 原理: 降压变压器，将220V交流降为低压（如2.2V）
- 匝数比: Np:Ns = 100:1 → 副边电压 = 原边/100
- 隔离: 原副边电气隔离，确保安全

**电流互感器(CT)**:
- 原理: 通过电磁感应，原方电流产生磁场，副方感应电压
- 公式: $I_p \times N_p = I_s \times N_s$（安匝平衡）
- 副方通常接小电阻（ burden resistor），将电流转为电压测量
- **多匝缠绕**: 原方导线绕Np匝穿过CT → 灵敏度提高Np倍

**互感器误差来源**:
- 磁芯饱和（大电流时）
- 相位误差（角差）
- 频率响应（对谐波有衰减）

## 三、调理电路方案设计

### 3.1 完整功率分析仪调理链路

```
AC 220V供电
    │
    ├──-> [电压互感器(PT)]  -- 220V→2.2V (隔离)
    │         │
    │         v
    │    [偏置电路]  -- 将交流抬升至ADC量程中间(1.65V for 3.3V ADC)
    │         │
    │         v
    │    [抗混叠LPF]  -- fc > 550Hz (10次谐波@50Hz基波)
    │         │
    │         v
    │    [ADC_CH1]  -- 电压通道
    │
    └──-> [电流互感器(CT)]  -- 4A→小电流→burden电阻→电压
              │
              v
         [偏置电路]
              │
              v
         [抗混叠LPF]
              │
              v
         [ADC_CH2]  -- 电流通道 (与电压同步采样!)
              │
              v
         [MCU - TI TMS320F280025]
              │
              ├──-> [RMS计算] → U, I
              ├──-> [功率计算] → P, S, PF
              ├──-> [FFT] → 基波+谐波 → THD
              │
              v
         [LCD显示]
              │
              v
         [低功耗管理]
```

### 3.2 关键电路模块

#### (1) 电压互感器(PT) + 偏置

**电路**:
```
AC 220V ──[PT 100:1]── AC 2.2V ──[分压 1:1]── AC 1.1V ──[+1.65V偏置]── 0.55V~2.75V
```

- PT副边输出2.2V (峰峰值约3.1V)
- 电阻分压至1.1Vpp (适配3.3V ADC)
- 用电阻分压加1.65V直流偏置，使信号在0.55V~2.75V范围

**器件**: ZMPT107B (微型PT), 2mA型

#### (2) 电流互感器(CT) + Burden电阻

**电路**:
```
负载电流 Ip ──[CT 1000:1]── Is = Ip/1000 ──[ burden R=100Ω ]── Vs = Is×R
```

- 4A原方电流 → 副方4mA
- 100Ω burden电阻 → 400mV电压
- **多匝缠绕**: 原方绕10匝 → 等效灵敏度提高10倍 → 40mA副方 → 4V (超量程)
- 实际: 绕5匝 → 等效20A量程/4A=5倍灵敏度

**器件**: ZMCT103C (微型CT), 5A型

#### (3) 同步采样

**关键**: 电压和电流必须**同步采样**，否则功率计算有误差。

**方法**:
- 使用MCU的双ADC同步模式
- 或一个ADC+模拟开关快速切换（误差<1%允许）
- 采样率: ≥2.5kSPS (50Hz基波，10次谐波=500Hz，Nyquist要求>1kHz)

#### (4) 低功耗设计

**目标**: 工作状态功耗<50mW

**策略**:
- 使用低功耗MCU (MSP430FR5994, 待机<1μA)
- ADC间歇采样（每秒采样10个周期，其余休眠）
- LCD使用反射式（无背光）
- 关闭未用外设
- 使用LDO而非DC-DC（降低纹波但效率略低）

## 四、关键算法设计

### 4.1 RMS计算

```python
def calculate_rms(samples):
    """计算有效值"""
    N = len(samples)
    sum_squares = np.sum(samples**2)
    rms = np.sqrt(sum_squares / N)
    return rms
```

### 4.2 有功功率计算

```python
def calculate_power(v_samples, i_samples):
    """计算有功功率"""
    N = len(v_samples)
    instantaneous_power = v_samples * i_samples
    P = np.mean(instantaneous_power)
    return P
```

### 4.3 谐波分析 (FFT)

```python
def harmonic_analysis(i_samples, fs, f_base=50):
    """
    分析电流谐波
    返回: 基波幅度, 谐波幅度列表, THD
    """
    N = len(i_samples)
    
    # 去直流
    i_ac = i_samples - np.mean(i_samples)
    
    # FFT
    fft_result = np.fft.fft(i_ac)
    magnitude = 2/N * np.abs(fft_result[:N//2])
    freqs = np.fft.fftfreq(N, 1/fs)[:N//2]
    
    # 提取基波和谐波
    harmonics = []
    for n in range(1, 11):  # 1~10次
        target_freq = n * f_base
        idx = np.argmin(np.abs(freqs - target_freq))
        harmonics.append(magnitude[idx])
    
    I1 = harmonics[0]
    I2_10 = np.array(harmonics[1:])
    
    # THD
    thd = np.sqrt(np.sum(I2_10**2)) / I1 * 100 if I1 > 0 else 0
    
    return I1, I2_10, thd
```

### 4.4 完整测量流程

```python
def measure_all(v_adc, i_adc, fs, pt_ratio, ct_ratio, burden_r):
    """
    完整参数测量
    
    参数:
        v_adc: ADC采样电压值 (V)
        i_adc: ADC采样电流值 (V) (burden电阻上的电压)
        fs: 采样率
        pt_ratio: PT匝数比
        ct_ratio: CT匝数比
        burden_r: burden电阻值
    
    返回:
        U, I, P, S, PF, THD, harmonics
    """
    # 1. 去偏置 (减去1.65V)
    v_ac = v_adc - 1.65
    i_ac = i_adc - 1.65
    
    # 2. 计算有效值 (ADC值→实际值)
    v_rms_adc = calculate_rms(v_ac)
    i_rms_adc = calculate_rms(i_ac)
    
    # 3. 转换到实际值
    U = v_rms_adc * pt_ratio  # PT变比还原
    I = i_rms_adc / burden_r * ct_ratio  # CT+burden还原
    
    # 4. 功率
    P = calculate_power(v_ac, i_ac) * (pt_ratio * ct_ratio / burden_r)
    S = U * I
    PF = P / S if S > 0 else 0
    
    # 5. 谐波
    I1, I_harm, THD = harmonic_analysis(i_ac, fs)
    I1_actual = I1 / burden_r * ct_ratio
    I_harm_actual = I_harm / burden_r * ct_ratio
    
    return U, I, P, S, PF, THD, I1_actual, I_harm_actual
```

## 五、设计要点与陷阱

### 5.1 ADC量程设计

**电压通道**: 
- 220Vrms → PT降压至2.2Vrms → 分压至1.1Vrms (峰峰值3.1V)
- 加1.65V偏置 → 信号范围0.1V~3.2V
- 12-bit ADC (3.3V满量程) → LSB = 0.8mV → 分辨率约0.07% ✅

**电流通道**:
- 4Arms → CT变比1000:1 → 4mA → 100Ω burden → 400mVrms
- 加1.65V偏置 → 信号范围1.25V~2.05V
- 12-bit ADC → 分辨率约0.2% ✅

**陷阱**: 如果burden电阻选太大（如500Ω），2A电流就会产生1V电压，可能饱和。

### 5.2 同步采样的重要性

如果电压和电流采样不同步（有时间差Δt），功率误差:
$$\Delta P \approx \omega \cdot \Delta t \cdot U \cdot I \cdot \sin(\phi)$$

**示例**: 
- 50Hz, Δt=1ms (一个采样周期@1kSPS)
- φ=30° → ΔP/P ≈ 2π×50×0.001×sin(30°) ≈ 15.7% ❌

**解决方案**: 
- 双ADC同步采样（如STM32的Dual ADC模式）
- 或采样率提高到10kSPS以上，使Δt<0.1ms

### 5.3 谐波测量的FFT参数选择

**基波频率**: 50Hz（中国电网）
**10次谐波**: 500Hz
**Nyquist要求**: fs > 1000Hz
**建议采样率**: fs = 2.5kHz或更高

**FFT点数**:
- 要分辨50Hz基波 → N = fs/50 = 50点（fs=2.5kHz时）
- 为了频率分辨率更高，取N=256或512
- **必须整数个周期**: 50Hz信号，采样256点@2.56kHz → 正好5个周期（推荐！）

**频谱泄漏**: 
- 如果采样不是整数周期，会有频谱泄漏
- **解决方案**: 使用窗函数（Hanning/Blackman），或确保fs/f_base = 整数

### 5.4 CT的多匝缠绕

**原理**: 原方绕Np匝 → 等效原方电流 = Np × I_actual

**灵敏度提升**: 
- 1匝, 4A → CT副方4mA
- 5匝, 4A → 等效20A → 副方20mA (灵敏度×5)

**量程变化**:
- 1匝, 量程4A
- 5匝, 量程变为4A/5 = 0.8A (因为CT认为有20A)
- **需要在软件中设置匝数比，补偿量程变化**

### 5.5 低功耗设计

**功耗预算**:
- MCU (MSP430, 运行): ~2mA @ 3.3V = 6.6mW
- ADC (内置): ~1mA = 3.3mW
- LCD (无背光): ~1mA = 3.3mW
- 运放: ~0.5mA = 1.65mW
- 互感器: 极小（无源）
- **总计**: ~15mW ✅ (远低于50mW)

## 七、总结

2024-B题是一道**综合性极强的电力测量仪器题目**，融合：
1. **模拟前端设计**: 隔离采样（PT/CT）、偏置电路、抗混叠滤波
2. **数字信号处理**: RMS计算、功率计算、FFT谐波分析
3. **低功耗设计**: <50mW功耗约束
4. **安全规范**: 强弱电隔离

**关键技术路径**:
1. PT/CT隔离采样 → 偏置+滤波 → ADC
2. 同步采样u,i → 计算U,I,P,PF
3. FFT分析电流 → 提取基波+谐波 → THD
4. 低功耗管理 → 间歇采样+休眠

**与前序题目的关系**: 此题是2021-A（FFT谐波分析）和2023-C（交流阻抗测量）的**综合升级**，同时加入了电力系统特有的隔离采样和低功耗要求。

**最深刻的认识**: 功率分析仪是工业现场最常见的仪器之一，从高端电能质量分析仪到智能电表，核心都是"采样→计算→显示"。本题要求<50mW功耗，实际上已经接近智能电表的设计标准。
