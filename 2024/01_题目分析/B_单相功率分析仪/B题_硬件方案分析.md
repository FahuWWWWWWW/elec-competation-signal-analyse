# 2024-B 单相功率分析仪 — 硬件实现方案分析

## 方案A：纯MCU方案（STM32F407VGT6 低功耗实现）

### 1. 系统架构

```
AC 220V ──┬── [ZMPT107B PT] ── [分压+偏置] ──┬── [二阶LPF fc=1kHz] ── [STM32F407 ADC1_CH]
          │                                    │                        Dual-ADC同步模式
          ├── [ZMCT103C CT] ── [Burden R+vref]─┴── [二阶LPF fc=1kHz] ── [STM32F407 ADC2_CH]
          │                                                              │
          │                                                         ├── 去直流+去偏置
          │                                                         ├── RMS计算 U,I
          │                                                         ├── 有功功率 P
          │                                                         ├── FFTHz(256点) → 谐波→THD
          │                                                         ├── PF = P/(U×I)
          │                                                         └── [0.96" OLED I2C] 显示
          │
          └── 3.7V Li-Po ── TPS78233 LDO (3.3V) ── 系统供电
```

### 2. 前端调理电路

**电压通道**：
- ZMPT107B 电压互感器，变比1000:1000，输入串联200kΩ限流电阻，副边输出0~2mA
- 副边并联100Ω采样电阻 → 输出AC 0~200mVrms (对应220Vrms)
- 一级同相放大 OPA2376 (G=11) → AC 2.2Vrms
- 直流偏置：用REF3333产生3.3V，经分压至1.65V通过运放跟随加入
- 二阶Sallen-Key LPF：fc=1kHz，R=15.9kΩ，C=10nF
- 最终信号范围：0.55V~2.75V，适配3.3V ADC

**电流通道**：
- ZMCT103C 电流互感器，变比1000:1，原方绕5匝
- 4A × 5匝 = 20A等效 → 副边20mA
- Burden电阻10Ω → 0.2Vrms
- 二级同相放大 OPA2376 (G=10) → 2.0Vrms
- 同电压通道偏置和滤波

### 3. 后处理与算法

- ADC：STM32F407 双ADC同步模式，12bit，3.3V参考
- 采样率：2.56kHz（每周期51.2点，10个周期512点）
- RMS：平方和均值开方，N=512点
- FFT：256点Cortex-M4 DSP库实FFT，Hanning窗抑制频谱泄漏
- 谐波提取：找频域峰值对应50/100/150/.../500Hz bin
- 功耗优化：每秒突发采样100ms(256点)，其余时间待机
- 显示：0.96寸OLED，SPI接口，刷新率1Hz

### 4. 关键参数验证

**ADC采样率验证**：
- 最高谐波频率 f_h = 10×50 = 500Hz
- Nyquist要求 f_s > 2×500 = 1000Hz
- 实际 f_s = 2560Hz，满足要求，整数周期：2560/50 = 51.2 → 取256点=5个周期

**频率分辨率**：
- Δf = f_s / N = 2560 / 256 = 10Hz
- 50Hz基波在bin5，500Hz在bin50 → 可清晰分离

**功耗估算**：
- STM32F407 (168MHz运行，ADC+DSP)：~28mA @ 3.3V → 92.4mW ❌ 超限
- 优化：降频至32MHz→ ~8mA + 间歇工作(10%占空比) → ~2.6mA + 外设 ~3mA
- 总功耗约 (2.6+3+OLED 0.5)×3.3 ≈ 20mW ✅

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| 电路结构简单，PCB面积小 | F407功耗偏高需降频+间歇 |
| BOM成本最低 | 12bit ADC分辨率有限(THD精度受限) |
| 全部自主可控 | FFT占用CPU，无法同时做其他任务 |
| 无需外部ADC/专用芯片 | Hanning窗降低频率分辨率 |

## 方案B：MCU+专用电能计量芯片（STM32H743VGT6 + ADE9000）

### 1. 系统架构

```
AC 220V ──┬── [ZMPT107B PT] ── [分压] ──┬── [ADE9000 VP/IN]
          │                              │    ├── 内置PGA×1/2/4
          ├── [ZMCT103C CT] ── [Burden] ─┴── [ADE9000 IP/IN]    │
          │                                       │              │
          │                              ┌── [SPI] ←──┐         │
          │                              │            │         │
          │                         [STM32H743]  [25MHz XTAL]   │
          │                              │                      │
          │                         ├── 读取U/I/P/Q/PF/THD     │
          │                         ├── 读取谐波寄存器(至10次)  │
          │                         └── [LCD显示]               │
          │                                                      │
          └── 3.7V Li-Po ── TPS78233 LDO ── H743 ↓ @480MHz需~3.3V/100mA
                                             → 使用DCDC(TPS62740)降压
```

### 2. 前端调理电路

**电压通道**：
- ZMPT107B，初级串200kΩ限流，变比1000:1000
- 副边输出经差分放大器 OPA2376 转为单端
- 信号范围0~1Vpp，直接进入ADE9000 VP输入
- ADE9000内部PGA固定为×1，差分输入

**电流通道**：
- ZMCT103C，原方绕5匝
- Burden电阻10Ω → 差分电压200mVpp
- 进入ADE9000 IP输入，PGA=×4 → 800mVpp

**供电**：
- 3.7V锂电池 → TPS62740 (DC-DC降压至3.3V, 效率90%)
- ADE9000模拟供电 AVDD = 3.3V，数字供电 DVDD = 1.8V（内部LDO）

### 3. 后处理与算法

- ADE9000：7阶Σ-Δ ADC，24bit，采样率8kSPS
- 硬件完成RMS、有功/无功/视在功率、PF、THD、谐波分析
- 内部FFT计算谐波分量至63次
- STM32H743通过SPI读取结果寄存器
- H743负责LCD显示、按键控制、蓝牙通信
- 低功耗：H743降频至240MHz，ADE9000采样率降至4kSPS

### 4. 关键参数验证

**ADC分辨率**：
- ADE9000 24bit Σ-Δ ADC，ENOB ~20bit
- 电压动态范围：220Vrms / 满量程500Vrms → -7dBFS
- 20bit ENOB → 电压分辨率：500V / 2^20 ≈ 477μV → 相对精度 < 0.001%

**功耗估算**：
- STM32H743 @240MHz：~45mA × 3.3V = 148.5mW (过高)
- 优化：工作模式10ms/每秒 (0.1%占空比) + 待机120μA
- 平均功耗 ≈ 0.1% × 148.5 + 99.9% × 0.4mW ≈ 0.55mW
- ADE9000连续工作：3.3V × 12mA = 39.6mW
- 总计：~40mW ✅

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| ADE9000硬件完成全部计量，精度极高(24bit) | 专用芯片成本高(¥45) |
| THD精度远优于纯MCU方案 | 需要DC-DC满足H743功耗 |
| 谐波支持至63次(远超10次要求) | 电路复杂度高于方案A |
| H743资源充裕可扩展蓝牙/WiFi | 低功耗需精心设计占空比睡眠 |

## 方案C：FPGA异构方案（Artix-7 XC7A35T + STM32F407VGT6）

### 1. 系统架构

```
AC 220V ──┬── [ZMPT107B PT] ── [分压+偏置] ──┬── [三阶LPF fc=800Hz]
          │                                    │        │
          ├── [ZMCT103C CT] ── [Burden+vref] ──┴── [三阶LPF fc=800Hz]
          │                                                           │
          │                                             ┌── [ADS9220 16bit ADC ×2] (SPI)
          │                                             │       │
          │                                             │  [Artix-7 XC7A35T]
          │                                             │       │
          │                                             │  ├── ADC SPI控制+同步
          │                                             │  ├── 定点FFT(512点, 并行)
          │                                             │  ├── RMS/P/PF硬件计算
          │                                             │  ├── 谐波提取引擎
          │                                             │  └── [UART/SPI] ──→
          │                                             │                   │
          │                                             │          [STM32F407]
          │                                             │              │
          │                                             │        ├── UI控制
          │                                             │        ├── LCD显示
          │                                             │        └── 按键/蓝牙
          │
          └── 3.7V ── TPS78233 (3.3V) ── FPGA(1.0V内部)+F407(3.3V)
                     ── TPS62202 (1.0V FPGA core)
```

### 2. 前端调理电路

**抗混叠滤波**：三阶巴特沃斯LPF，fc=800Hz
- 使用OPA2376 × 2构建三阶MFB拓扑
- R1=10kΩ, C1=22nF, R2=10kΩ, C2=22nF, R3=10kΩ, C3=10nF
- 在1kHz处衰减 ~ -12dB，有效抑制50Hz以上混叠

**ADC**：两片ADS9220 (16bit, 2MSPS SPI)
- 可编程采样率，设为2.56kHz
- 差分输入，PGA可编程
- 两个ADC共享同一转换时钟，保证同步精度±1ns

### 3. 后处理与算法

- FPGA(XC7A35T)负责实时信号处理：
  - SPI主控，双ADC同步采样
  - 并行流水线定点FFT (512点, 16bit, 32级流水)
  - 硬件RMS计算引擎（平方→累加→开方CORDIC）
  - 硬件谐波提取（找峰+幅度计算）
- STM32F407负责：
  - 通过UART/SPI接收FPGA计算结果
  - OLED/LCD显示刷新
  - 按键输入、蓝牙通信
  - 系统睡眠唤醒调度

### 4. 关键参数验证

**ADC SNR**：
- ADS9220 SNR = 93.5dB (typ)
- Vref=3.3V，输入满量程
- 理论ENOB = (93.5 - 1.76) / 6.02 = 15.24bit
- 实际电压通道跨导约为0.01V/Vrms → 可分辨 3.3V/2^15.24/220 ≈ 0.005% ✅

**FFT吞吐量**：
- FPGA系统时钟50MHz，FFT 512点
- 流水线周期数：512+log₂(512)×4 ≈ 560 clk
- 耗时：560 / 50MHz ≈ 11.2μs
- 每通道处理时间 < 20μs → 可实时处理2.56kHz数据流 ✅

**功耗估算**：
- XC7A35T：~40mA @1.0V core + 20mA @3.3V IO ≈ 110mW ❌
- 总功耗严重超标，不适合低功耗场景
- 电源需独立控制：完成测量后MCU关断FPGA电源

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| FPGA并行处理，FFT速度极快 | 功耗远超50mW限制 ❌ |
| 双通道16bit ADC精度高 | BOM成本最高 |
| 硬件流水线持续吞吐 | 开发复杂度高(Vivado+STM32IDE) |
| 灵活扩展(可加更多通道) | FPGA+双ADC占PCB面积大 |
