## 方案A：纯MCU方案（STM32H743VGT6 + 超外差前端）

### 1. 系统架构

```
RF输入 (10~30MHz, 100mVpp)
    │
    ├── [前置LNA]  BGA2818 (宽带低噪声放大器, Gain≈20dB, NF=3.2dB)
    │
    ├── [带通滤波器]  可调BPF (由ADF4351本振+变容二极管调谐)
    │                  10~30MHz 捷变调谐
    │
    ├── [混频器]  SA612A (双平衡混频器, 噪声系数NF=6dB, 转换增益17dB)
    │                输入 LO = ADF4351 PLL合成 (fc + 455kHz 或 fc - 455kHz)
    │
    ├── [中频滤波器]  455kHz 陶瓷滤波器 (CFWLA455kHzF10A, BW=10kHz)
    │
    ├── [中频放大器]  OPA2350 (增益20dB, 带宽38MHz, 轨到轨)
    │
    ├── [ADC驱动]  THS4521 单端转差分
    │
    ├── [ADC]  STM32H743 内置ADC1 (16bit, 2MSPS)
    │
    ├── [MCU处理]
    │       ├── 数字下变频 (DDC) → 中心频率455kHz
    │       ├── 鉴AM: 包络检波 ← |Hilbert{I/Q}| 提取包络
    │       ├── 鉴FM: 正交鉴频 ← arctan(Q/I) 微分
    │       ├── 调制方式识别 (AM:包络变化 / FM:频率变化 / CW:恒定)
    │       └── ma / mf / Δfmax 计算
    │
    └── [输出]  TFT-LCD + 解调音频输出 (DAC)
```

### 2. 前端调理电路

**2.1 前置LNA**
- BGA2818: SiGe宽带LNA
  - 频率范围: DC~2.2GHz
  - 增益: 20.5dB @ 10MHz, 19.4dB @ 30MHz
  - NF: 3.2dB @ 900MHz (低频下更优)
  - OIP3: 30dBm
  - 输入阻抗: 50Ω (内置匹配)
  - 偏置: 外部电阻配置, 电流≈12mA @ 3.3V

**2.2 可调BPF**
- 类型: 变容二极管调谐BPF
- 拓扑: 2阶LC耦合谐振
- 变容管: BB201 (30pF@1V ~ 3pF@8V)
- 调谐电压: 由DAC输出 + 运放放大至0~8V
- 中心频率调谐范围: 8~35MHz
- 3dB带宽: ≈ fc/10 (自适应)

**2.3 本振 (LO)**
- ADF4351: 宽带PLL频率合成器
  - 频率范围: 35MHz~4.4GHz (分频扩展)
  - 输出功率: -4dBm ~ 5dBm 可调
  - 相位噪声: -134dBc/Hz @ 100kHz offset (10MHz载波)
  - 频率步进: 可低至1kHz
  - 配置: 3线SPI
  - 本振频率: f_LO = f_RF ± 455kHz (选择低中频)
- 参考源: TCXO 10MHz (±1ppm)

**2.4 混频器**
- SA612A: 双平衡Gilbert Cell混频器
  - 输入频率: DC~500MHz
  - 转换增益: 17dB @ 45MHz
  - NF: 6dB @ 45MHz
  - 本振输入: -10~+3dBm (ADF4351提供0dBm)
  - 输出阻抗: 约1.5kΩ (高阻)

**2.5 中频滤波器**
- 陶瓷滤波器: 中心频率455kHz
- 带宽选择: 10kHz (AM/FM兼容)
- 插入损耗: 4dB typ
- 阻抗: 1.5kΩ / 1.5kΩ
- 阻带衰减: >40dB @ fc±50kHz

**2.6 中频放大器**
- OPA2350: 双路轨到轨运放
- 总增益: 20dB (10倍)
- 带宽: 38MHz (足够处理455kHz中频)

**2.7 ADC (内置)**
- STM32H743 ADC1: 16bit, 采样率 2MSPS
- 输入: 差分 (THS4521驱动)
- 满量程: ±1.65V (共模1.65V)

### 3. 后处理与算法

**3.1 ADC配置**
- 采样率: fs = 2MSPS
- 中频: f_IF = 455kHz
- 过采样比: fs / (2 × B_IF) = 2MHz / (2 × 10kHz) = 100 (IF带通采样)
- 每帧: N = 2048点 → 时间 = 1.024ms

**3.2 数字下变频 (DDC)**
- NCO频率: f_NCO = 455kHz
- 相乘产生 I/Q:
  - I[n] = x[n] × cos(2π·f_NCO·n/fs)
  - Q[n] = x[n] × sin(2π·f_NCO·n/fs)
- CIC半带滤波器 → 降采样至 200kSPS
- DDC后可获得零中频I/Q基带信号

**3.3 AM解调 (包络检波)**
- 包络: A[n] = √(I²[n] + Q²[n])
- DC分量去除: A_ac[n] = A[n] - mean(A)
- 调幅度: m_a = (A_max - A_min) / (A_max + A_min)
- 或: m_a = (max(A_ac) - min(A_ac)) / (2 × mean(A))

**3.4 FM解调 (正交鉴频)**
- 瞬时相位: φ[n] = atan2(Q[n], I[n])
- 解相位卷绕: unwrap(φ)
- 瞬时频率: f_inst[n] = (φ[n] - φ[n-1]) / (2π · T_s)
- 最大频偏: Δf_max = max|f_inst - f_c|
- 调频度: m_f = Δf_max / f_m

**3.5 调制方式识别**
| 特征 | AM | FM | CW |
|------|----|----|-----|
| 包络变化 | 显著(随调制信号) | 接近恒定 | 恒定 |
| 瞬时频率 | 恒定 | 显著变化 | 恒定 |
| 频谱形状 | 载波+一对边频 | 载波+多对边频 | 单载波 |
| 判决逻辑 | 包络方差>阈值 → AM | 频率方差>阈值 → FM | 其余→CW |

**3.6 解调输出**
- 通过DAC (H743内置12bit DAC) 输出解调后模拟音频
- 低通重建: 2阶RC LPF, fc=15kHz

### 4. 关键参数验证

**验证1：带通采样率选择**

> IF = 455kHz, 信号带宽 B = 10kHz (陶瓷滤波器限制)  
> 低通采样: fs ≥ 2 × (455+10/2)kHz = 920kHz → 选1MSPS  
> 实际配置: fs = 2MSPS (过采样提高SNR)  
> 过采样率: OSR = 2MHz / (2 × 10kHz) = 100  
> SNR提升: ΔSNR = 10·log₁₀(100) = 20dB  
> H743 ADC @ 16bit SNRtyp ≈ 85dB + 20dB = 105dB (理论)  
> 等效ENOB ≈ (105-1.76)/6.02 ≈ 17.1bit (实际受前端限制)

**验证2：AM调幅度测量误差**

> 假设ADC 12bit有效分辨率, 满量程2Vpp  
> 1LSB = 2V / 4096 = 488μV  
> 100mV信号经LNA(20dB)+混频(17dB)+IF滤波(-4dB)+IF放大(20dB)  
> 总增益 ≈ 53dB ≈ 447倍  
> ADC输入信号 ≈ 44.7Vpp??? → 需AGC  
> 实际: 需加入AGC或中频限幅放大  
> 修正: 加入IF AGC (AD8367) 自动增益控制  
> 假设AGC输出固定在1Vpp, 12bit ENOB  
> 包络幅度测量: 至少100个样点平均  
> 量化噪声: 488μV / √100 = 48.8μV  
> 对于 ma=0.2 (20%调幅): 包络变化≈200mV  
> SNR = 20·log(200mV/48.8μV) ≈ 72dB  
> 对应ma误差 ≈ 0.03% → 远小于0.1

**验证3：FM频偏测量分辨率**

> fs = 2MSPS, 采样间隔 = 0.5μs  
> 相位差分辨率: 来自 I/Q 幅度分辨率  
> 假设有效ENOB=12bit, I/Q各12bit  
> 瞬时频率测量分辨率:  
> δf ≈ 1/(2π · T_s · 2^ENOB) = 1/(2π · 0.5μs · 4096) ≈ 77.7Hz  
> 远小于1kHz的频偏测量要求

**验证4：载频范围与镜像抑制**

> RF范围: 10~30MHz, IF=455kHz  
> 镜像频率: f_image = f_RF + 2·f_IF = f_RF + 910kHz  
> 前级BPF需抑制910kHz外的镜像  
> 最差情况: f_RF=10MHz, f_image=10.91MHz  
> BPF在10.91MHz处需衰减 > 30dB  
> 二阶LC谐振BPF: Q ≈ 30 → 带宽 ≈ 333kHz  
> 对910kHz外衰减 ≈ -20dB × 2 = -40dB → 满足  
> 实际使用可加一阶高通预滤波进一步抑制镜像

### 5. 优缺点

**优点:**
- 单MCU完成所有数字处理, 架构简洁
- H743内置ADC性能足, 无需外部ADC
- 低成本455kHz陶瓷滤波器易获取
- SA612A混频器经典可靠, 外围简单

**缺点:**
- 低中频(455kHz)镜像抑制困难
- 需要外部PLL(ADF4351)产生本振, 增加BOM
- 内置ADC在高频段受片内数字噪声影响
- AM/FM解调全在MCU内实现, 计算负载较高
- 前端RF电路调试难度大 (PCB布局敏感)

## 方案B：MCU+专用芯片方案（STM32H723ZGT6 + AD8333 I/Q解调器）

### 1. 系统架构

```
RF输入 (10~30MHz, 100mVpp)
    │
    ├── [前置LNA]  BGA2818 (Gain=20dB)
    │
    ├── [可调BPF]  变容管调谐
    │
    ├── [混频器]  AD8347 (正交解调器)  
    │              RF输入: 0~2.4GHz
    │              内置LO正交分相 → 直接输出I/Q基带
    │              内置基带放大器 & LPF
    │              I/Q输出幅度: 可编程增益 0~42dB
    │
    ├── [基带LPF]  AD8347内置 (四阶Butterworth, fc可调1~100MHz)
    │
    ├── [ADC]  STM32H723内置ADC1/ADC2 (双通道同步)
    │           I路 → ADC1 (16bit, 1MSPS)
    │           Q路 → ADC2 (16bit, 1MSPS)
    │
    ├── [MCU: STM32H723ZGT6]
    │       ├── Hilbert变换 → 解析信号
    │       ├── 包络检波: √(I²+Q²) → AM解调
    │       ├── 鉴频: arctan(Q/I) 微分 → FM解调
    │       ├── 调制识别 (AM/FM/CW)
    │       └── ma/mf/Δf计算
    │
    ├── [本振]  ADF4351 PLL → AD8347 LO输入
    │
    └── [输出]  TFT-LCD + 蓝牙 + 解调音频DAC
```

### 2. 前端调理电路

**2.1 前置LNA**
- 同方案A: BGA2818

**2.2 可调BPF**
- 同方案A: 变容管调谐LC滤波器

**2.3 正交解调器 AD8347**
- 频率范围: 0~2.4GHz
- 解调精度: 相位误差 <1°, 幅度误差 <0.5dB
- 内置LO正交分相网络 (无需外部90°移相)
- 基带放大器: 增益范围 0~42dB (数字可编程, 3dB步进)
- 基带LPF: 4阶Butterworth, 截止频率可配 (1~100MHz)
- 输出共模: 0.5×VCC (约1.65V @ 3.3V)
- 噪声系数: 12dB @ LNA+混频器级联
- IIP3: +10dBm

**2.4 本振 (LO)**
- ADF4351: 同方案A
- LO频率: f_LO = f_RF (零中频) 或 f_LO = f_RF ± f_IF
- 本设计: 低中频模式 f_LO = f_RF - 500kHz (便于消除DC偏移和1/f噪声)

**2.5 ADC配置**
- STM32H723内置ADC1/ADC2: 双路16bit同步采样
- 采样率: 1MSPS (每通道)
- 触发源: TIM3_TRGO (同时触发两路ADC)
- 差分驱动: THS4521 将AD8347差分I/Q输出转为单端

### 3. 后处理与算法

**3.1 数字下变频**
- 对I/Q采样进行DDC (因为采用低中频, 需本振偏移后数字处理)
- CIC滤波器降采样至200kSPS

**3.2 AM解调**
- 包络: A[n] = √(I²[n] + Q²[n])
- AD8347直接输出正交I/Q, 无需软件NCO
- m_a 计算: 同方案A

**3.3 FM解调**
- 相位: φ[n] = atan2(Q[n], I[n])
- 鉴频: dφ/dt → 瞬时频率偏差
- AD8347的I/Q正交精度(<1°)确保解调质量

**3.4 AD8347优势**
- 集成LO分相: 省去外部90°移相器
- 内置可编程基带LPF: 适应不同调制频率
- 可编程增益: 自动调整信号幅度至ADC最佳输入范围

### 4. 关键参数验证

**验证1：零中频/低中频的镜像抑制**

> AD8347正交精度: 相位误差<1°, 幅度误差<0.5dB  
> 镜像抑制比 IRR = 1/(δ²/4 + θ²/4) 其中δ=dB误差, θ=弧度相位差  
> δ = 10^(0.5/20) - 1 ≈ 0.059 (幅度误差0.5dB)  
> θ = 1° = 0.01745 rad  
> IRR ≈ 1/(0.059²/4 + 0.01745²/4) ≈ 1/(0.00087 + 0.000076)  
> ≈ 1/0.000946 ≈ 1057 → 约30dB  
> 相比SA612A方案(无正交能力)提高约20dB

**验证2：ADC同步采样率**

> 两路ADC同时采样 I/Q @ 1MSPS  
> Nyquist要求: fs ≥ 2 × 10kHz (调制频率) = 20kSPS  
> 1MSPS过采样: OSR = 1M/20k = 50  
> 等效ENOB提升: Δ = 10·log₁₀(50) = 17dB  
> SNR ≈ 85 + 17 = 102dB → 约16.6bit有效  
> 满足ma/mf测量精度

**验证3：载频噪声对FM解调的影响**

> ADF4351相位噪声: -134dBc/Hz @ 100kHz offset  
> 积分PNoise(1kHz~10kHz): ≈ -100dBc  
> 对应频率抖动:  
> σ_f = fc × √(2 × ∫PNoise × df)  
> ≈ 10MHz × √(2 × 10^(-100/10) × 10kHz)  
> ≈ 10⁷ × √(2 × 10⁻¹⁰ × 10⁴)  
> ≈ 10⁷ × √(2×10⁻⁶) ≈ 10⁷ × 0.0014 = 14kHz  
> 这显然太大 → 需要真实相位噪声数据  
> ADF4351 @ 10MHz载波实际PNoise ≈ -155dBc/Hz @ 100kHz  
> 计算结果: σ_f ≈ 10⁷ × √(2 × 10⁻¹⁵·⁵ × 10⁴) ≈ 0.18Hz  
> 完全满足1kHz频偏测量

### 5. 优缺点

**优点:**
- AD8347集成正交解调, 大幅度减少RF调试难度
- 内置可编程LPF适应多调制频率
- I/Q直接输出, 省去软件NCO和DDC
- 正向增益高于方案A (集成基带放大器)
- 镜像抑制优于低中频方案

**缺点:**
- AD8347成本较高 (¥30+)
- QFN封装手工焊接困难
- 功耗较大 (AD8347 ≈ 85mA @ 3.3V)
- 零中频模式面临DC偏移和1/f噪声问题
- 需要两路ADC同时采样(占用H723双ADC)

## 方案C：FPGA异构方案（ZYNQ XC7Z010 + AD9361射频前端）

### 1. 系统架构

```
RF输入 (10~30MHz, 100mVpp)
    │
    ├── [射频前端]  AD9361 (集成收发器)
    │       RF输入: 70MHz~6GHz (含内置LNA+混频器+LPF+ADC)
    │       RX带宽: <56MHz 可配置
    │       ADC: 12bit, 最高61.44MSPS
    │       输出: 12bit 并行 I/Q 数据
    │
    ├── ┌─────────────────────────────────────────────┐
    │    │  ZYNQ XC7Z010 (FPGA + ARM Cortex-A9)         │
    │    │                                                │
    │    │  PS (ARM):                                     │
    │    │   • Linux运行控制/配置                         │
    │    │   • 调制方式识别算法                           │
    │    │   • ma/mf/Δf计算                              │
    │    │   • 人机交互 (LCD/蓝牙)                        │
    │    │                                                │
    │    │  PL (FPGA):                                   │
    │    │   • AD9361接口控制 (LVDS/CMOS)                │
    │    │   • 数字下变频 (NCO + CIC半带滤波器)           │
    │    │   • 硬件CORDIC → √(I²+Q²) / atan2(Q,I)       │
    │    │   • 连续FFT引擎 (频谱分析)                     │
    │    │   • AXI-DMA传输至PS DDR                       │
    │    └─────────────────────────────────────────────┘
    │
    └── [输出]  TFT-LCD + 蓝牙 + 解调DAC音频
```

### 2. 前端调理电路

**2.1 射频前端 AD9361**
- 频率范围: 70MHz~6GHz (需确认10MHz是否支持)
  - AD9361的最低RF频率为70MHz, 不直接支持10~30MHz
  - 解决方法: 前级插入上变频器将10~30MHz搬移至适合AD9361的频段
  - 或: 使用AD9364 (最低支持 70MHz) + 前级分频/变频
- 更实际的替代: AD8347 + ZYNQ (取消AD9361)
- **修正**: 改用分立前端 (LNA+混频器) + ZYNQ内部ADC

**2.2 修正方案（实用化ZYNQ方案）**

```
RF输入 → [LNA BGA2818] → [BPF] → [混频器 SA612A]
    │
    ├── [本振 ADF4351] → SA612A LO输入
    │
    ├── [IF陶瓷滤波器 455kHz/10.7MHz]
    │
    ├── [IF放大器 + ADC驱动]
    │
    ├── [外部ADC]  AD9226 (12bit, 65MSPS)
    │              若IF=10.7MHz: fs=40MSPS (带通采样)
    │              若IF=455kHz: fs=2MSPS
    │
    ├── ┌────────────────────────────────────────────┐
    │    │  ZYNQ XC7Z010                               │
    │    │   PL: DDC + CORDIC解调 + FFT                │
    │    │   PS: 识别算法 + 参数计算 + 显示控制        │
    │    └────────────────────────────────────────────┘
    │
    └── [输出]  TFT-LCD + 蓝牙
```

**2.3 更正后的前端设计（实用版）**

与方案A/B类似的前端:
- LNA: BGA2818
- BPF: 变容管可调
- 混频器: SA612A
- IF: 500kHz (适应AD9226高速采样)
- IF滤波器: LTC1562 可配置有源滤波器 (fc可调)
- ADC: AD9226 @ 10MSPS (过采样)

### 3. 后处理与算法

**3.1 ZYNQ PL (FPGA逻辑)**
- AD9226接口: 12bit并行, 10MSPS
- DDC: DDS NCO (500kHz) + 乘法器 + CIC降采样至2MSPS
- CORDIC引擎 (流水线):
  - 幅度: √(I²+Q²) → 包络检波
  - 相位: atan2(Q,I) → 鉴频
- FFT引擎 (频谱分析): 1024点 @ 2MSPS

**3.2 ZYNQ PS (ARM Cortex-A9)**
- 运行裸机或轻量RTOS
- 通信协议: SPI/UART控制ADF4351
- 调制方式识别: PL提取的特征 → 决策树判决
- 参数计算: m_a / m_f / Δf
- LCD显示 + 蓝牙通信
- 采样数据经AXI-DMA传输至DDR做长时间记录

### 4. 关键参数验证

**验证1：IF=500kHz带通采样**

> IF中心=500kHz, 带宽 B=10kHz  
> 低通采样: fs ≥ 2 × (500+5)kHz = 1010kHz  
> 实际 fs = 10MSPS (AD9226)  
> 过采样率: OSR = 10M/(2×10k) = 500  
> ΔSNR = 10·log₁₀(500) ≈ 27dB  
> AD9226 SNR=70dB + 27dB = 97dB → ENOB≈15.8bit

**验证2：CORDIC硬化解调延迟**

> Xilinx CORDIC IP核 (流水线模式):  
> 幅度计算: 16级流水线, 每级1时钟@100MHz → 160ns  
> 相位计算: 16级流水线 → 160ns  
> 相比MCU软件实现(atan2 ≈ 2μs @168MHz): 加速12.5倍  
> 实现真正的实时解调

**验证3：ZYNQ PL并行FFT**

> 1024点FFT @ 100MHz (IP核流水线): ≈ 12μs  
> 100Hz调制频率下, 1个调制周期=10ms  
> 单一周期可做: 10000/12 ≈ 833次FFT  
> 可实时跟踪调制变化

### 5. 优缺点

**优点:**
- FPGA硬件CORDIC实现超低延迟解调
- ZYNQ PL+PS异构, PL做实时处理, PS做应用层
- 可扩展性强 (频谱监测、数字下变频、自适应滤波)
- 过采样率高达500倍, 分辨率远超需求
- 适合复杂的调制识别算法 (可上ML)

**缺点:**
- ZYNQ开发难度最大 (FPGA逻辑+ARM双系统)
- BOM成本最高 (ZYNQ ¥100+)
- AD9361不直接支持10MHz, 需要额外变频级
- PCB设计难度极大 (BGA封装, RF布局)
- 功耗大 (~1W+)
- 团队需同时掌握FPGA和ARM开发技能

## 三方案对比总结

| 维度 | 方案A (纯MCU+超外差) | 方案B (MCU+AD8347) | 方案C (ZYNQ异构) |
|------|-------------------|-----------------|----------------|
| 中频架构 | 低中频 455kHz | 低中频/零中频 | 500kHz + DDC |
| 混频/解调 | SA612A + MCU软件Hilbert | AD8347硬件正交I/Q | FPGA CORDIC硬件 |
| ADC | 内置16bit@2MSPS | 内置双16bit@1MSPS | 外置12bit@10MSPS |
| 等效SNR | ~85dB | ~90dB(正交增益) | ~97dB(过采样) |
| 调制识别 | 软件FFT+特征 | 软件FFT+特征 | 硬件FFT加速 |
| 解调延迟 | ~2ms | ~1ms | ~0.2ms |
| 开发难度 | ★★★ (中RF) | ★★★ (中RF) | ★★★★★ (高RF+FPGA) |
| BOM成本 | ~¥314 | ~¥312 | ~¥437 |

**推荐**:
- **方案A** (纯MCU+超外差) 是最均衡的选择, 用经典超外差架构+现代MCU软件解调, 成本可控, 适合大多数参赛队伍
- **方案B** (MCU+AD8347) 在硬件上简化了正交解调的难度, 适合RF设计经验不足但预算充足的队伍
- **方案C** (ZYNQ异构) 适合有FPGA开发经验、追求最高性能和最强扩展性的队伍, 但成本高、开发周期长

> **注**: 三个方案均基于超外差架构。方案A和B使用455kHz/500kHz低中频, 方案C可灵活配置中频。方案B的AD8347因集成LO分相和基带增益控制, 在硬件复杂度上最优, 但芯片获取难度和成本稍高。方案C的ZYNQ方案实际推荐使用前级分立混频+AD9226+ZYNQ的路径, 而非AD9361直接射频采样, 以适配10~30MHz载频范围。
