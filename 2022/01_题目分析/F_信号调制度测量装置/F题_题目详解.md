# 2022年 TI 杯全国大学生电子设计竞赛（省赛） - 信号及仪器仪表类题目详解

## F题：信号调制度测量装置

### 一、任务概述
设计制作一个**信号调制度测量装置**，能够测量并显示信号源输出的被测信号的调制度等参数，识别并显示被测信号的调制方式（AM、FM、未调载波CW），并输出解调信号。

**核心难点**：
- **高频信号处理**：载频高达10MHz~30MHz，传统MCU难以直接采样，需采用**带通采样**或**模拟解调+低频采样**方案。
- **调制方式自动识别**：装置需自主判断信号是AM、FM还是CW，这要求算法具备一定的智能性。
- **参数测量精度**：调幅度ma误差≤0.1，调频度mf误差≤0.3，这对模拟电路精度和算法校准提出了高要求。

### 二、基本要求详解
1. **AM信号测量**：
   - **被测信号**：电压峰峰值100mV的普通单音调幅（AM）波。
   - **载频**：10MHz；**调制信号**：频率1~3kHz的正弦信号。
   - **调幅度（ma）**：范围 $0.2 < m_a \leq 1$，测量误差绝对值 $|\Delta| \leq 0.1$。
     - *深刻解释*：
       - $m_a = \frac{V_{max} - V_{min}}{V_{max} + V_{min}}$，反映载波被调制的深度。$m_a=1$时为满调制（过调制会导致包络失真）。
       - 误差≤0.1意味着若实际$m_a=0.5$，测量值必须在0.4~0.6之间。这对包络检波器的线性度和ADC精度要求极高。
   - **解调输出**：输出解调后的1~3kHz正弦波，要求波形无明显失真。

2. **FM信号测量**：
   - **被测信号**：电压峰峰值100mV的单音调频（FM）波。
   - **载频**：10MHz；**调制信号**：频率3~5kHz。
   - **调频度（mf）**：范围 $1 < m_f \leq 6$，测量误差绝对值 $|\Delta| \leq 0.3$。
     - *深刻解释*：
       - $m_f = \frac{\Delta f_{max}}{f_m}$，即最大频偏与调制频率之比。例如，若$f_m=3kHz$，$m_f=5$，则$\Delta f_{max}=15kHz$。
       - FM信号频谱包含无穷多对边频，其幅度由贝塞尔函数决定。测量mf通常通过分析频谱结构或鉴频后的幅度关系。
   - **最大频偏（$\Delta f_{max}$）**：要求测量并显示，单位为kHz。
     - *解释*：$\Delta f_{max}$是FM波瞬时频率偏离载频的最大值。可通过鉴频器将频率变化转换为电压变化后测量。
   - **解调输出**：输出解调后的3~5kHz正弦波，无明显失真。

### 三、发挥部分详解
1. **频率范围扩展**：
   - **载频**：10MHz ~ 30MHz（步进0.5MHz）。
   - **调制频率**：5kHz ~ 10kHz（步进1kHz）。
   - *深刻解释*：
     - 载频扩展到30MHz，意味着谐波分量更高。若采用数字方案，ADC采样率需满足带通采样定理；若采用模拟方案，包络检波器和鉴频器的带宽需足够。
     - 调制频率提高到10kHz，对解调输出端的低通滤波器截止频率提出更高要求（需>10kHz）。

2. **自动识别**：
   - 装置应能**自主识别**$u_m$的调制方式（调幅、调频或未调载波CW）。
   - *深刻解释*：
     - **识别原理**：
       - **CW（连续波）**：频谱仅含载频单峰，包络为恒定直流。
       - **AM**：频谱含载频+一对对称边频（间隔为调制频率），包络随调制信号变化。
       - **FM**：频谱含载频+多对边频（幅度按贝塞尔函数分布），包络恒定但瞬时频率变化。
     - 实现方法：可通过FFT分析频谱结构（AM边频少且对称，FM边频多且多对），或通过包络检波+鉴频器组合判断。

3. **一键启动**：
   - 按下启动键后，装置应在10秒内连续完成调制方式识别、参数测量和显示，期间不得有人工干预。
   - *解释*：考验系统的自动化程度和算法效率。

### 四、关键参数解析
- **带通采样定律**：对于中心频率$f_c$、带宽$B$的带通信号，采样率$f_s$满足：$\frac{2f_c + B}{k+1} \leq f_s \leq \frac{2f_c - B}{k}$（k为整数）。合理选择k可使$f_s$远低于$2f_c$。
- **贝塞尔函数与FM频谱**：FM信号的频谱分量幅度$J_n(m_f)$由第一类贝塞尔函数决定。当$m_f$较小时，主要能量集中在载频和第一对边频；当$m_f$增大时，高阶边频能量增加。
- **鉴频器设计**：常用方法包括**斜率鉴频**（微分+包络检波）、**相位鉴频**（相移网络+乘法器）或**锁相环鉴频**（PLL）。

## D题：混沌信号发生器

### 一、任务概述
设计并制作一个能够产生**混沌信号**的发生器。混沌信号是一种由确定性系统产生的、具有类随机特性的非周期信号。

**核心难点**：
- **电路稳定性**：蔡氏电路等混沌电路对参数敏感，需精确调节才能进入混沌态而非周期态。
- **波形观测**：需在示波器X-Y模式下观察到典型的**混沌吸引子**（如双涡旋结构）。
- **状态切换**：通过调节参数，使电路在**平衡态、周期态、混沌态**之间可控转换。

### 二、基本要求与参数详解
1. **电路形式**：
   - 通常采用**蔡氏电路（Chua's Circuit）**，由两个电容、一个电感、一个线性电阻和一个**非线性负阻**（蔡氏二极管）组成。
   - *深刻解释*：
     - 蔡氏电路是首个被实验验证的、用简单电子元件实现混沌的电路。
     - **非线性负阻**是产生混沌的关键。其伏安特性呈分段线性，包含三个区域（两个负斜率区和一个正斜率区），使得系统具有折叠和拉伸相空间的能力。

2. **混沌吸引子观测**：
   - 在示波器上观察$v_{C1}$-$v_{C2}$（或$i_L$-$v_{C1}$）相图，应能看到**双涡旋（Double Scroll）**结构。
   - *解释*：双涡旋是蔡氏电路最经典的混沌吸引子，形似两个相连的漩涡，代表系统状态在两个不稳定平衡点之间不断切换但永不重复。

3. **状态可调性**：
   - 通过改变线性电阻$R$的值，电路应能从**稳定平衡点**（阻尼振荡）→**周期振荡**（极限环）→**混沌振荡**→**超混沌/周期窗**转换。
   - *深刻解释*：
     - 随着$R$减小，电路能量损耗减少，系统逐渐失稳。
     - **费根鲍姆常数**：在倍周期分岔通往混沌的路径中，分岔间隔之比趋近于费根鲍姆常数$\delta \approx 4.669$。若实现倍周期分岔路径，可观察到2周期、4周期、8周期...直至混沌。

### 三、关键参数解析
- **李雅普诺夫指数（Lyapunov Exponent）**：衡量系统对初始条件的敏感依赖性。至少存在一个正李雅普诺夫指数是混沌的判据之一。
- **分岔图**：以电路参数（如$R$）为横轴、状态变量采样值为纵轴绘制的图，可直观展示周期到混沌的演变过程。
- **非线性负阻的实现**：常用两个运算放大器（如OP07、TL082）和若干电阻构成**蔡氏二极管**，精确实现分段线性负阻特性。
