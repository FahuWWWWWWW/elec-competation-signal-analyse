# 2023年电赛C题「电感电容测量装置」详解

## 一、题目复述与指标拆解

### 1.1 任务描述

基于TI公司的MCU，设计并制作电感及其品质因数Q、电容及其损耗角正切D的测量装置。被测元件接入，一键启动后，在规定时间内自动完成测量。测量装置要提供专用于监测测试频率的信号输出接口，用于实时监测装置的测试频率。

### 1.2 指标拆解

| 指标项 | 基本要求 | 发挥部分 | 考核本质 |
|--------|----------|----------|----------|
| **电容量** | 1nF~100nF, 误差≤5% | — | **谐振/阻抗法测C** |
| **电容D值** | 0.005~1, 误差≤5% | — | **损耗角正切测量** |
| **测试频率** | 1kHz~100kHz (固定) | 同基本 | **频率选择影响精度** |
| **测量时间** | ≤1s | ≤5s | **算法效率** |
| **电感量** | — | 10μH~100μH, 误差≤5% | **谐振/阻抗法测L** |
| **电感Q值** | — | 1~200, 误差≤5% | **Q = ωL/R** |
| **MCU限制** | **必须使用TI MCU** | 同基本 | **TI DSP库/ADC** |
| **校准** | 自备仪器校准 | 同基本 | **误差溯源** |

### 1.3 核心测量对象

| 被测量 | 物理本质 | 测量方法 |
|--------|---------|---------|
| **电容C** | 存储电荷能力 | LC谐振法 / 阻抗电压法 |
| **损耗角正切D** | 介质损耗: D = tan(δ) = R/(1/ωC) = ωRC | 相位测量 / 实部虚部分离 |
| **电感L** | 存储磁场能力 | LC谐振法 / 阻抗电压法 |
| **品质因数Q** | 储能与损耗比: Q = ωL/R = 1/D | 同D值测量 |

**重要关系**: Q = 1/D（对于理想单一元件）

## 二、核心物理原理

### 2.1 实际电容器模型

理想电容: Zc = 1/(jωC)

实际电容: **串联等效模型**
```
    Rs    C
    ─┬────┤├────
     │         │
    ===        │
```
阻抗: Z = Rs + 1/(jωC) = Rs - j/(ωC)

损耗角正切: **D = tan(δ) = Rs · ωC**

> **物理意义**: D表示电容器的能量损耗。D=0.005 (低损耗薄膜电容) ~ D=1 (高损耗电解电容)。

### 2.2 实际电感器模型

理想电感: ZL = jωL

实际电感: **串联等效模型**
```
    R    L
    ─┬──∩∩∩──
     │      │
    ===     │
```
阻抗: Z = R + jωL

品质因数: **Q = ωL / R**

损耗角正切: **D = 1/Q = R/(ωL)**

> **物理意义**: Q表示电感器的能量效率。Q=200 (高品质射频电感) ~ Q=1 (低品质工频电感)。

### 2.3 阻抗测量法 (电压-电流法)

**原理**: 向被测元件注入已知频率ω、已知幅度I的正弦电流，测量两端电压V的幅度和相位。

$$Z = \frac{V}{I} = |Z| \cdot e^{j\theta}$$

**通过阻抗计算参数**:

对于电容:
- |Z| = √(Rs² + (1/ωC)²) ≈ 1/(ωC) （当D<<1时）
- θ = -arctan(1/(Rs·ωC)) = -arctan(1/D)
- **C = 1/(ω · |Z| · sin|θ|)** 或 **C ≈ 1/(ω · |Z|)**（低D时）
- **D = 1/tan|θ|** 或 **D = Re(Z)/Im(Z)**

对于电感:
- |Z| = √(R² + (ωL)²) ≈ ωL（当Q>>1时）
- θ = arctan(ωL/R) = arctan(Q)
- **L = |Z|/(ω · sinθ)** 或 **L ≈ |Z|/ω**（高Q时）
- **Q = tanθ** 或 **Q = Im(Z)/Re(Z)**

### 2.4 LC谐振法

**原理**: 将被测元件与已知参考元件组成LC谐振回路，测量谐振频率f0。

**测电容** (已知L):
$$f_0 = \frac{1}{2\pi\sqrt{LC}} \Rightarrow C = \frac{1}{(2\pi f_0)^2 L}$$

**测电感** (已知C):
$$f_0 = \frac{1}{2\pi\sqrt{LC}} \Rightarrow L = \frac{1}{(2\pi f_0)^2 C}$$

**损耗测量**: 
- 测量谐振峰的幅度和3dB带宽BW
- Q = f0 / BW
- D = 1/Q

**优缺点**:
- ✅ 电路简单，只需测量频率
- ❌ 需要精确已知参考元件
- ❌ 谐振频率范围可能很宽（1nF+10μH → f0≈1.6MHz）
- ❌ 无法直接测量D/Q，需要额外电路测带宽

### 2.5 频率选择策略

| 被测参数 | 推荐频率 | 原因 |
|---------|---------|------|
| **1nF电容** | ~100kHz | Xc = 1/(ωC) ≈ 1.6kΩ，便于电压测量 |
| **100nF电容** | ~10kHz | Xc ≈ 160Ω，仍远大于导线电阻 |
| **10μH电感** | ~100kHz | XL = ωL ≈ 6.3Ω，需恒流源驱动 |
| **100μH电感** | ~50kHz | XL ≈ 31Ω |

**策略**: 选择频率使被测阻抗在100Ω~10kΩ范围内（最佳ADC测量范围）。

## 三、调理电路方案设计

### 3.1 推荐方案：阻抗电压法 + DFT相位检测

```
[MCU - TI TMS320F280025 或 MSP430FR5994]
    |
    +---> [DDS / PWM] -- 产生测试频率正弦波 (1kHz~100kHz)
    |         |
    |         v
    |    [低通滤波器] -- 滤除PWM谐波，得到纯净正弦
    |         |
    |         v
    |    [VI转换器] -- 恒流源，向被测元件注入已知电流 I
    |         |
    |         +---> [被测元件 Zx] -- 电容或电感
    |         |         |
    |         |         v
    |         |    [电压测量点 Vx]
    |         |         |
    |         |         v
    |         |    [仪表放大器] -- 差分放大，抑制共模
    |         |         |
    |         |         v
    |         |    [ADC采样] -- 同步采样电压和电流参考
    |         |         |
    v         v
[DFT算法] -- 计算V和I的幅度比 + 相位差
    |
    v
[阻抗计算] -- Z = V/I = |Z|∠θ
    |
    v
[参数提取] -- C/L/D/Q
    |
    v
[LCD显示]
```

### 3.2 关键电路模块

#### (1) 信号源：DDS vs PWM

**方案A: DDS (AD9833)**
- 优点: 纯正弦波，频率精确，幅度稳定
- 缺点: 需要SPI控制，成本~¥15

**方案B: MCU PWM + 低通滤波**
- 原理: MCU产生PWM，经RC低通滤波成正弦
- 优点: 零成本（使用TI MCU内置PWM）
- 缺点: 谐波需要充分滤除，频率分辨率受限于PWM频率

**方案C: 软件DAC (DMA + 查找表)**
- 原理: MCU通过DMA定时器输出正弦查找表到DAC
- 优点: 波形纯净，频率精确
- 缺点: 占用DMA资源

**推荐（电赛）**: 方案B（PWM）或方案C（软件DAC），充分利用TI MCU资源。

#### (2) 恒流源 (VI转换器)

**方案A: 运放恒流源**
```
          R1
    Vin ──/\/\───┬─── Iout
                 │
                ┌┴┐
                │ │ Rload (被测元件)
                └┬┘
                 │
                GND
```
Iout = Vin/R1（若运放构成Howland电流泵）

**方案B: 电阻限流**
- 简单串联大电阻Rs >> |Zx|，则I ≈ Vin/Rs
- 缺点: 需要精确知道Rs，且Zs会引入误差

**推荐**: 方案A（Howland电流泵）
- 器件: OPA365 (TI)
- 电流幅度: 1mA ~ 10mA（避免元件饱和/发热）

#### (3) 电压测量与放大

**仪表放大器**: INA333 (TI) 或自搭三运放仪表放大器
- 增益: 1~1000可调（PGA）
- 输入阻抗: 高阻（不加载被测元件）
- CMRR: >80dB

**抗混叠滤波器**: RC低通，截止频率>测试频率×2
- 防止ADC采样时的混叠

#### (4) ADC同步采样

**方案A: 双ADC同步**
- 同时采样电压和电流参考
- 优点: 无相位误差
- 缺点: 需要两个ADC通道严格同步

**方案B: 单ADC + 电阻参考**
- 通过已知参考电阻Rs计算电流: I = Vref/Rs
- 优点: 只需测量电压
- 缺点: 需要精确知道Rs

**推荐**: 方案B（单ADC），通过精确参考电阻间接测电流。

### 3.3 DFT相位检测算法

```python
def measure_impedance(v_samples, i_ref_samples, fs, N):
    """
    通过DFT测量阻抗的幅度和相位
    
    参数:
        v_samples: 被测元件两端电压采样序列
        i_ref_samples: 电流参考采样序列（或通过Rs计算）
        fs: 采样率
        N: DFT点数
    
    返回:
        Z_mag: 阻抗幅度
        Z_phase: 阻抗相位 (度)
    """
    # 1. 加窗减少频谱泄漏
    window = np.hanning(N)
    v_windowed = v_samples * window
    i_windowed = i_ref_samples * window
    
    # 2. DFT计算基波分量
    k = int(f_test / fs * N)  # 测试频率对应的bin
    
    V_complex = np.sum(v_windowed * np.exp(-2j*np.pi*k*np.arange(N)/N))
    I_complex = np.sum(i_windowed * np.exp(-2j*np.pi*k*np.arange(N)/N))
    
    # 3. 计算阻抗
    Z = V_complex / I_complex
    Z_mag = np.abs(Z)
    Z_phase = np.angle(Z, deg=True)
    
    return Z_mag, Z_phase
```

## 四、关键算法设计

### 4.1 电容测量算法

```python
def measure_capacitor(v_samples, i_samples, fs, N, f_test):
    Z_mag, Z_phase = measure_impedance(v_samples, i_samples, fs, N, f_test)
    
    # 电容: Z = Rs - j/(ωC)
    # Re(Z) = Rs, Im(Z) = -1/(ωC)
    omega = 2 * np.pi * f_test
    
    Rs = Z_mag * np.cos(np.radians(Z_phase))
    Xc = -Z_mag * np.sin(np.radians(Z_phase))  # 容抗，应为负值
    
    C = 1 / (omega * abs(Xc))
    D = abs(Rs / Xc)  # D = tan(δ) = Rs / |Xc| = Rs * ωC
    
    return C, D
```

### 4.2 电感测量算法

```python
def measure_inductor(v_samples, i_samples, fs, N, f_test):
    Z_mag, Z_phase = measure_impedance(v_samples, i_samples, fs, N, f_test)
    
    # 电感: Z = R + jωL
    # Re(Z) = R, Im(Z) = ωL
    omega = 2 * np.pi * f_test
    
    R = Z_mag * np.cos(np.radians(Z_phase))
    Xl = Z_mag * np.sin(np.radians(Z_phase))  # 感抗，应为正值
    
    L = Xl / omega
    Q = Xl / R if R > 0 else np.inf
    
    return L, Q
```

### 4.3 自动量程切换

| 被测范围 | 推荐频率 | PGA增益 | 原因 |
|---------|---------|---------|------|
| 1~10nF | 100kHz | 10 | Xc≈1.6~16kΩ |
| 10~100nF | 10kHz | 1 | Xc≈160~1.6kΩ |
| 10~30μH | 100kHz | 100 | XL≈6~19Ω |
| 30~100μH | 50kHz | 10 | XL≈9~31Ω |

**自动量程策略**:
1. 先用高频率(100kHz)+高增益测量
2. 如果ADC溢出 → 降低增益
3. 如果ADC分辨率不足 → 降低频率
4. 通过软件判断最佳量程

## 五、设计要点与陷阱

### 5.1 测试频率的选择陷阱

**错误**: 选择100kHz测量100nF电容
- Xc = 1/(2π×100kHz×100nF) = 15.9Ω
- 导线电阻(~0.1Ω)和接触电阻(~0.01Ω)的影响: 0.1/15.9 = 0.6%
- D值测量误差会很大

**正确**: 选择10kHz测量100nF电容
- Xc = 159Ω，导线电阻影响仅0.06%

**原则**: 选择频率使|Z| > 100Ω（避免导线电阻影响）。

### 5.2 D值测量的难点

D值很小(0.005)意味着:
- 相位角接近-90°（电容）或+90°（电感）
- 相位测量误差1° → D值误差约3.5%
- **需要精确的相位检测**

**解决方案**:
1. 高分辨率ADC（16-bit）
2. 大量DFT点数（N≥1024）
3. 多次测量取平均
4. 精密校准（用已知低D标准件）

### 5.3 Q值测量的难点

Q=200意味着:
- 相位角接近90°
- 等效串联电阻R = ωL/Q = 2π×50kHz×100μH/200 = 0.157Ω
- 需要测量小于0.2Ω的电阻！

**解决方案**:
1. 恒流源注入大电流(>10mA)，产生足够大的电压降
2. 高增益仪表放大器(×1000)
3. 四线制(Kelvin)测量消除接触电阻

## 七、总结

2023-C题是一道**经典阻抗测量题目**，其核心是：

1. **阻抗测量法**: 通过电压-电流法测量|Z|和θ，然后计算C/L/D/Q
2. **频率选择**: 必须根据被测参数范围选择最优测试频率
3. **相位精度**: D/Q值测量本质是相位测量，需要高精度DFT
4. **TI MCU限制**: 必须充分利用TI的ADC和DSP资源

**关键技术路径**:
1. 信号源: PWM+低通 或 DMA+DAC
2. 恒流源: Howland电流泵
3. 电压测量: 仪表放大器+ADC
4. 算法: DFT提取基波幅度和相位
5. 计算: Z=V/I → C/L/D/Q

**与前序题目的关系**:
- 2023-B: TDR时域反射 → **时域信号处理**
- 2023-C: 阻抗测量 → **频域信号处理**
- 两者互补，分别训练时域和频域分析能力
- 2023-D: 调制识别 → **频域分析进阶**

**最深刻的认识**: 阻抗测量是一切元件参数测量的基础。从简单的万用表到高端的阻抗分析仪，核心都是"测电压、算阻抗、提取参数"。
