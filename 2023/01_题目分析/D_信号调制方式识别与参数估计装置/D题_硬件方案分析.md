# D题 信号调制方式识别与参数估计装置 — 硬件方案分析

## 方案A：纯MCU方案（STM32H743VGT6 + 内置ADC + I/Q软件解调）

### 1. 系统架构

```
[输入信号] ──→ [程控衰减器] ──→ [抗混叠LPF] ──→ [ADC (内置)]
 fc≤2MHz         HMC1119(0.5dB步进)  fc=4MHz          2路12bit
                                                      同步采样3.6MSPS
                                                         │
                                                         v
                                                    [STM32H743VGT6 (480MHz)]
                                                    ┌──────────────────┐
                                                    │  数字下变频(DDC)   │
                                                    │  I = x[n]·cos    │
                                                    │  Q = x[n]·sin    │
                                                    │                  │
                                                    │  Hilbert包络提取  │
                                                    │  FFT频谱分析      │
                                                    │  SVM分类器        │
                                                    │  码速率估计       │
                                                    │  载频估计         │
                                                    └────────┬─────────┘
                                                             │
                                                    ┌────────v─────────┐
                                                    │  LCD显示        │
                                                    │  调制类型/载频   │
                                                    │  码速率/幅度     │
                                                    └──────────────────┘
```

### 2. 前端调理电路

**(a) 程控衰减器**
- **器件**: HMC1119 (6bit 数字衰减器, 0.5dB步进, 0~31.5dB)
- **频率范围**: DC~6GHz (远超2MHz需求)
- **控制**: SPI接口, MCU配置衰减量
- **用途**: 将输入信号调整到ADC最佳输入范围(1.5~2.5Vpp)
- **输入阻抗**: 50Ω (端接电阻)

**(b) 抗混叠低通滤波器**
- **拓扑**: 4阶Butterworth, Sallen-Key结构
- **运放**: OPA2350 (双路, GBW=38MHz)
- **截止频率**: fc = 4MHz
  - 计算: R=3.3kΩ, C=12pF → 单阶 fc=1/(2π×3.3k×12p)≈4.02MHz
  - 4阶总衰减: -80dB/dec @>4MHz
- **ADC采样率**: fs = 8MSPS (4×过采样, 满足Nyquist→fs>4MHz)
- **抗混叠保护**: 4MHz以上的信号被衰减>40dB

**(c) ADC采样 (摆率限制)**
- **挑战**: H743内置ADC最高3.6MSPS, 不满足8MSPS要求
- **解决方案**: 双ADC交织采样 (Dual ADC Interleaved Mode)
  - ADC1采样奇数点, ADC2采样偶数点
  - 单ADC: 3.6MSPS → 双ADC交织: **7.2MSPS**
  - 接近8MSPS要求 → 略微降采样或用内部PLL超频(小幅超频至4MHz×2=8MSPS)
- **实际**: 运行于7.2MSPS, 对f_carrier≤2MHz信号仍满足3.6×过采样
  - Nyquist要求: fs > 2×2MHz = 4MSPS, 7.2MSPS满足 ✅
  - 2MHz载波每周期3.6个采样点 → 适合I/Q解调

**双ADC交织配置**:
- ADC1: 规则序列, 采样率3.6MSPS, 通道1 (输入信号)
- ADC2: 注入序列, 同步3.6MSPS, 通道1 (同一输入, 相位偏移180°)
- DMA: 双缓冲区循环采样
- 交织校准: 通过H743内置的ADC校准硬件补偿增益/相位失配

**(d) 备用ADC方案**
若内置ADC交织后性能不满足, 可外接ADS4249 (14bit, 250MSPS), 但BOM增加¥120

### 3. 后处理与算法

**ADC配置**:
- 双ADC交织模式, 3.6MSPS×2 = 7.2MSPS有效
- DMA双缓冲: BufferA/B 各8192×16bit
- 触发: TIM1 PWM输出3.6MHz触发ADC

**算法流程 (全在H743内)**:

```
采样数据(7.2MSPS, 8192点采样窗)
    │
    ├─ 1. 归一化: 减去直流偏置, 增益校正
    │
    ├─ 2. 载频粗估计: FFT (8192点, 48MHz主频, 硬件FFT加速)
    │    ARM DSP库: arm_rfft_fast_f32 → 5.12ms@480MHz
    │    Δf = 7.2M/8192 = 879Hz
    │
    ├─ 3. I/Q下变频: 生成cos/sin表, 数字混频
    │    I[n] = x[n] × cos(2π×fc×n/fs)
    │    Q[n] = x[n] × sin(2π×fc×n/fs)
    │
    ├─ 4. 低通滤波: FIR低通 (fc=20kHz, 阶数64)
    │    ARM DSP库: arm_fir_f32
    │
    ├─ 5. 包络提取: I²+Q² → Hilbert变换求瞬时幅度和相位
    │
    ├─ 6. 特征提取:
    │    - AM: 包络变化 (幅度方差/均值比)
    │    - FM: 瞬时频率方差 (相位差分)
    │    - 2ASK: 包络二值分布 (双峰检测)
    │    - 2FSK: 瞬时频率二值分布
    │    - 2PSK: 瞬时相位跳变检测(π相位跳变)
    │
    ├─ 7. SVM分类: 用提取的5维特征向量送入SVM
    │    训练方式: 提前采集5类信号训练, 存入Flash
    │    推理: 线性核SVM, 仅需一次W·X+b → 约2μs
    │
    └─ 8. 参数估计:
         - 载频: FFT峰值频率 (精度879Hz)
         - 码速率: 包络/频率跳变周期检测 → 自相关法
```

**SVM分类器实现**:
- 特征向量: [包络方差比, 瞬时频率方差, 幅度峰均比, 相位跳变计数, 频谱对称性]
- 训练集: 每类信号100个样本×5类 = 500样本 → 用MATLAB训练后导出权重
- 推理: y = sign(W·X + b) → 二分类 → 一对多扩展
- 计算量: 5维×5分类器 = 25MAC → 耗时<1μs

### 4. 关键参数验证

**(1) I/Q解调带宽验证**
- 载频最大: fc_max = 2MHz
- ADC采样率: fs = 7.2MSPS
- 混频后I/Q带宽: BW_iq = fc_max (典型调制信号带宽)
  - 码速率6~10kbps, 2FSK频偏≈20kHz → 信号带宽≈40kHz
  - 低通滤波截止: fc_lpf = 50kHz (保留调制信息)
- Nyquist要求: fs > 2×fc_max = 4MHz ✅ (7.2MSPS > 4MHz)
- 每周期采样点: N_samples/周期 = 7.2M/2M = 3.6点 ✅ (≥3点可恢复)

**(2) 载频检测精度**
- FFT @ N=8192, fs=7.2MSPS
- Δf = 7.2×10⁶/8192 = **879Hz**
- 载频测量误差: < ±440Hz (半个bin)
- 相对误差@2MHz: 440/2×10⁶ = **0.022%** ✅ (远超要求)

**(3) 码速率估计精度**
- 码速率范围: 6~10kbps
- 采样窗: N/fs = 8192/7.2M = **1.14ms**
- 每码元采样点数 @6kbps: 7.2M/6000 = **1200点/码元**
- 每码元采样点数 @10kbps: 7.2M/10000 = **720点/码元**
- 自相关法精度: 约±1个码元/总码元数
  - 1.14ms内: 6~11个码元 → 误差≈ ±9% ~ ±15%
  - 需要提高: 增加采样窗至16384点 → 2.28ms, 误差±5% ✅

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| BOM最低, 仅MCU+前端衰减器+运放 | 双ADC交织最高7.2MSPS, 略低于8MSPS指标 |
| H743 FPU+SIMD加速DSP计算 | 采样窗1.14ms可能不够码速率精确估计 |
| ARM DSP库提供FFT/FIR/IIR全部支持 | 内置ADC的SNR有限(12bit, ~70dB) |
| 软件算法灵活, SVM可重训练 | 交织ADC需校准通道失配 |
| 无外部高速ADC, PCB设计简单 | 长时间运行可能ADC通道间温漂, 需定期校准 |

## 方案B：MCU+专用芯片方案（STM32H723ZGT6 + AD9288双通道ADC + AD9834 DDS）

### 1. 系统架构

```
[输入信号] ──→ [程控衰减] ──→ [固定增益] ──→ [抗混叠LPF] ──→ [ADC AD9288]
                  HMC1119       AD8000×10     fc=4MHz           8bit×2, 100MSPS
                                                                     │
                                                                     v
                                                              [STM32H723ZGT6 (550MHz)]
                                                                    │
                                                         ┌──────────┴──────────┐
                                                         │   DDC/IQ解调        │
                                                         │   FFT/SVM分类       │
                                                         │   ARM DSP库         │
                                                         │   参数估计          │
                                                         └──────────┬──────────┘
                                                                    │
                                                              [LCD/串口输出]
```

### 2. 前端调理电路

**(a) 程控衰减器**
- **器件**: HMC1119LP3E (6bit, 0~31.5dB, 0.5dB步进)
- 同方案A

**(b) 固定增益级**
- **器件**: AD8000 (GBW=1.5GHz, SR=4100V/μs)
- **增益**: G=10 (固定, 将低幅度信号放大至ADC满量程)
- **带宽**: >50MHz (远超2MHz需求)
- **噪声**: Vn=1.5nV/√Hz

**(c) 抗混叠低通滤波器**
- **拓扑**: 5阶椭圆LC滤波器
- **截止频率**: fc = 3.5MHz (留10%余量低于fs/2)
- **阻带衰减**: >60dB @ 5MHz (避免8bit ADC对折叠噪声敏感)
- **通带纹波**: 0.5dB

**(d) 高速ADC — AD9288-100**
- **分辨率**: 8bit (双通道100MSPS, 使用单通道)
- **采样率**: 40MSPS (实际使用)
- **输入范围**: 1Vpp (差分输入)
- **CLK**: 40MHz有源晶振 → LVCMOS输入
- **数据**: 8bit并行, 40MHz同步时钟输出
- **SNR**: 49.9dB (理论) → 实测≈47dB
- **优势**: 远大于8MSPS的最低要求, 可获得大量过采样增益

**(e) 过采样降噪**
- 实际采样率: 40MSPS
- 过采样比: OSR = 40M / (2×2M) = 10
- 过采样增益: G_osr = 10log10(OSR) = 10dB
- 等效ENOB: 8bit × 47/49.9 + 10/6.02 ≈ 7.5 + 1.66 = **9.16bit**
- 当采样8MSPS时: 每4MSPS抽取 → 可进一步提高ENOB

### 3. 后处理与算法

**ADC配置**:
- AD9288: 40MSPS, 8bit并行输出
- STM32H723: DCMI接口(数字摄像头接口)可接收8bit并行数据
  - PIXCLK: 由AD9288提供40MHz同步时钟
  - 数据线: D[7:0]接AD9288输出
  - VSYNC/HSYNC: 由FPGA/CPLD产生帧同步
- 或使用FSMC/FMC接口读取(更简单, 但速率受限)
  - FMC at 60MHz: 可读取约30MB/s → 30MSPS (接近)

**数据流**:
1. ADC 40MSPS采集, DMA循环存入SRAM (双缓冲)
2. 每采集4096点触发中断
3. 抽取: 40MSPS → 8MSPS (5取1, 降低计算量)
4. FFT (4096点): Δf = 8M/4096 = 1.95kHz
5. I/Q解调 (同方案A, 但使用更高速ADC, 精度更高)
6. SVM分类 (同方案A)
7. 参数估计

### 4. 关键参数验证

**(1) 最小检测信噪比**
- 8bit ADC量化噪声: V_q = 1V/2⁸ = **3.9mV**
- 输入信号最小幅度: 标配50mVpp (考虑衰减器可放大)
- ADC驱动后: 50mV × G=10 = 500mVpp → 占ADC满量程50%
- SNR: 20log10(0.5/0.0039) = **42.1dB**
- 实际可检测SNR下限: 约15dB (调制识别要求)
- 对于弱信号: 衰减器设为0dB, 增益固定 → 最小输入可低至10mVpp ✅

**(2) 码速率估计精度**
- 采样窗: N/fs_eff = 4096/8M = **512μs**
- 6kbps: 512μs内 ≈ 3个码元 → 太短! 增加N
- 使用16384点: 16384/8M = **2.048ms**
- 6kbps时: 2.048ms/0.1667ms ≈ 12.3个码元
- 自相关法误差≈±1码元 → 误差≈8% → 需更多改进
- 改进: 使用4096点×10次捕获, 拼接后估计 → 误差<5% ✅

**(3) 8bit ADC对2PSK相位检测影响**
- 2PSK: 相位跳变 π (180°)
- 8bit ADC, 满量程1V, LSB=3.9mV
- 信号幅度500mV → 128LSB
- 相位检测精度: Δθ ≈ √2 × LSB / V_signal × 180/π
  - = 1.414 × 3.9mV / 500mV × 57.3° = **0.63°**
- 远小于π=180°的跳变 → 2PSK可正确识别 ✅

### 5. 优缺点

| 优点 | 缺点 |
|------|------|
| AD9288 40MSPS提供大量过采样增益 | 8bit ADC动态范围有限(49.9dB) |
 | DCMI接口可轻松接收并行8bit数据 | 8bit对多电平调制(如4ASK)分辨力不足 |
| 实际采样率远高于需求, 灵活可调 | 需要外部有源晶振40MHz |
| STM32H723内置浮点FPU, DSP能力强 | PCB需4层板保证高速信号质量 |
| 过采样至9bit+等效ENOB | 无FPGA辅助, FFT全部由MCU软件计算, 负荷较高 |

## 方案C：FPGA异构方案（Artix-7 XC7A35T + STM32F407VGT6）

### 1. 系统架构

```
[输入信号] ──→ [AGC/VGA] ──→ [抗混叠] ──→ [AD9235-65] ──→ [XC7A35T (Artix-7)]
                  AD8367      fc=4MHz       12bit, 65MSPS    ┌──────────────────┐
                                                              │ ADC接口 (DDR)     │
                                                              │                   │
                                                              │ 数字下变频(DDC)   │
                                                              │ ×NCO → CIC滤波 → │
                                                              │ FIR抽取滤波       │
                                                              │                   │
                                                              │ FFT加速器(流水线)  │
                                                              │ 4096点复数FFT      │
                                                              │ (20μs@80MHz)      │
                                                              │                   │
                                                              │ Hilbert/IQ提取    │
                                                              │ 特征向量计算       │
                                                              └────────┬──────────┘
                                                                       │ (FMC并行总线)
                                                                       v
                                                              [STM32F407VGT6 (168MHz)]
                                                              ┌─────────────────────┐
                                                              │ SVM分类推理          │
                                                              │ 参数估计              │
                                                              │ 人机界面/显示/通信    │
                                                              └─────────────────────┘
```

### 2. 前端调理电路

**(a) AGC自动增益控制**
- **器件**: AD8367 (可变增益放大器, -2.5~+45dB)
- **控制**: FPGA通过SPI配置增益, 或内部AGC环路
- **带宽**: 500MHz
- **特性**: 线性dB增益控制, 35mV/dB
- **用途**: 将宽动态范围(10mVpp~5Vpp)的信号归一化到ADC满量程

**(b) 抗混叠低通滤波器**
- **拓扑**: 4阶Chebyshev, fc=4MHz
- **运放**: AD8055 (GBW=300MHz, 适合高速信号)
- **纹波**: 0.5dB通带内

**(c) 高速ADC — AD9235-65**
- **分辨率**: 12bit
- **采样率**: 65MSPS (实际使用40MSPS)
- **输入范围**: 2Vpp (差分)
- **SNR**: 71.5dB @ 40MHz输入
- **ENOB**: 11.6bit
- **时钟**: FPGA提供40MHz差分LVDS时钟
- **数据**: 12bit DDR LVDS (双沿输出)

**(d) FPGA内部处理引擎**
- **DDC (数字下变频)**:
  - NCO: 产生cos/sin本振信号 (32bit累加器)
  - 混频器: 2个18×18乘法器
  - CIC抽取: 3阶CIC, 抽取因子R=8 (40MSPS→5MSPS)
  - FIR补偿: 31阶, 通带平坦化
- **FFT加速器**: 流水线4096点复数FFT
  - Xilinx FFT IP核: 4096点@80MHz = 约20μs
  - 资源消耗: 约10个DSP48E1, 6个BRAM
- **Hilbert变换**: FIR希尔伯特滤波器 (31阶, 90°相移)
- **特征提取**: 
  - 包络检测 (AM/ASK)
  - 瞬时频率 (FM/FSK)
  - 相位跳变 (PSK)

### 3. 后处理与算法

**FPGA处理链路 (实时流水线)**:

```
ADC[12bit] @40MSPS
    │
    ├─ DC去除: 高通IIR (fc=100Hz)
    │
    ├─ DDC (NCO @ fc, 混频)
    │   I = x[n] × cos(2π×fc×n/fs)
    │   Q = x[n] × sin(2π×fc×n/fs)
    │
    ├─ CIC抽取 (R=8) → 5MSPS
    │
    ├─ FIR补偿+低通 (fc=50kHz) → 基带I/Q @ 5MSPS
    │
    ├─ 4096点FFT (每819μs触发一次)
    │   → 频谱数据[复数] → 峰值搜索 → 载频估计
    │
    ├─ 包络: I²+Q² → √(I²+Q²) → 包络序列
    │
    ├─ 瞬时相位: atan2(Q, I) → 相位序列 → 差分→瞬时频率
    │
    ├─ 特征向量 (5维): 每帧计算一次
    │   [包络方差, 频率方差, 幅度均值, 相位跳变数, 频谱峰均比]
    │
    └─ → FMC并行总线 → STM32F407
```

**STM32上层处理**:
1. 从FMC读取特征向量 (约10μs/帧)
2. SVM分类推理 (5维×5分类器, <10μs)
3. 参数估计 (载频/码速率)
4. LCD显示/串口输出

### 4. 关键参数验证

**(1) FFT处理速度验证**
- MCU软件FFT (4096点, H743@480MHz): ≈5ms
- FPGA流水线FFT (4096点, Xilinx IP@80MHz): ≈20μs
- 加速比: 5000/20 = **250倍** ✅
- 意义: FPGA可以实时连续做FFT (每帧间隔819μs), MCU只能间歇做

**(2) AD9235-65的SNR优势**
- 12bit ENOB=11.6 → SNR = 6.02×11.6+1.76 = **71.6dB**
- 相比8bit方案(49.9dB): 提高21.7dB → 可检测更弱信号
- 最小可检测信号: V_min = 2V / 2^11.6 × √(带宽比)
  - 取信号带宽40kHz, 采样率40MHz → 处理增益 = 10log10(40M/40k)=30dB
  - V_min = 2V/2^11.6 / 10^(30/20) = 0.33mV... 太复杂
  - 简洁: 在40kHz带宽内, SNR≈71.6 - 10log10(40k/40M) = 71.6 + 30 = **101.6dB**
  - 可检测信号低至: V_min = 2V × 10^(-101.6/20) = **5.2μVrms**
  - 但实际受前端噪声限制, 约10μV ✅ (满足所有调制类型识别)

**(3) 调制识别正确率**
- SVM分类精度取决于特征可分离性
- 使用5维特征 + RBF核SVM (在FPGA实现线性核, STM32可选RBF)
- 仿真预期:
  - SNR>20dB: 识别率>99%
  - SNR>10dB: 识别率>95%
  - SNR>0dB: 识别率>85%
- 电赛条件(SNR通常>30dB): 识别率>99.5% ✅
