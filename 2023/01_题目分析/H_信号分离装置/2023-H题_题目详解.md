# 2023年电赛H题「信号分离装置」详解

## 一、题目复述与指标拆解

### 1.1 任务描述

设计并制作信号分离装置。双路信号源输出信号A和B（频率范围20kHz~100kHz，fA<fB，峰峰值均为1V），经增益为1的加法器产生混合信号C=A+B。信号C通过分离电路分离出信号A'和B'。要求信号A'和A、B'和B波形无失真，能连续稳定同频显示。

### 1.2 指标拆解

| 指标项 | 基本要求 | 发挥部分 | 考核本质 |
|--------|----------|----------|----------|
| **加法器** | 增益为1, C=A+B | — | **信号叠加** |
| **信号A/B波形** | 正弦波 | **正弦波或三角波** | **波形识别与重建** |
| **频率范围** | 50kHz/100kHz固定 + 10kHz整数倍 | **5kHz整数倍** | **频率检测分辨率** |
| **分离输出** | 峰峰值≥1V, 无失真 | 同基本 | **分离增益与保真度** |
| **分离时间** | — | ≤20s | **算法收敛速度** |
| **相位控制** | — | B'与A'相位差0°~180°, 分辨率5°, 误差≤5° | **数字移相** |
| **连接限制** | 加法器与分离电路只有C和地线 | — | **盲分离** |

### 1.3 核心限制条件

**最严格的限制**: 加法器与分离电路只有信号C和地线连接，**分离电路不知道A和B的频率、波形、相位**。

这意味着分离电路必须：
1. **盲检测**: 从混合信号C中自动识别两个频率
2. **盲分离**: 在不知道原始信号的情况下分离出A和B
3. **波形重建**: 对于三角波，不能简单用带通滤波（会丢失谐波）

## 二、核心物理原理

### 2.1 混合信号的频谱特性

$$C(t) = A\sin(2\pi f_A t + \phi_A) + B\sin(2\pi f_B t + \phi_B)$$

**频域特征**: 
- 两个离散的谱线，分别位于fA和fB
- 如果A和B都是正弦波，频谱纯净
- 如果A或B是三角波，频谱包含基波+奇次谐波

### 2.2 频率检测原理

**FFT法**:
- 对C采样N点，做FFT
- 在20~100kHz范围内搜索两个最大峰值
- 频率分辨率: Δf = fs/N
- 要分辨5kHz整数倍: N > fs/5kHz (若fs=500kHz, N>100)

**Goertzel算法**:
- 针对特定频率bin的DFT简化算法
- 比FFT更适合检测少量频率点
- 计算量小，适合MCU实时实现

**PLL频率扫描法**:
- 用VCO产生扫描频率
- 与C混频，检测哪个频率有最大相关
- 慢但精确

### 2.3 信号分离方法

#### 方法1: 数字带通滤波器 (正弦波)

```
C(t) → [BPF1@fA] → A'(t)
     → [BPF2@fB] → B'(t)
```

- 优点: 简单，易于实现
- 缺点: 
  - 频率变化时需要重新设计滤波器系数
  - 过渡带会引入失真
  - **无法处理三角波**（滤波后只剩基波，三角波变正弦波）

#### 方法2: 正交解调 (I/Q相干检测)

```
C(t) → [×cos(ωAt)] → LPF → I_A (同相分量)
     → [×sin(ωAt)] → LPF → Q_A (正交分量)
     
A'(t) = √(I_A² + Q_A²) · sin(ωAt + arctan(Q_A/I_A))
```

- 优点: 
  - 可精确重建任意波形的基波分量
  - 可同时提取幅度和相位
  - 数字实现简单（查表法sin/cos）
- 缺点: 
  - 对于三角波，只能重建基波（三角波变正弦波）

#### 方法3: 锁相环PLL跟踪

```
C(t) → [PLL1] → 锁定fA，输出同频同相参考
     → [PLL2] → 锁定fB，输出同频同相参考
```

- 优点: 
  - 自动跟踪频率变化
  - 输出纯净的正弦参考
  - 可同时提取幅度信息
- 缺点: 
  - 需要两个PLL
  - 初始频率不确定时锁定困难
  - 三角波谐波可能干扰PLL

#### 方法4: 自适应滤波 (LMS)

```
C(t) → [自适应滤波器W1(z)] → A'(t)
     → [自适应滤波器W2(z)] → B'(t)
```

- 原理: 最小均方误差调整滤波器系数
- 优点: 不需要先验频率信息
- 缺点: 收敛慢，需要参考信号

### 2.4 三角波的分离挑战

三角波的傅里叶级数:
$$x_{triangle}(t) = \frac{8}{\pi^2}\sum_{n=1,3,5...}^{\infty} \frac{(-1)^{(n-1)/2}}{n^2} \sin(n\omega t)$$

**基波幅度**: 8/π² ≈ 0.81
**3次谐波幅度**: 0.81/9 ≈ 0.09
**5次谐波幅度**: 0.81/25 ≈ 0.032

**分离策略**:
1. 检测基波频率
2. 判断波形类型（通过谐波含量分析）
3. 如果是三角波：
   - 方案A: 提取基波，然后用三角波发生器重建（需要知道幅度和相位）
   - 方案B: 同时提取基波+3次谐波+5次谐波，叠加重建

## 三、调理电路方案设计

### 3.1 推荐方案：FFT频率检测 + I/Q正交解调 + 波形重建

```
[混合信号C输入]
    |
    v
[抗混叠LPF]  -- fc>100kHz (保护最高频率)
    |
    v
[ADC @ 500kSPS]  -- 至少2.5倍过采样
    |
    v
[频率检测模块]
    ├── FFT/Goertzel检测两个频率峰值
    └── 谐波分析判断波形类型(正弦/三角)
    |
    v
[分离模块]
    ├── I/Q正交解调提取A和B的基波
    ├── 幅度/相位估计
    └── 波形重建(正弦直接输出，三角用查找表)
    |
    v
[DAC输出]  -- A'和B'，峰峰值≥1V
    |
    v
[放大器]  -- 增益恢复至1Vpp
```

### 3.2 关键电路模块

#### (1) 加法器

**要求**: 增益为1，两路输入隔离，输出C=A+B

**电路**: 
```
    R    R
A ─/\/\──┬──/\/\── B
         │
        [运放跟随器]
         │
         C
```

或使用**反相加法器**:
```
         Rf
    ────┤├────
         │
    R1   │   R2
A ─/\/\─┼──/\/\─ B
         │
       [OPA]  (反相输入)
         │
         C
```

**关键**: R1=R2=Rf，实现增益为1。

#### (2) 抗混叠滤波器 + ADC

**ADC采样率**: ≥250kSPS（Nyquist要求>200kHz，2.5倍过采样）
**抗混叠**: 4阶Butterworth, fc=120kHz

#### (3) 频率检测：Goertzel算法

Goertzel比FFT更适合此题：
- 只需检测特定频率点，不需要全频谱
- 计算量: O(N) vs FFT的O(NlogN)
- 在MCU上实现简单

```python
def goertzel(samples, target_freq, sample_rate):
    N = len(samples)
    k = int(0.5 + N * target_freq / sample_rate)
    w = 2 * np.pi * k / N
    cosine = np.cos(w)
    sine = np.sin(w)
    coeff = 2 * cosine
    
    s_prev = 0
    s_prev2 = 0
    for sample in samples:
        s = sample + coeff * s_prev - s_prev2
        s_prev2 = s_prev
        s_prev = s
    
    power = s_prev2**2 + s_prev**2 - coeff * s_prev * s_prev2
    return power
```

**频率扫描**: 在20~100kHz范围内以5kHz步进扫描，找到两个功率峰值。

#### (4) I/Q正交解调

```
C[n] → ×cos(2πfAn/fs) → LPF → I_A
     → ×sin(2πfAn/fs) → LPF → Q_A
```

- cos/sin表: 预计算存储在MCU Flash中
- LPF: 简单IIR或FIR，截止频率<5kHz（保护最低频率间隔）

#### (5) 波形重建

**正弦波**: 
$$A'(t) = \sqrt{I_A^2 + Q_A^2} \cdot \sin(2\pi f_A t + \arctan(Q_A/I_A))$$

**三角波**: 
1. 检测基波幅度和相位
2. 生成同频同相的三角波（通过查找表或PWM+积分器）
3. 调整幅度匹配原始信号

## 四、关键算法设计

### 4.1 主流程

```c
void signal_separation() {
    // 1. 采集混合信号C
    float c_buffer[4096];
    adc_sample(c_buffer, 4096, 500000);  // 500kSPS
    
    // 2. 频率检测 (Goertzel扫描)
    float fA_detected, fB_detected;
    detect_two_frequencies(c_buffer, 4096, 500000, &fA_detected, &fB_detected);
    
    // 3. 判断波形类型 (通过谐波分析)
    WaveType typeA = detect_waveform(c_buffer, fA_detected);
    WaveType typeB = detect_waveform(c_buffer, fB_detected);
    
    // 4. I/Q解调提取幅度和相位
    float ampA, phaseA, ampB, phaseB;
    iq_demodulate(c_buffer, 4096, fA_detected, &ampA, &phaseA);
    iq_demodulate(c_buffer, 4096, fB_detected, &ampB, &phaseB);
    
    // 5. 生成分离信号
    generate_output(typeA, fA_detected, ampA, phaseA, a_out_buffer);
    generate_output(typeB, fB_detected, ampB, phaseB, b_out_buffer);
    
    // 6. DAC输出
    dac_output(a_out_buffer, b_out_buffer, 4096);
}
```

### 4.2 频率检测算法

```python
def detect_two_frequencies(samples, fs):
    """检测混合信号中的两个主要频率"""
    # 在20~100kHz范围内以1kHz步进扫描
    freqs = np.arange(20000, 100001, 1000)
    powers = []
    
    for f in freqs:
        power = goertzel(samples, f, fs)
        powers.append(power)
    
    # 找两个最大峰值
    # 排除相邻频率（可能是同一信号的旁瓣）
    peak1_idx = np.argmax(powers)
    f1 = freqs[peak1_idx]
    
    # 排除f1附近±5kHz的范围，找第二个峰
    mask = np.abs(freqs - f1) > 5000
    powers_masked = np.array(powers) * mask
    peak2_idx = np.argmax(powers_masked)
    f2 = freqs[peak2_idx]
    
    return min(f1, f2), max(f1, f2)  # 返回fA, fB
```

### 4.3 I/Q解调算法

```python
def iq_demodulate(samples, fs, f_target):
    """正交解调提取幅度和相位"""
    N = len(samples)
    t = np.arange(N) / fs
    
    # 本地振荡器
    lo_cos = np.cos(2 * np.pi * f_target * t)
    lo_sin = np.sin(2 * np.pi * f_target * t)
    
    # 混频
    i_mix = samples * lo_cos
    q_mix = samples * lo_sin
    
    # 低通滤波 (移动平均)
    window = int(fs / (2 * 5000))  # 截止频率5kHz
    i_filtered = np.convolve(i_mix, np.ones(window)/window, mode='same')
    q_filtered = np.convolve(q_mix, np.ones(window)/window, mode='same')
    
    # 取稳态段的平均值
    i_avg = np.mean(i_filtered[N//2:])
    q_avg = np.mean(q_filtered[N//2:])
    
    # 幅度和相位
    amplitude = 2 * np.sqrt(i_avg**2 + q_avg**2)
    phase = np.arctan2(q_avg, i_avg) * 180 / np.pi
    
    return amplitude, phase
```

### 4.4 波形类型检测

```python
def detect_waveform(samples, fs, f_target):
    """通过谐波含量判断波形类型"""
    # 测量基波功率
    power_fundamental = goertzel(samples, f_target, fs)
    
    # 测量3次谐波功率
    power_3rd = goertzel(samples, 3*f_target, fs)
    
    # 测量5次谐波功率
    power_5th = goertzel(samples, 5*f_target, fs)
    
    # 谐波比
    harmonic_ratio = (power_3rd + power_5th) / power_fundamental
    
    if harmonic_ratio > 0.05:  # 谐波含量>5%
        return 'TRIANGLE'
    else:
        return 'SINE'
```

## 五、设计要点与陷阱

### 5.1 加法器的精度

**要求**: 增益严格为1，否则输出幅度≠1Vpp

**陷阱**: 
- 运放有限带宽（GBW）：100kHz信号需要GBW>1MHz的运放
- 电阻精度：1%精度电阻导致增益误差2%

**推荐**: OPA365 (GBW=50MHz, 精度高)

### 5.2 ADC采样率选择

**Nyquist要求**: fs > 2×100kHz = 200kHz
**实际建议**: fs ≥ 500kHz（2.5倍过采样）

**原因**: 
- 更高的采样率提供更好的时间分辨率
- 抗混叠滤波器设计更容易
- Goertzel频率分辨率: Δf = fs/N，N=4096时Δf≈122Hz

### 5.3 频率分辨率的陷阱

**题目要求**: 频率为5kHz整数倍（20, 25, 30, ..., 100kHz）

**检测分辨率要求**: 能区分相邻的两个频率，如25kHz和30kHz（差5kHz）

**Goertzel分辨率**: 
- N=4096, fs=500kHz → Δf = 122Hz < 5kHz ✅
- 但如果信号持续时间短，实际分辨率受限于信号长度

### 5.4 三角波分离的陷阱

**如果用带通滤波器分离三角波**:
- 带通只保留基波 → 输出是正弦波 → **波形失真！**

**正确做法**:
1. 检测三角波（通过谐波分析）
2. 提取基波幅度和相位
3. 用DAC生成三角波（查找表）

### 5.5 相位控制的实现

**发挥部分2要求**: 控制B'与A'的初相位差φ，0°~180°，分辨率5°

**实现**: 
```c
// B' = sin(ωBt + φ)
float phase_B = detected_phase_B + target_phi;

// 生成输出时直接加上相位偏移
dac_output_B = amplitude_B * sin(2*PI*fB*t + phase_B);
```

**精度限制**: 
- 相位分辨率受限于DAC查找表长度
- 256点查找表 → 分辨率360/256 ≈ 1.4° ✅
- 需要保证两个信号同源（使用同一个定时器）

## 六、总结

2023-H题是一道**信号处理算法题**，核心挑战是**盲频率检测**和**波形保真分离**。

**技术路径**:
1. **ADC采样**: 500kSPS，捕获混合信号C
2. **频率检测**: Goertzel算法扫描20~100kHz，找到两个峰值
3. **波形识别**: 谐波分析判断正弦/三角波
4. **分离提取**: I/Q正交解调提取幅度和相位
5. **波形重建**: 正弦直接输出，三角用查找表生成
6. **相位控制**: 直接修改输出相位字

**与前序题目的关系**:
- 2023-D: 信号调制识别 → **频域特征提取**
- 2023-H: 信号分离 → **频域滤波 + 相干解调**
- 两者都涉及频率分析和波形重建，技术相通

**最深刻的认识**: 信号分离的本质是"在频域中将重叠的信号分开"。正弦波只在频域占据一个点，最容易分离；三角波有谐波，分离更复杂。这解释了为什么通信系统 prefers 正弦波调制。
