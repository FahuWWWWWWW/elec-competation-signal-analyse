# 2023年电赛B题「同轴电缆长度与终端负载检测装置」详解

## 一、题目复述与指标拆解

### 1.1 任务描述

设计一个同轴电缆长度与终端负载检测装置。待测电缆始端通过电缆连接头与装置连接，电缆终端可开路或接入电阻、电容负载。设置"长度检测"和"负载检测"两个按键，用以选择和启动相应功能。负载电阻值范围：10Ω~30Ω，电容值范围：100pF~300pF。装置由不大于6V的单电源供电。

### 1.2 指标拆解

| 指标项 | 基本要求 | 发挥部分 | 考核本质 |
|--------|----------|----------|----------|
| **电缆长度检测范围** | 1000cm ≤ L ≤ 2000cm | **L ≤ 100cm**（最小检测长度） | **TDR时域分辨率** |
| **长度检测相对误差** | ≤ 5% | **≤ 1%** | **时间测量精度** |
| **负载类型判断** | 正确判断开路/电阻/电容 | 同基本 | **反射波形特征提取** |
| **负载参数测量** | 无需测量具体值 | **电阻/电容值，误差≤10%** | **阻抗计算精度** |
| **检测时间** | ≤ 5s | ≤ 5s | **算法效率/信号处理速度** |
| **供电** | ≤ 6V单电源 | 同基本 | 低功耗/便携性 |
| **禁用器件** | 不得使用测距传感器及摄像头 | — | 必须用电气方法测量 |

### 1.3 核心测量对象

| 被测量 | 物理本质 | 测量方法 |
|--------|---------|---------|
| **电缆长度** | 电磁波在电缆中的传播时间 | **TDR（时域反射计）** |
| **开路终端** | 反射系数 Γ = +1 | TDR波形极性判断 |
| **电阻负载** | 反射系数 Γ = (RL-Z0)/(RL+Z0) | TDR波形幅度测量 |
| **电容负载** | 阻抗 Zc = 1/(jωC)，频率相关 | 频率响应分析/FDR |

## 二、核心物理原理

### 2.1 传输线理论回顾

同轴电缆是一种**分布参数传输线**，其特性由四个分布参数决定：

- 分布电阻 R' (Ω/m)
- 分布电感 L' (H/m)
- 分布电容 C' (F/m)
- 分布电导 G' (S/m)

**特性阻抗**:
$$Z_0 = \sqrt{\frac{R' + j\omega L'}{G' + j\omega C'}}$$

对于低损耗电缆（高频），近似为：
$$Z_0 \approx \sqrt{\frac{L'}{C'}}$$

典型同轴电缆（如RG58）：Z0 ≈ 50Ω，v ≈ 0.66c ≈ 2×10^8 m/s

**传播速度**:
$$v = \frac{1}{\sqrt{L'C'}} = \frac{c}{\sqrt{\varepsilon_r}}$$

其中 ε_r 是绝缘介质的相对介电常数（PE: 2.3, PTFE: 2.1）。

### 2.2 反射系数与TDR原理

当传输线终端阻抗 ZL 与特性阻抗 Z0 不匹配时，会产生反射。

**电压反射系数**:
$$\Gamma = \frac{Z_L - Z_0}{Z_L + Z_0}$$

**不同终端的反射系数**:

| 终端类型 | 负载阻抗 ZL | 反射系数 Γ | TDR波形特征 |
|---------|-----------|-----------|------------|
| **开路** | ∞ | **+1** | 反射波与入射波同相，幅度加倍 |
| **短路** | 0 | **-1** | 反射波与入射波反相，抵消 |
| **匹配** | Z0 | **0** | 无反射 |
| **电阻** | RL | (RL-Z0)/(RL+Z0) | 同相/反相取决于RL>Z0或RL<Z0 |
| **电容** | 1/(jωC) | 频率相关 | 波形畸变，上升沿变缓 |

### 2.3 TDR（时域反射计）测量长度

**基本原理**:
1. 在电缆始端注入一个快速阶跃脉冲（上升时间 < 1ns）
2. 脉冲沿电缆传播，遇到终端产生反射
3. 反射波返回始端，被检测电路捕获
4. 测量入射波与反射波的时间差 Δt

**长度计算公式**:
$$L = \frac{v \cdot \Delta t}{2}$$

其中 v 是电磁波在电缆中的传播速度（约为0.66c ~ 0.8c）。

**关键关系**:
- 电缆长度 100cm → 往返时间 ≈ 10ns (v=0.66c)
- 电缆长度 1000cm → 往返时间 ≈ 100ns
- 电缆长度 2000cm → 往返时间 ≈ 200ns

**因此**：
- 要分辨100cm长度的1%误差（即1cm），需要时间分辨率 ≈ 0.1ns
- 这要求TDR脉冲的上升时间 < 0.5ns，ADC采样率 > 2GSPS 或 使用等效采样技术

### 2.4 FDR（频域反射计）与S参数法

**FDR原理**:
- 向电缆注入扫频正弦信号
- 测量始端反射系数随频率的变化 S11(f)
- 反射系数的频率周期性（驻波）与电缆长度相关

**长度-频率关系**:
$$L = \frac{c}{4 \cdot \Delta f \cdot \sqrt{\varepsilon_r}}$$

其中 Δf 是反射系数幅度出现相邻极大值的频率差。

**FDR vs TDR对比**:

| 特性 | TDR | FDR |
|------|-----|-----|
| **信号类型** | 阶跃/脉冲 | 扫频正弦 |
| **时间分辨率** | 由脉冲上升时间决定 | 由扫频范围决定 |
| **频率分辨率** | 差 | 好 |
| **短电缆检测** | 困难（需要极快脉冲） | 容易（需要宽频扫频） |
| **负载类型判断** | 直观（波形形状） | 需要频谱分析 |
| **电路复杂度** | 低（高速脉冲+采样） | 中（DDS+混频） |
| **实现难度（电赛）** | 中等 | 中等 |

## 三、调理电路方案设计

### 3.1 TDR方案：脉冲反射法

```
[MCU/STM32] 
    |
    v
[高速脉冲发生器]  -- 快速边沿(上升时间<1ns)
    |                可用阶跃恢复二极管(SRD)或高速逻辑门
    v
[脉冲耦合电路]  -- 定向耦合器或电阻桥，分离入射/反射
    |              【关键电路：方向性决定测量精度】
    v
[电缆端口] ======== 待测同轴电缆 ======== [终端负载]
    |
    v
[采样保持电路]  -- 高速采样，或等效采样
    |              【关键器件：高速ADC或采样头】
    v
[信号处理]  -- 时间差测量、波形分析
    |          【可用MCU + 高速比较器】
    v
[显示]
```

### 3.2 FDR方案：扫频反射法

```
[DDS/PLL信号源]  -- 产生扫频正弦波(如10MHz~500MHz)
    |
    v
[定向耦合器]  -- 分离入射波和反射波
    |
    +---> [入射检波器] -- 测量入射功率参考
    |
    +---> [反射检波器] -- 测量反射功率
    |         |
    |         v
    |    [电缆端口] ====== 待测电缆 ====== [负载]
    |
    v
[ADC采样]  -- 同步采样入射和反射信号
    |
    v
[计算S11(f)]  -- S11 = V反射/V入射
    |
    v
[FFT分析/峰值检测]  -- 提取驻波频率间隔Δf
    |
    v
[计算L和ZL]
```

### 3.3 推荐方案：TDR + 等效采样

**为什么选择TDR+等效采样？**
1. **电赛可行性**：不需要GHz级实时ADC（太贵、太难）
2. **等效采样原理**：通过多次触发、逐步延迟采样，将ns级信号"拉伸"到μs级
3. **实现简单**：用STM32的定时器产生精密延迟，配合普通ADC即可

**等效采样TDR框图**:

```
[STM32 GPIO]
    |
    +---> [脉冲发生器] -- 固定周期触发脉冲(如1kHz)
    |         |
    |         v
    |    [电阻桥/定向耦合器]
    |         |
    |         +---> [电缆]
    |         |
    |         +---> [采样点]
    |
    +---> [可编程延迟线] -- 精密延迟τ (0~200ns, 步进0.1ns)
    |         |
    |         v
    |    [采样门/高速比较器]
    |         |
    v         v
    [ADC] <-- 采样电压
    |
    v
[等效波形重建] -- 多次采样拼接成完整TDR波形
```

**等效采样时间分辨率**:
- 延迟步进：0.1ns（由定时器分辨率决定，72MHz时钟→13.9ns，需PLL倍频或专用延迟芯片）
- 等效采样率：10GSPS（0.1ns步进对应）
- **关键器件**：可编程延迟线（如DS1023，步进0.25ns）或MC100EP195

## 四、核心算法设计

### 4.1 长度检测算法

```python
def measure_length(tdr_waveform, dt_sample, v=0.66*3e8):
    """
    从TDR波形中提取电缆长度
    
    参数:
        tdr_waveform: TDR采样序列
        dt_sample: 等效采样时间间隔 (s)
        v: 信号传播速度 (m/s)
    
    返回:
        length: 电缆长度 (m)
    """
    # 1. 找到入射脉冲的上升沿（或参考点）
    incident_idx = find_edge(tdr_waveform, threshold=0.5)
    
    # 2. 找到反射脉冲的上升沿（或峰值）
    # 方法A: 寻找波形中第二个明显跳变
    reflected_idx = find_second_edge(tdr_waveform, incident_idx + min_samples)
    
    # 方法B: 互相关法（更鲁棒）
    # template = tdr_waveform[incident_idx : incident_idx+width]
    # correlation = np.correlate(tdr_waveform, template, mode='same')
    # reflected_idx = argmax(correlation[incident_idx+guard:])
    
    # 3. 计算时间差
    delta_t = (reflected_idx - incident_idx) * dt_sample
    
    # 4. 计算长度（往返时间要除以2）
    length = v * delta_t / 2
    
    return length
```

### 4.2 负载类型判断算法

```python
def identify_load(tdr_waveform, dt_sample, Z0=50):
    """
    从TDR波形判断终端负载类型
    
    返回: 'open', 'resistor', 'capacitor'
    """
    # 提取反射波形部分
    reflected = extract_reflection(tdr_waveform)
    
    # 特征1: 反射波极性（与入射波比较）
    polarity = sign(correlation(incident, reflected))
    
    # 特征2: 反射波幅度（归一化到入射波）
    amplitude_ratio = max(abs(reflected)) / max(abs(incident))
    
    # 特征3: 波形形状（电容负载会导致波形畸变/微分）
    shape_factor = calculate_shape_factor(reflected)
    
    if amplitude_ratio > 0.9 and polarity > 0:
        return 'open'  # 开路: 同相，幅度≈1
    elif shape_factor > threshold:
        return 'capacitor'  # 电容: 波形畸变，微分特征
    else:
        return 'resistor'  # 电阻: 同相或反相，波形不变形
```

### 4.3 负载参数估计算法

**电阻值估计**:
$$R_L = Z_0 \cdot \frac{1 + \Gamma}{1 - \Gamma}$$

其中 Γ 可从TDR波形中测量：
$$\Gamma = \frac{V_{反射}}{V_{入射}}$$

**电容值估计**（两种方法）：

方法A（TDR法）: 电容负载的TDR响应呈指数衰减
$$V_{反射}(t) \propto e^{-t/(Z_0 C_L)}$$
通过拟合指数衰减时间常数 τ = Z0·CL，可得 CL。

方法B（FDR法）: 测量特定频率下的阻抗
$$Z_C = \frac{1}{j\omega C}$$
$$\Gamma(f) = \frac{Z_C - Z_0}{Z_C + Z_0}$$
通过扫频测量Γ(f)，拟合出CL。

## 五、关键电路模块设计

### 5.1 高速脉冲发生器

**方案A: 阶跃恢复二极管(SRD)**
- 原理: 利用SRD的电荷存储效应产生极快上升沿(<100ps)
- 器件: 1N5711、MA44769等
- 电路: 正弦波激励 + SRD + 短路传输线脉冲整形
- 优点: 上升沿极快
- 缺点: 需要RF设计经验，调试困难

**方案B: 高速逻辑门 + 差分驱动**
- 器件: 74LVC1G74 (触发器) + 74LVC2G86 (异或门做边沿检测)
- 或: MAX4003 (高速比较器)
- 上升沿: ~1ns（对于100cm分辨率足够）
- 优点: 易获取，易调试
- 缺点: 上升沿不如SRD快

**方案C: 雪崩晶体管**
- 器件: 2N2369、FMMT417
- 原理: 利用晶体管雪崩击穿产生纳秒级脉冲
- 优点: 脉冲幅度高(>50V)
- 缺点: 需要高压电源(~90V)，调试危险

**推荐（电赛）**: 方案B + 可选的SRD升压级

### 5.2 定向耦合器/电阻桥

**方案A: 电阻桥（惠斯通电桥变体）**

```
        R1(50Ω)        R2(50Ω)
    信号源 ──┬───[电缆端口]───┬─── 地
             │                │
            R3(50Ω)        R4(50Ω)
             │                │
             └───[差分放大器]──┘
```

- 优点: 宽带，直流可用
- 缺点: 方向性有限(~20dB)，插入损耗大(6dB)

**方案B: 微带定向耦合器**
- 耦合度: 10~20dB
- 方向性: >20dB
- 需要PCB设计，带宽有限（针对特定频段优化）

**方案C: 传输线变压器（Ruthroff 1:1巴伦）**
- 器件: FT37-43磁环 + 漆包线
- 优点: 宽带(1MHz~500MHz)，自制容易
- 缺点: 体积大，高频性能受限

**推荐（电赛）**: 方案A（电阻桥）或方案C（传输线变压器）

### 5.3 采样电路

**方案A: 等效采样**
- 核心: 精密延迟线 + 高速采样门
- 延迟芯片: DS1023 (0.25ns步进)、MC100EP195
- 采样门: 高速二极管桥或模拟开关（ADG9xxx系列，带宽不够）
- 实际: 可用高速比较器（如LT1719，响应<1ns）做阈值检测

**方案B: 高速ADC实时采样**
- ADC: AD9200 (10MSPS)、AD9288 (100MSPS)
- 采样率要求: 对于100cm分辨率，需~1GSPS（太贵、太难）
- **不适合电赛**

**方案C: 双斜率/时间间隔测量**
- 原理: 测量脉冲发射到反射返回的时间间隔
- 方法: 用高速计数器 + 精密时基
- 分辨率: 由时钟频率决定（100MHz → 10ns → 对应1m电缆）
- 需要: 内插技术或等效采样提高分辨率

**推荐（电赛）**: 方案A（等效采样）或方案C（时间间隔测量+内插）

### 5.4 信号处理与显示

- **MCU**: STM32F407 (168MHz) 或 STM32H743 (480MHz)
- **显示**: TFT LCD 2.8寸 (320x240)
- **按键**: "长度检测" + "负载检测"
- **算法**: 边沿检测、互相关、指数拟合

## 六、设计要点与陷阱

### 6.1 速度因子不确定性的处理

**问题**: 不同电缆的速度因子(v/c)不同（PE: 0.66, PTFE: 0.70, 泡沫PE: 0.78）

**解决方案**:
1. **校准**: 用已知长度的电缆校准速度因子
2. **自适应**: 根据电缆型号自动选择（电赛中可预先测量自备电缆的v）
3. **相对测量**: 先测量整根电缆长度，按比例计算截短后的长度

### 6.2 短电缆检测的盲区

**问题**: 脉冲宽度必须小于往返时间，否则入射波和反射波重叠

**脉冲宽度与最小长度关系**:
- 脉冲宽度 τp → 最小可分辨长度 Lmin = v × τp / 2
- 对于100cm最小长度: τp < 2×1m / (0.66×3×10^8) ≈ 10ns

**优化**:
- 使用更窄的脉冲（<5ns）
- 或使用后沿检测而非前沿检测
- 或使用FDR（频域法天然适合短电缆）

### 6.3 电缆盘绕的影响

**问题**: 电缆盘绕会引入额外寄生参数吗？

**分析**:
- 同轴电缆的电磁场被限制在内外导体之间，外部盘绕对内部场分布影响极小
- 但严重弯折（半径过小）会改变特性阻抗，引入反射
- **电赛允许盘绕**，说明正常盘绕不影响测量

### 6.4 电阻负载与电容负载的区分

**陷阱**: 小电阻（如10Ω）在TDR上看起来像什么？
- Γ = (10-50)/(10+50) = -0.667 → 反射波反相，幅度为入射波的67%
- 这与短路（Γ=-1）类似，但幅度较小

**电容负载的TDR特征**:
- 对于阶跃输入，电容的初始响应是短路（电容两端电压不能突变）
- 然后按指数充电至开路
- TDR波形: 初始负跳变（类似短路），然后指数回升至正电平（类似开路）
- **这是区分电容的关键特征**

### 6.5 精度提升技巧

**对于长度精度从5%提升到1%**:
1. **提高时间测量分辨率**: 等效采样步进从1ns提升到0.1ns
2. **波形平均**: 多次测量取平均，降低噪声
3. **互相关定位**: 用互相关替代简单阈值检测，精度提升10倍
4. **速度因子精确校准**: 用2000cm电缆校准v，误差<0.5%

**对于负载参数测量精度10%**:
1. **幅度校准**: 精确测量入射波幅度作为参考
2. **温度补偿**: 电阻和电容值随温度变化
3. **多频点测量**: 电容值在多个频率下测量取平均

## 七、与2025-D题（简易以太网双绞线测试仪）的关联

2025-D题要求测量双绞线（而非同轴电缆）的长度、开路/短路/交叉线对等。核心技术同样是**TDR**，但有以下差异：

| 对比项 | 2023-B（同轴电缆） | 2025-D（双绞线） |
|--------|-------------------|----------------|
| **线缆类型** | 同轴（单端） | 双绞线（差分） |
| **特性阻抗** | 50Ω/75Ω | 100Ω（差分） |
| **速度因子** | ~0.66 | ~0.67 |
| **测量参数** | 长度 + 负载类型/值 | 长度 + 线对拓扑 |
| **终端负载** | 电阻/电容/开路 | 开路/短路/交叉 |
| **调理电路** | 单端TDR | 差分TDR |

**技术延续性**: 完成2023-B后，2025-D的TDR技术可以直接迁移，只需改为差分驱动和差分采样。

---

## 八、总结

2023-B题是电赛信号及仪器仪表类中**唯一一道聚焦电缆测量的题目**，其核心是**TDR时域反射技术**。

**关键技术路径**:
1. **信号源**: 高速脉冲发生器（上升沿<1ns）
2. **耦合网络**: 电阻桥或定向耦合器，分离入射/反射
3. **采样**: 等效采样技术（避免GHz级实时ADC）
4. **算法**: 边沿检测/互相关 → 时间差 → 长度
5. **负载识别**: 反射波形极性+幅度+形状分析
6. **参数估计**: Γ→RL，或指数拟合→CL

**核心难点**:
- **短电缆盲区**: 需要窄脉冲（<5ns）或FDR替代
- **时间分辨率**: 1%精度对应0.1ns分辨率，需要精密延迟线
- **负载区分**: 电容的初始短路+后续开路特征必须准确提取

**与前序题目（2023-D）的关系**:
- 2023-D: 2MHz信号调制识别 → **频域信号处理**
- 2023-B: 电缆长度检测 → **时域信号处理**
- 两者互补，分别训练频域和时域分析能力
