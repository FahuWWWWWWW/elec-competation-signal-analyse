# 2023-D 信号调制方式识别与参数估计装置 — 架构候选

- **日期：** 2026-06-10
- **基于：** 需求冻结 (01-requirements.md)

## 候选方案对比

### 方案 A：低成本离散 IQ 混频（推荐）

**核心器件：** STM32H750VBT6 + SA612A×2 + AD9834 + AD8307 + AD8331

**系统框图：**

```
[BNC 50Ω] → [ESD/TVS] → [AD8331 VGA] ─┬→ [SA612A-I] → [LPF-I] → [STM32 H750 ADC-I]
                    AGC DAC ← STM32       │
                                          ├→ [SA612A-Q] → [LPF-Q] → [STM32 H750 ADC-Q]
                                          │
                                          └→ [AD8307 对数检波] → STM32 ADC (扫频)
                                                          ↑
                                               AD9834 DDS → LO 0°/90°
                                                         [Si5351 备选]

[TJC 串口屏] ← UART ← [STM32H750] ← [USB] / [3S LiPo 11.1V]
```

| 项目 | 说明 |
|------|------|
| BOM 预估 | ¥180 ~ ¥220 |
| 开发周期 | 4~6 周 |
| 优势 | 成本低，器件通用易采购，SA612A 资料丰富 |
| 风险 | 两路 SA612A 增益一致性需校准；LO 90° 移相网络在 100kHz~10MHz 宽带内精度有限 |
| LO 方案 | AD9834 产生 0° 输出，外部 RC 90° 移相网络或使用正交分频器 (如 74AC74) |

---

### 方案 B：集成零中频解调器

**核心器件：** STM32H743VIT6 + ADL5380 + AD9834 + AD8307 + AD8331

```
[BNC] → [AD8331 VGA] → [ADL5380 IQ 解调器] → [LPF] → [STM32 H743 ADC]
                              ↑
                         AD9834 DDS LO
```

| 项目 | 说明 |
|------|------|
| BOM 预估 | ¥280 ~ ¥350 |
| 开发周期 | 4~6 周 |
| 优势 | ADL5380 内部集成 I/Q 混频器+LO 分相，I/Q 幅度/相位匹配好 (0.05dB/0.2°)，省去外部移相网络 |
| 风险 | ADL5380 约 ¥30~50，稍超预算上限；最低工作频率 30MHz（查手册确认，可能不覆盖 100kHz） |
| 替代 | MAX2829/AD8347 等宽带零中频收发器 |

---

### 方案 C：中频直接采样 + 数字下变频

**核心器件：** STM32H750VBT6 + AD9226 (12b 65MSps) + AD8367 VGA + AD8307 + AD9834/ADF4351

```
[BNC] → [AD8367 VGA] → [ADC AD9226 65MSps] → [STM32H750 FMC 并行] → 数字混频+LPF
                    ↑                                       ↑
               AD8307 检波                                DDS NCO
```

| 项目 | 说明 |
|------|------|
| BOM 预估 | ¥250 ~ ¥320 |
| 开发周期 | 6~8 周 |
| 优势 | 无需模拟 IQ 混频器，数字 I/Q 完美正交，灵活性最高 |
| 风险 | 65MSps 并行 ADC 对 PCB 布线要求高；FMC 接口占用大量引脚；软件复杂度高；AD9226 约 ¥40~60 |
| 适用 | 对参数精度要求极高时的选项 |

## 推荐方案

**推荐方案 A (STM32H750 + SA612A × 2)：** 预算匹配度最高，器件通用易采购，开发资料丰富。主要风险是 I/Q 正交精度需软件校准，可通过已知频率的载波 (由扫频结果给出) 做数字相位补偿。

请用户确认是否按方案 A 推进选型，或选择其他方案。
