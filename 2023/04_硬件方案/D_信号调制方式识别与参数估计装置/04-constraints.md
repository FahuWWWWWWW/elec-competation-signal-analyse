## 电源树

### 电源轨

```
                          ┌─────────────────────┐
USB 5V ─┬─────────────────┤  理想二极管 (SS34)   ├──┐
         │                 └─────────────────────┘  │
         │                                           ├──→ 5V_RAW (5.0V)
3S LiPo (9.6-12.6V) ─→ LM2596 (Buck, 5V) ──────────┘
                              │
        ┌─────────────────────┼──────────────────────┐
        │                     │                      │
    ADP150-3.3            AMS1117-3.3            TJC 5V
    (低噪声 LDO)          (数字 LDO)              直供
        │                     │
        │                 VCC_3V3_DIG (3.3V)
        │                     │
    VCC_3V3_ANA          STM32H750 → 1.2V (内置 LDO)
    (3.3V 模拟电源)       W25Q64
    AD8331 VDD           Si5351A VDD
    SA612A VCC (5V)      (3.3V)
    OPA2365 VCC+
    AD8307 VDD
    REF3033 (参考)
```

### 电流预算

| 电源轨 | 电压 | 最大电流 | 主要负载 |
|--------|------|----------|----------|
| 5V_RAW | 5.0V | 500mA | AD8331 (25mA), SA612A×2 (5mA), OPA2365×2 (10mA), TJC 屏 (~100mA), AD8307 (7.5mA) |
| VCC_3V3_DIG | 3.3V | 300mA | STM32H750 (~200mA), Si5351A (25mA), W25Q64 (20mA), 上拉/接口 |
| VCC_3V3_ANA | 3.3V | 50mA | OPA2365 模拟部分, REF3033, ADC 参考 |
| VREF+ | 3.3V | 10mA | STM32H7 ADC 参考电压 (来自 REF3033) |

### 上电时序

1. 5V_RAW 上电 (LM2596 / USB)
2. VCC_3V3_DIG 上电 (AMS1117 输出)
3. VCC_3V3_ANA 上电 (ADP150 输出)
4. STM32H7 复位释放 (内置 POR)
5. STM32 初始化 → 配置 Si5351A → 开始扫频

## PCB 约束

| 项目 | 要求 |
|------|------|
| **层数** | 4 层 (顶层信号 + GND + 电源 + 底层信号) |
| **板材** | FR-4, Tg ≥ 130°C |
| **板厚** | 1.6mm |
| **阻抗** | RF 输入路径 50Ω 微带线 (线宽 ~1.8mm @ 1.6mm FR-4) |
| **模拟分区** | 顶层左 1/3 为模拟区 (RF 输入→VGA→混频器→LPF), 远离数字区域 |
| **数字分区** | 顶层右 2/3 为数字区 (STM32 + Si5351A + W25Q64 + TJC 接口) |
| **电源分区** | 底部中层为电源层, 分割 5V / 3V3_DIG / 3V3_ANA |
| **地平面** | 第二层完整 GND, 模拟区下方不分割 |
| **关键走线** | RF_IN 从 BNC 到 AD8331 走最短路径; LO 走线远离 RF_IN 避免耦合 |
| **ADC 输入** | I/Q 基带信号走差分/屏蔽走线, 远离时钟和数字信号 |
| **隔离** | 模拟/数字地通过 0Ω 电阻/磁珠单点连接于 ADC 下方 |
| **去耦** | 每颗 IC 放置 0.1µF+10µF 去耦电容, 尽量靠近电源引脚 |
| **散热** | LM2596 下方铺设铜皮散热; 无其他高热器件 |

## 关键性能指标

| 参数 | 目标 | 约束来源 |
|------|------|----------|
| 模拟带宽 (RF) | 100kHz ~ 10MHz | AD8331 (DC-120MHz) + SA612A (DC-500MHz) |
| 基带带宽 | DC ~ 200kHz (可切换) | 4 阶有源 LPF, 截止频率 200kHz |
| AGC 增益范围 | -4.5dB ~ +55.5dB | AD8331 LNA+VGA+Postamp |
| 输入动态 | 0.1mVpp ~ 1Vpp (80dB) | AD8331 增益 + AD8307 检波闭环 |
| ADC 采样率 | ≥ 500kHz (每通道 I/Q) | STM32H7 双 ADC 同步模式 |
| ADC 分辨率 | 12-bit (有效 ~10-bit) | STM32H7 内置 ADC |
| 载波频率精度 | ≤ 1% | Si5351A 晶振精度 ±25ppm |
| 调制识别准确率 | ≥ 90% (仿真验证) | signal_toolkit 算法移植 |
| 响应时间 | ≤ 2s | 扫频 ~500ms + 分析 ~500ms |
