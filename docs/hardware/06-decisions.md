# 2023-D 信号调制方式识别与参数估计装置 — 决策门

## 已确定 (可进入原理图设计)

| 项目 | 依据 |
|------|------|
| 架构方向 | STM32H750 + SA612A × 2 离散 IQ 混频 (用户确认) |
| MCU | STM32H750VBT6, 立创 EDA 封装已内置 |
| DDS | Si5351A-B-GTR, 3 路输出, I²C 控制 |
| IQ 混频 | SA612A × 2 双平衡混频器 (淘宝可购) |
| VGA | AD8331ARQZ, LNA+VGA+Postamp 单芯片完成 |
| 对数检波 | AD8307ARZ, 92dB 动态范围 |
| 基带运放 | OPA2365 × 2, 50MHz, 轨到轨 |
| 显示 | 陶晶弛 TJC 串口屏 (型号待定, 2.4" 或 3.5") |
| 供电 | USB 5V + 3S LiPo (9.6-12.6V), LM2596→5V, AMS1117→3.3V |
| 输入接口 | BNC 50Ω |
| 工作模式 | AGC 闭环自动增益控制 |
| 载波检测 | DDS 扫频 + AD8307 峰值搜索 |
| Datasheet 已下载 | STM32H750VBT6, SA612A, AD8307, AD8331, Si5351A |

## 待确认

## 高风险项

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| SA612A LO 90° 相移精度不足 | I/Q 镜像抑制下降, 影响解调精度 | 中 | Si5351A 两路输出 (OUT0/OUT1) 用各自相位寄存器设置精确 90°, 替代外部 RC 移相网络 |
| AD8307 缺货 | 无法到货, 延迟开发 | 高 | 备选 1: 淘宝 AD8307 模块; 备选 2: 换 AD8313; 备选 3: 用 STM32 ADC 直接采样 + FFT 峰值检测替代扫频 |
| AD8331 低库存 (LCSC 仅 6 件) | 买不到, 需换方案 | 高 | 备选 1: 淘宝 AD8331 模块; 备选 2: AD603 + 外部 LNA (2SC3356); 备选 3: VCA821 |
| I/Q 通道增益/相位失配 | AM 解调失真, 镜像串扰 | 中 | 软件校准: 输入已知载波, 测量 I/Q 幅度比和相位差后补偿 |
| STM32H750 内部 Flash 仅 128KB | 固件放不下 | 中 | 使用外部 QSPI Flash (W25Q64, 8MB) xip 执行, 或升级为 STM32H743 (2MB Flash) |
