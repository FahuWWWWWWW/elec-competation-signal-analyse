假设清单

  1. 装置为桌面仪器形态，无严苛体积/重量约束
  2. 工作环境为实验室/比赛现场，温度 10~40°C，无特殊防护要求
  3. 不涉及无线通信模块，无 EMI 认证要求
  4. 软件算法可参考已有的 signal_toolkit Python 库移植至 STM32（C 语言实现）
  5. TJC 串口屏分辨率和尺寸由用户后续指定（默认按 2.4" 320×240 规划）
  6. 电池方案使用 3S 锂电池 (11.1V 标称, 9.6~12.6V)，通过降压转换器供电

已确认的架构方向

  • 方案架构：STM32H7 + 模拟 IQ 正交混频下变频 + 基带 LPF + ADC 采样
  • 载波搜索：DDS (AD9834/Si5351) 扫频 LO + AD8307 对数检波峰值判定
  • AGC：前端 VGA (AD8331/AD8336) + DAC 闭环控制
  • 显示：TJC 串口屏，通过 UART 发送指令刷新
  • 供电：USB 5V 主供电，18650 电池备用，自动切换

候选方案对比

方案 A：低成本离散 IQ 混频（推荐）

核心器件： STM32H750VBT6 + SA612A×2 + AD9834 + AD8307 + AD8331

系统框图：

[BNC 50Ω] → [ESD/TVS] → [AD8331 VGA] ─┬→ [SA612A-I] → [LPF-I] → [STM32 H750 ADC-I]
                    AGC DAC ← STM32       │
                                          ├→ [SA612A-Q] → [LPF-Q] → [STM32 H750 ADC-Q]
                                          │
                                          └→ [AD8307 对数检波] → STM32 ADC (扫频)
                                                          ↑
                                               AD9834 DDS → LO 0°/90°
                                                         [Si5351 备选]

[TJC 串口屏] ← UART ← [STM32H750] ← [USB] / [3S LiPo 11.1V]
项目 说明
BOM 预估 ¥180 ~ ¥220
开发周期 4~6 周
优势 成本低，器件通用易采购，SA612A 资料丰富
风险 两路 SA612A 增益一致性需校准；LO 90° 移相网络在 100kHz~10MHz 宽带内精度有限
LO 方案 AD9834 产生 0° 输出，外部 RC 90° 移相网络或使用正交分频器 (如 74AC74)

方案 B：集成零中频解调器

核心器件： STM32H743VIT6 + ADL5380 + AD9834 + AD8307 + AD8331

[BNC] → [AD8331 VGA] → [ADL5380 IQ 解调器] → [LPF] → [STM32 H743 ADC]
                              ↑
                         AD9834 DDS LO
项目 说明
BOM 预估 ¥280 ~ ¥350
开发周期 4~6 周
优势 ADL5380 内部集成 I/Q 混频器+LO 分相，I/Q 幅度/相位匹配好 (0.05dB/0.2°)，省去外部移相网络
风险 ADL5380 约 ¥30~50，稍超预算上限；最低工作频率 30MHz（查手册确认，可能不覆盖 100kHz）
替代 MAX2829/AD8347 等宽带零中频收发器

方案 C：中频直接采样 + 数字下变频

核心器件： STM32H750VBT6 + AD9226 (12b 65MSps) + AD8367 VGA + AD8307 + AD9834/ADF4351

[BNC] → [AD8367 VGA] → [ADC AD9226 65MSps] → [STM32H750 FMC 并行] → 数字混频+LPF
                    ↑                                       ↑
               AD8307 检波                                DDS NCO
项目 说明
BOM 预估 ¥250 ~ ¥320
开发周期 6~8 周
优势 无需模拟 IQ 混频器，数字 I/Q 完美正交，灵活性最高
风险 65MSps 并行 ADC 对 PCB 布线要求高；FMC 接口占用大量引脚；软件复杂度高；AD9226 约 ¥40~60
适用 对参数精度要求极高时的选项

推荐方案

推荐方案 A (STM32H750 + SA612A × 2)： 预算匹配度最高，器件通用易采购，开发资料丰富。主要风险是 I/Q 正交精度需软件校准，可通过已知频率的载波 (由扫频结果给出) 做数字相位补偿。

请用户确认是否按方案 A 推进选型，或选择其他方案。

上电时序

  1. 5V_RAW 上电 (LM2596 / USB)
  2. VCC_3V3_DIG 上电 (AMS1117 输出)
  3. VCC_3V3_ANA 上电 (ADP150 输出)
  4. STM32H7 复位释放 (内置 POR)
  5. STM32 初始化 → 配置 Si5351A → 开始扫频

PCB 约束

项目 要求
层数 4 层 (顶层信号 + GND + 电源 + 底层信号)
板材 FR-4, Tg ≥ 130°C
板厚 1.6mm
阻抗 RF 输入路径 50Ω 微带线 (线宽 ~1.8mm @ 1.6mm FR-4)
模拟分区 顶层左 1/3 为模拟区 (RF 输入→VGA→混频器→LPF), 远离数字区域
数字分区 顶层右 2/3 为数字区 (STM32 + Si5351A + W25Q64 + TJC 接口)
电源分区 底部中层为电源层, 分割 5V / 3V3_DIG / 3V3_ANA
地平面 第二层完整 GND, 模拟区下方不分割
关键走线 RF_IN 从 BNC 到 AD8331 走最短路径; LO 走线远离 RF_IN 避免耦合
ADC 输入 I/Q 基带信号走差分/屏蔽走线, 远离时钟和数字信号
